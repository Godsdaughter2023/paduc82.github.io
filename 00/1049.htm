john sr wilson say:

This should work if the names are exactly the same

Sub deleter()

Dim osld As Slide

Dim S As Long

Dim strName As String

On Error Resume Next

strName = ActiveWindow.Selection.ShapeRange(1).Name

If Not strName = "" Then

For Each osld In ActivePresentation.Slides

For S = osld.Shapes.Count To 1 Step -1

If osld.Shapes(S).Name = strName Then

osld.Shapes(S).Delete

End If

Next S

Next

Else

MsgBox "Looks like nothing was selected"

End If

End Sub

But this is the only way i've figured out how to do it so unless you can come up with something else its this or nothing :). I will start by saying that I only use firefox as a web browser and you will have to download an extra plugin to get this to work. This workaround is for people who STILL want to use skype (as a stand alone program) and not uninstall it entirely, but simply dont want skype to start when they use outlook email. I have seen some microsoft solutions that kinda p*ssed me off saying things like "you can change your skype status to invisible on outlook.com" or "here's how to remove skype from your computer". No i dont want to set it to invisible and no i dont want to delete skype entirely, I JUST DONT WANT IT ON MY EMAIL! jeez. lol.

anyways, now that i got that off my chest, here's how i do it. essentially you have to "break" the skype connection to outlook.com

1. install AdBlockPlus plugin into your browser. Im not going to go into the detail of how to install a plugin cause its really easy (google it :) ) but as previously stated I use AdBlockPlus for firefox. i think it might be for chrome too. cant vouch for anything else.

2. add ABP (adblockplus) to your browser toolbar. for firefox you hit the menu icon in the top corner (looks like 3 stripes), click customize, then drag abp icon into your top toolbar.

3. now click on the icon for abp, that you just added in your toolbar, and go to filter preferences

4. now you're in the abp menu. go to the custom filters tab and click on add filter group, then with that group selected click add filter. in that filter put login.skype.com, then click close

5. refresh your outlook.com and go to the skype sidebar. If you've done everything right it will take a couple mins to try and log into skype but will eventually say "We couldn't sign you in. Please try to sign in again or refresh the page." click on the skype icon at the top of outlook.com to minimize it again.

6. congratulations. you've done what microsoft never intended and never wanted to allow. you've broken skype for outlook.com. Now you can check your emails and still sign into skype (if you have it installed on your computer) separately.
