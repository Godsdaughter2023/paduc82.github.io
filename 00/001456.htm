Aside from the Page Number facility on the Insert tab of the ribbon, there are a number of other ways to insert page numbers that you could try:

Put the selection where you want the page numbers to appear and

  • press ALT+SHIFT+p
  • press CTRL+F9 to insert a pair of field delimiters { } and inside them type page - { page } then Press ALT+F9 to toggle off the display of the field codes.
  • In the text section of the Insert tab of the ribbon, expand the Quick Parts drop down and click on Field... and then select the Page item from the list of Field names, select the desired formatting (if not the default Arabic numerals) and click on OK.
