Function ConcatenateRange(ByVal cell_range As Range, _
                    Optional ByVal seperator As String) As String

Dim newString As String

hansv mvp say:

I assume that you meant K8:M41, N8:P41 etc., otherwise you'd get a circular reference.

Here is a macro you can run. Create a backup of your workbook before you try it.

Sub ConcatenateColors()
    Const sep = ","
    Dim c As Long
    Dim i As Long
    Dim j As Long
    Dim s As String
    Dim a As Variant
    Dim n(3) As Long
    Dim clr(1 To 3) As Long
    clr(1) = vbGreen
    clr(2) = RGB(255, 128, 0)
    clr(3) = vbRed
    Application.ScreenUpdating = False
    For c = 8 To 20 Step 3
        s = ""
        a = Cells(8, c).Resize(34, 3).Value
        For j = 1 To 3
            For i = 1 To 34
                If a(i, j) <> "" Then
                    If s = "" Then
                        s = a(i, j)
                    Else
                        s = s & sep & a(i, j)
                    End If
                End If
            Next i
            n(j) = Len(s)
        Next j
        Cells(5, c).Value = s
        For i = 1 To 3
            If n(i) > n(i - 1) Then
                Cells(5, c).Characters(n(i - 1) + 1, n(i) - n(i - 1)).Font.Color = clr(i)
            End If
        Next i
    Next c
    Application.ScreenUpdating = True
End Sub

Microsoft Word graphic layer - Microsoft Community

wayne979 say:

Microsoft Word graphic layer

Hi. Question about setting up a document in Word. I have Word 2004 v11.2. I'm creating a digital letterhead. I have an A4 page graphic that I have placed behind the text layer. Everything works but the document can be disturbed inadvertantly. When the cursor moves outside the text area it becomes a 'hand'. This hand can select and move the graphic underneath the text layer. My question: can I lock the graphic layer, under the text layer, in position so it cant be moved without unlocking it in some way? Thanks for your answer.
bob jones aka: cybertaz say:
Don't think of layers in those terms when working with Word -- They do not exist :-) There are only 1 each of 3 layers: are attached to the text as either In Line or Anchored to a paragraph [floating]. If you'd like to find out more see Word Doesn’t Know What a Page Is  as well as the related links at the bottom of that page.

The most common method for what you described would be to place the graphics in the Header/Footer layer of the document. Doing so would prevent any accidental or intentional selection while working in the text layer. There are 3 possible down sides to consider:
  1. For viewing on screen the Header/Footer content appears dimmed while working in the Text layer. That cannot be changed.
  2. If documents are expected tol be more than a single page, at least 1 Section Break would have to be used. Otherwise the graphics will appear on all pages.
  3. In order to prevent a user from accessing the graphic Protection would have to be used which would prevent accessing the Header/Footer for that section altogether.

Regards,
Bob J.
