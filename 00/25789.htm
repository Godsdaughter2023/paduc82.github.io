Tabs in Formulas - Microsoft Community

kgriba say:

Tabs in Formulas

If I have a an Excel workbook with 50 worksheets, say with tabs named CostCodes1, CostCodes2, CostCodes3, CostCodes4 ........ CostCodes50, and then on the first tab, CostCodes1, there is a cell, J10, that grabs and sums data from each of the remaining tabs CostCodes2 thru to CostCodes50, is there a way that can be done neatly?

That is, does the formula need to actually 'hit' each and every tab, or is there a VB script that can loop through (counter?) all the tabs to get the summed value?

Thanks!

Kristine

vijay a. verma say:

What you require is a 3D formula...

Hence, if you have tab called Start and End, if you put =SUM(Start:End!A1:D10), it will sum range A1:D10 all tabs lying between Start and End including Start and End.

So, if you sheets are lying sequentially, then you can simply write =SUM(Costcodes2:Costcodes50!A1:D10)

Where A1:D10 is your range.

If you feel that all sheets may not lie between Costcodes2 and Costcodes50 and they can be before / after, 3D formulas permit wildcards also..

You can write

=SUM('*'!A1:D10) - It means all sheets except where this formula is kept.

You can also write using wildcards -

=SUM('CostCodes*'!A1:D10)

Step 3: Determine whether the problem returns

If only one service is selected in the Service list, and you still experience the problem, the selected service causes the problem. Go to step 6. If no service causes this problem, go to step 4.
