If you start the installation from within the current edition you have installed, it will reinstall that.

So, if you are now running Windows 10 Home, it will download Windows 10 Home unless specified otherwise.

The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate. You will prompted to enter a product key a couple times through the installation, click I don't have a key and Do this later.

Media Streaming and iPod sync are totally unrelated.

The only official way to put music onto an iPod is using Apple's iTunes program.

Other ways exist (I used to use this program called MGTEK that allowed you to sync music onto an iPod using Windows's own media player program.  It's not official, Apple will tell you not to use it, and even Microsoft will tell you that WMP is deprecated, but it's there for people who like taking risks.

Media Streaming, on the other hand, lets you send your media files to other computers or devices on your network, such as Xboxes or supported DLNA media player systems.
