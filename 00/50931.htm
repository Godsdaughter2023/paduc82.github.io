ralf_g say:

The 64bit version of Win 10 would have to be installed in order to use the full 8GB of RAM. In order to get that though you would have to do a clean re-installation of Windows 10 plus acquire 64bit drivers for all of the computer hardware and peripherals. Unfortunately 32bit Windows cannot simply be converted or upgraded to 64bit.

There's no difference in cost between 32bit or 64bit Windows so if your computer was supplied with the 32bit Windows installed there might be some hardware/software compatibility issue with 64bit. Best to check in the computer manufacturer's support website to see if it is possible to run 64bit Windows on that model.

alan wils88 say: Will reinstalling Windows is a solution to this problem.

ralf_g say:

No, you'd need to replace the existing 32bit Windows with a new 64bit Windows installation.

Simply reinstalling Windows will only put 32bit Windows back on the computer, so you still wouldn't be able to access the extra 4GB Ram. Understand that installing Windows 10 64bit will not preserve any of your installed software, drivers or files. If you don't have a backup of your files, or software installation media, you will lose them. The installation process will replace everything in your original Windows with a bare-bones copy of 64bit Windows 10. All of your existing software will need to be re-installed from scratch (if you have the installation media to do that). You will need to get 64bit versions of all of the computer hardware drivers before you start installing the new Windows version.

This page outlines how to go about making the change over:

https://www.howtogeek.com/228042/how-to-switch-from-32-bit-windows-10-to-64-bit-windows-10/
