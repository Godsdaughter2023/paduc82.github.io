You can download Samsung's HUtil diagnotics program from http://www.samsung.com/global/business/hdd/support/downloads/Hutil210_iso_for_CDROM_drive.zip

It is an ISO image so unzip the file and write the ISO image to CD.

Then restart the PC and boot up with CD. To do this you might have to change the boot order/priority in the BIOS to allow you to boot from CD first.

Once you have booted into the HUtil program it will detect your Samsung hard drive. Select the hard drive.

On the Main Menu enter on Tool. Go down the Tool menu and enter on Self Diagnostic. Allow the diagnostics to run, once done it will ask you if you want to run a surface scan - enter yes. The surface scan will take a while to complete. If any errors are found it will report them.

Once again I recommend backing up your data as soon as possible, preferably before you run the HUtil program.

For more info on the HUtil program you can go here: http://www.samsung.com/global/business/hdd/support/utilities/Support_HUTIL.html 

 

When write cache is enabled: When data is written back to the hard drive it is written to the cache which is much faster than actually writing to the hard drive. An acknowledgement is then sent saying the data has been written to the hard drive when in actual fact it hasn't. This way while the data is actually being written to the hard drive the rest of the system can carry on working. However since cache memory is volatile if the power goes off the data in the cache gets destroyed and if it hasn't been written to the hard drive yet it is lost.

So your settings are correct. Usually hard drives are setup for performance and the write cache is enabled. While flash drives and memory cards are setup for quick removal and write cache is disabled.

If you want to read more on this : http://www.pcguide.com/ref/hdd/op/cacheWrite-c.html
