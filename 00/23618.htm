bcf1968 say: How does one get into the "release preview ring"?

cbarnhorst say:

Same as any other ring;  on the Windows Insider Program page in Settings/Update and Security.

You don't choose a ring on that page.  You choose the type of content you want in the middle of the page (drop-down menu).  For each of those choices the ring(s) available shows in the drop-down menu in the bottom of the page.  If you choose "Fixes, apps, and drivers" in the middle drop-down, you will see "Release Preview" in the bottom box.  If you want to opt for Release Preview, you do so by choosing "Fixes apps, and drivers" in the middle box.

Remember that Release Preview means that you will preview updates for the current public release of Windows, not the next one.  You will not receive new builds of the next version of Windows.  You will receive cumulative updates and other fixes for the same build of Windows that the public is currently using.  Release Preview has nothing to do with the version of Windows being tested in Slow and Fast Rings.  It is a separate program.  The idea is to check out updates about to be released to the public for any issues that MSFT needs to catch ahead of time.

Ring taxonomy for the past six months:

Fast Ring, Active Development of Windows has been testing RS4 (Spring Creators Update)

Fast Ring, Skip Ahead has been testing RS5 (as yet unnamed)

Slow Ring has been testing RS4 (Spring Creators Update)

Release Preview has been testing RS3 (Fall Creators Update)

So if you want to test new features, Release Preview is not for you.

The history of the RP ring goes back to Windows 7 and 8.1.  Those two versions of Windows receive Optional Updates so MSFT releases upcoming updates a couple of weeks before Patch Tuesday as Optional Updates.  Anyone who wants to can try them out before they become Important Updates on Patch Tuesday.  A user can try either the Security Rollups or Quality Rollups or both one month and skip them the next because they are Optional Updates. 

The problem is that Windows 10 does not have Optional Updates (because Windows 10 is a service and not a product).  So in order to provide users a chance to preview Security and Quality rollups (called Cumulative Updates in Windows 10) MSFT created a special Insider program called Release Preview.  These Update Previews are not optional, but only Insiders on the RP ring receive them before Patch Tuesday, hence the "ring" connotation.

Having said all that, be aware that there has been an issue with the RP ring for the past year.  The updates that are supposed to be flighted to the RP ring a couple of weeks early haven't been.  They have been flighting on Patch Tuesdays, which is too late for the program to accomplish its mission of giving MSFT advance notice of issues before they reach the public.

So, you can opt into the RP ring on the WIP page as you wish.

stefan blom say:

I would start with a search for ^w^p (=blank space followed by paragraph mark) and replace with a ^p (paragraph mark). Doing that ensures that there are no trailing spaces (or tab characters) in text paragraphs.

Then use a macro that looks for completely blank paragraphs and applies Normal style to those (the assumption here is, of course, that Normal does not have a number or bullet format, or any other formatting that you don't want to expose to your blank paragraphs):

Sub ApplyNormalStyletoBlankParas()
'Macro created by MVP Stefan Blom, March 2017
Dim p As Paragraph
For Each p In ActiveDocument.Content.Paragraphs
If p.Range.Start = p.Range.End - 1 Then
p.Style = wdStyleNormal
End If

Next p
End Sub

As a long-term solution, avoid blank paragraphs altogether.
