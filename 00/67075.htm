If you're using a DVD to calibrate your TV, you're not going to get proper settings. DVDs are standard definition which uses the 601 color space, while HDTV uses 709. Google the AVS HD 709 calibration system. For your Xbox, use the MP4 files. If you have a Blu-ray player, check if it has a USB port try the MP4's on there. If that doesn't work, most Blu-ray players can decode AVCHD, so do just out and buy yourself a cheap $.50 DVD-R and burn it. it should be about 4GBs. That will get you much better results than the DVD.
