The baseline is whatever monitor is set as the primary monitor.  That one will have the crispest graphics, even if scaled.  Other displays, even those set to 100%, will have artifacts and scaling blur if they aren't the primary.  Not very intuitive, but there's a lot of legacy of why that occurs (and why no modern app built with the new platform has that issue).  Unfortunately, your only option is to disable all DPI scaling (all screens set to 100%).  That should fix it up, at the expense of having an app "appear" a different size when dragged between screens (but at least it will stay clear/crisp).
