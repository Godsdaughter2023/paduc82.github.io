jay freedman say:
Select the first item in the table and insert a bookmark there.

Then go to the place where you want to repeat the item, and insert a cross-reference there, pointing to the bookmark.

Repeat these two steps for each of the other items.

When you update an item, be careful not to delete the bookmark (or, if you do, then insert a new bookmark there with the same name as the old bookmark). It helps to go to the Display page of the Word Options dialog and check the box for Show Bookmarks, so you can see where they are.
