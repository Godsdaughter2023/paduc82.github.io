Insert a picture outside margins WORD 2010

I want to insert an icon on the top right corner of the first page in the empty space that is considered outside of the margins.

jay freedman say:
Any graphic in Word can have one of a set of "text wrapping" settings. The In-Line With Text setting makes it behave like a character, but any of the others will allow you to drag the graphic anywhere on the page. You can right-click the icon and click Wrap Text, then click one of the other values such as Square.

If your cursor is in the text on page 1 when you insert the icon, the graphic will be "anchored" to that paragraph. However, if you move or delete that paragraph, the icon will also move or be deleted (although you can change the icon's Position settings to keep it from moving with the text, up to a point).

It's often better to anchor the graphic in a header, which helps to keep it out of harm's way. To have it only on the first page and not on all pages, do this:
  • Double-click in the header area to open the header pane.
  • On the Header & Footer tools tab of the ribbon, check the box for Different First Page. Notice that the blue tag changes from "Header" to "First Page Header".
  • Insert the icon, change its text wrapping to Square, and drag it to the desired position. Because the cursor is in the first-page header paragraph, that's where the icon's anchor is.
  • Close the header. The icon will appear dimmed in Page Layout view, but it will print (including to PDF) at full strength.

Function WtConvert(rng As Range) As Double
    'InStr is case sensitive so convert to uppercase and then compare to uppercase.

    Else
        WtConvert = rng.Value
    End If
   
End Function

The only thing you're missing is that mode doesn't work as a 3D formula. You can do it with VB code to build an array of the cell values. ALT+F11 to open VB editor, tight click 'ThisWorkbook' and insert module and paste the code below in. Close VB editor.

 

Back on the worksheet call with this formula in a cell

 

=GetMode("Top","Bottom",A1)

 

 

 

 

Function GetMode(First As String, Last As String, rng As Range)
Application.Volatile
Dim i As Long, shts As Long
    Dim MyArray() As Double
    First = Sheets(First).Index
    Last = Sheets(Last).Index
    shts = Last - First
    ReDim MyArray(0 To shts)
    For i = First To Last
       MyArray(i - First) = Sheets(i).Range(rng.Address)
    Next i
    GetMode = WorksheetFunction.Mode(MyArray)
End Function

jeeped say:
When you use a column table reference like Tabela1[Valor] in a formula, do not drag it to a new location. Use Fill Right or Copy and Paste to a new location. The columns will shift (e.g. relative reference) if you drag but will not shift (e.g. absolute reference) if you use Copy & Paste (Ctrl+C, Ctrl+V) or Fill Right (Ctrl+R).

Hii

There is an option in Outlook 2007, 2010 and 2013 to recognize internet link in message and turn that into a clickable link which in turned on by default.  I think that this option is accidentally got turned off.  You can change this default setting by following the given steps.

For Outlook 2007:

Tools >> Options>> Mail Format>> Editor Options>> Proofing >>tab Autocorrect Option

For Outlook 2010 and 2013:

File >> Options >> section Mail >> Editor Options >> AutoCorrect Options >> tab AutoFormat

Here make sure the following options are selected i.e. 

Internet and network paths with hyperlinks and Plain text e-mail documents
