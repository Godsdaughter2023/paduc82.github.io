Sheet-to-sheet formula in Excel - Microsoft Community

casey facer say:

Sheet-to-sheet formula in Excel

Hello,

I am trying to create a formula in Excel that references the difference in value between the same cells between only the current and previous sheet (month to month) so I don't have to define the sheet name each time I create a new sheet with the same overall format. Any suggestions??

ossiemac say:

You could use a UDF (User Defined Function). It is a VBA method and in VBA the worksheets can be referenced by their index number which is numbered from left to right across the worksheet tabs as they are displayed at the bottom. It then returns the name of the referenced worksheet. I am not aware of any other method of referencing the worksheets by number and hence returning the previous worksheet name.

To install the UDF

  • Alt and F11 to open the VBA editor
  • Select Menu item Insert -> Module
  • Copy the code below and paste into the VBA editor.
  • Close the VBA editor (Cross far top right of the VBA editor window and it has red background when cursor hovers over)
  • Use "Save As" to Save the Workbook as Macro Enabled.
  • Ensure that macros are enabled. (See Help for how to do this. Option to "Disable all with notification" should be OK.)

Example formula to subtract value from previous worksheet from current worksheet value.

=B2-INDIRECT(PrevSht()&"!B2")

Opposite example to subtract current worksheet value from previous worksheet value.

=INDIRECT(PrevSht()&"!B2")-B2

Copy and paste the following code into a standard module in the VBA editor.

Function PrevSht() As String
    PrevSht = Worksheets(Application.Caller.Parent.Index - 1).Name
End Function

letterhead template - remove header/footer on successive pages - Microsoft Community

scott_499 say:

letterhead template - remove header/footer on successive pages



I have created a letterhead with header & footer which I have saved as a Template.  Everything works well until I do a multiple page document and do not want the header & footer on successive pages.  Is there a way to do this?  I have tried the Format-Document menu and played with the different settings for Odd & Even pages and First Page different, but it doesn't really delete or hide the header & footer.  If I delete the header on pg. 3, it deletes it on the first page which is the only page I want it on.  Thanks,Scott
john mcghie say:
Each section in a document has three headers and three footers: FIrst Page, Left Page, and Right page.  Every document has a minimum of one section.

To do what you want, you need to set the header to "Different First Page" (only!).

That gives you a header on only the first page of each section.  The other pages will have no header or footer.

Word assumes that the FIRST page of a Section will be a Right Hand page (which is appropriate in English).  Check in Format Document to ensure that the document is set to start on a "Next Page" or a "Right Page".  If it is set to something else, the First Page header will work, but the others may be on the wrong sides :-)

All of this assumes that you are NOT adding section breaks to your multi-page document.  If you are, then you need to carefully manage which side the section starts on.  And you need to know that each time Word encounters a new section, it will cycle the headers and footers again: first Page, Left Page, then Right Page.

The headers and footers that occur in subsequent sections MAY be a copy of the headers and footers in the first section break (if you set "Same as Previous" to ON) or they may be independent (if you turn "Same as Previous" OFF).

There's more on Sections here: http://word.mvps.org/faqs/formatting/WorkWithSections.htm

Hope this helps
