Yes, what you are seeing is the default behavior of Microsoft Project.  The process you should follow with using the software is:

  1. Completely plan the project.
  2. Save a Baseline for the entire project by clicking Project > Set Baseline > Set Baseline.  This will capture all of your original starting values.  For tasks, the Baseline will capture Baseline Start, Baseline Finish, Baseline Duration, Baseline Work, and Baseline Cost for all tasks.  You can see these columns in the task Baseline table by the way (click View > Tables > More Tables > Baseline to see this table).

stephen boots say:

Is the computer powered on and not sleeping at 8 PM on Sunday? The scan will not wake the PC and it will not run a "catch-up" at a later time if the scheduled time was missed.

In settings, have you selected to scan only when the PC is not in use? If it is in use, and even a screensaver can make the PC appear to be in use, the scan will not run.

What you need to do is up to you. You can initiate a scan manually or you can wait and see if MSE will run a scan before it decides to warn you that a scan has not run in some time, suggesting that you run one manually at that time to clear the alert.

What operating system and service pack?

Prior Security Software?

Any security software still installed?
