      Note If you are prompted for an administrator password or for confirmation, type the password, or click Continue.


   4. Right-click inside the Administrator: Command Prompt window, and then click Paste to paste the command that you copied in step 1.

Hi burning_fly.

My Phone service works as a sync application. This means that if you sync your contents (photos) with cloud, any change that you apply in any one of the sides (client application or cloud), that change will be also applied in the other side.

If you want to delete contents on cloud that doesn't affect the server you need to stop the service in your phone. This means that you will loose the ability to sync with cloud if you don't run again the first run experience on client application. In meanwhile you can delete photos on cloud. Besides that after insert your credentials again in the service (in mobile device) to be able to sync if you select the datatype "Photos" to be synced with cloud your photos will go back to the cloud again.

Hope this answer your question.
