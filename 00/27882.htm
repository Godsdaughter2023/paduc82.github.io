    Hi David,

     

    Yes, Class notebook support Active Directory Security Groups and Office 365 Groups when adding students.

     

    Here are the steps: 1. Create a security group:a. In the Admin center preview, choose Groups > Groups.
    TIP: You can also enter text similar to add security group into the Tell me what you want to do ... search box and the New Group dialog will open if you choose Add a group from the Tasks list.

    b. On the Groups page, choose Add a group.

    c. In the Type drop-down, choose Security group

    d. Type a name and description for the group, and then choose Add > Close.

    e. To add members, select the Security group on the Groups page and choose Edit members on the Bulk actions for groups selected dialog.

    Type the name of the person you want to add in the Search box and then choose Add. To remove members, choose Remove next to their name. please refer to: Create, edit, or delete a security group.
