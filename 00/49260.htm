  • Press Windows logo key + R key.
  • Type gpedit.msc and press Enter key.
  • Under Computer Configuration, navigate to.

Administrative Templates à Windows Components Biometrics

In the left hand panel of the Windows, double click on the following:

  • Allow the use of biometrics
  • Allow users to log on using biometrics

After restarting the computer, I suggest you to disconnect and reconnect the fingerprint sensor device and check if the drivers are getting detected in Device Manager.

  • Press Windows logo key + I key.
  • Click on Accounts.
  • Go to Sign in options.

    Broken User Profile, Windows 10 - Microsoft Community

    aaronmch say:

    Broken User Profile, Windows 10

    Hello

    I was having trouble installing the Windows 10 November update, and my computer is connected to my Active Directory Domain that I have setup in my home.  I decided to remove my computer from the domain, install the update, then add it back.

    The update went fine, however when I logged back into my normal user profile the system created a new profile named USERNAME.DOMAIN in C:\Users.  So I logged into a different account and using the Registry changed the location of my profile to my normal user profile.  When I logged back in my settings were there, but I am not able to access the start menu, search, or the date popup.

    I then tried re-naming my profile folder, which upon login created a new profile, but all of the windows apps were missing.

    I'm not sure how to resolve this, can anyone help?

    Thank you

    Aaron

    aaronmch say:

    I managed to get this fixed (kind of), I was able to successfully create a new user profile.

    For reference:

    1. I backed up the user profile (C:\Users\USERNAME)
    2. Deleted the profile using the User Profiles tool (Control Panel > System and Security > System > Advanced System Settings > under User Profiles click Settings > delete the broken profile).
    3. Then in Control Panel under User Accounts I added a new user with the name, but selected the domain (User Accounts > Manager User Accounts > Add > enter username of user to add and Active Directory domain name).
    I was then able to login with a fresh profile and everything now works fine.
