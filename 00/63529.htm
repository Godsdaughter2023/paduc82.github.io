Let's start with the easy stuff...

If you already have some printers installed, they are not showing up in the Printers and Faxes window.

Check and see if the Print Spooler Service is running and if it is not running, start it.  To do that:

Click Start, Run and in the box enter:

services.msc

Scroll to the Print Spooler Service and if the Status column does not say "Started", right click the Print Spooler
Service, Properties and set the Startup type field to Automatic.  Click the Start button to start the Print Spooler
Service and verify the Status column now shows the Print Spooler Service is Started and the Startup Type says Automatic.

Reboot your system and check the status of the Print Spooler Service again to make sure the Print Spooler Service has
started Automatically and that your changes "stick".
