jllatham say:

I'm kind of interpreting your needs the same way that rich007a has, that you're working with a regular worksheet and not a UserForm.  His answer is spot on for that.  I can only add that the sequence of cells selected will also be controlled by the setting accessed from

Tools --> Options --> [Edit] tab --> Move Selection After Enter choice.

If it is set to "Down" the sequence will be top to bottom, left to right across the sheet.

If it is set to "Right" the sequence will be left to right, top to bottom.

These settings can be set through VBA code so if your normal setting doesn't work right, you could change it through the Workbook_Activate() and reset it to "default" in the Workbook_Deactivate() processes.

Lets say your normal/default setting is "Down" but you need it to go from left to right in this workbook, then when the user quits using the workbook, you need it go back to the default.  Code would look like this:


Private Sub Workbook_Activate()
  Application.MoveAfterReturnDirection = xlToRight
End Sub

Private Sub Workbook_Deactivate()
  Application.MoveAfterReturnDirection = xlDown
End Sub

If you only need the change on the one worksheet, then you can use the Worksheet_Activate() and Worksheet_Deactivate() events for that specific worksheet to do the same things.

 

rich007a say:

You say you want to use Excel as a form, so you're not using a UserForm in VBA right?

In that case, you need to format the cells you want to be editable by setting them as being Unlocked.

Right-click the cell and then Format Cells... go to the Protection tab.  Uncheck the Locked checkbox.

Once you've unlocked all the cells you need to, you can then protect the sheet.

Tools > Protection... > Protect Sheet.

Uncheck allow the user to "Select locked cells"

Check allow the user to "Select unlocked cells"

User will now be able to tab between the cells.

Don't think you can set the tab order though.  To do that you have to use a UserForm in VBA.
