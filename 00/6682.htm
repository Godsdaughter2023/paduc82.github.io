mike middleton say:

Without anything selected on the chart, a single click on one of the points should select the entire data series.

After that first click, i.e., after the entire data series is selected, a subsequent click should select a single XY point (a useful method for then formatting a single point).

The above command line will generate a report of the currently installed/activated license. (See Below)


NOTE: You might see multiple licenses in the /dstatus report.3.  Make note of value for “Last 5 characters of installed product key”, Run  C:\program files\Microsoft office\office16>cscript ospp.vbs /unpkey:“Last 5 of installed product key”   For example: C:\program files\Microsoft office\office16>cscript ospp.vbs /unpkey:G8YH4 (See Below) Repeat the step above if necessary until all keys are removed.

B.  Remove cached identities in HKCU registry:

Open Registry Editor, navigate to Computer\HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\Common\Identity\Identities and remove all of the identities under \Identities.

C.  Remove the stored Credentials in the Credential Manager:

Open Control Panel > Credential Manager.  Remove all Windows credentials listed for MicrosoftOffice.

D.  Restart computer, check if you can activate Office with your account.

If the error exists after you try the steps above, please help provide the information below for investigation:

3.  Please open an Office application such as Word, and then open a blank document>go to File tab>Account to capture a screenshot of your Office product information.

4.  Please help capture a screenshot of the message.
