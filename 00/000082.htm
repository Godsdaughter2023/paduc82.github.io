Countif with different criteria range as count range - Microsoft Community

excel_1982 say:

Countif with different criteria range as count range

Has anyone figured out a way to use countif similar to sumif where you can enter an optional range you want summed that is different than your criteria range? Is there a way to nest an if? Write the function in VBA?

bernie deitrick say:

SUMIF allows a range to be summed based on a separate criteria range - as long as the ranges are the same size, like

=SUMIF(A1:A10,"Test",B1:B10)

which will sum the values from B1:B10 where A1:A10 is "Test"

COUNTIFS  and SUMIFS both allow multiple criteria - but COUNTIF can never return a sum, just a count.

The alternatives are SUMPRODUCT (this is the same as the above SUMIF):

=SUMPRODUCT((A1:A10="Test")*B1:B10)

Or array-entered (entered using Ctrl-Shift-Enter) SUM functions, like this (again, the same as the above SUMIF)

=SUM(IF(A1:A10="Test",B1:B10))

If you need better help, then describe what you have with an example, and what you would want returned from your example.

julie fka profjulie say:
You can create a contact in your people hub called something like "Monday Conference Call" and  input the string (999-888-7777,,6543210#) in the phone number field.

Then when you are ready to dial into the conference call, just call that contact.  All the numbers will be dialed immediately in sequence, so the first time you do this it will look like this method won't work, since it appears that the pauses don't pause the call to wait for the rest of the sequence, but this is how I handle these kinds of calls on my windows phones and it's worked for me for the last 3 years.

Then you can pin the contact to your home screen and you're ready to go.
