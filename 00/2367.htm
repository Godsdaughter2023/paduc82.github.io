a. user say:

MichaelWeir -

Thank you for posting.

Zune and Xbox Music use different DRM systems, it was changed with Windows 8.  Combining the DRM collection between both into one folder, regardless of where it's saved, would cause playback errors down the road.

Also, Zune has always been finicky with putting DRM content on anything other than the primary hard drive, so I'm not sure whether it would actually work to set a OneDrive folder as the default save spot.

One last thing to note, Zune devices themselves can only recognize the DRM for music downloaded within the Zune software, so you'd have to keep things separate to ensure that will all play correctly.

As far as the non-DRM content goes, that should be safe to have it saved in a shared folder.

In the Zune software, you can go to Settings -> Software -> Collection, and manage your monitored folders there.  In Xbox Music, this is done within the app by pressing Windows+C to open the Charms bar, then select Settings -> Preferences -> Choose Where We Look on This PC.
