So, despite bumping multiple times to have someone at Microsoft help me with my problem, I managed to fix the issue on my own.

As I thought before, it's pretty much a registry problem that happens when upgrading to Windows 10 while having certain custom themes installed.

The registry value that needs to be changed specifically is the "CaptionFont" value under:

HKEY_CURRENT_USER\Control Panel\Desktop\WindowMetrics

However this is a hex value and should not be changed manually. Ideally what you want to do is update that registry value with a registry file from a clean Windows 10 user account:

  1. Get on a computer running Windows 10 that's not running your user account or switch to/create a new user account with a clean registry on your own computer.
  2. Right click the Start Menu and select "Run" from the dropdown menu.
  3. In the run window, type in "regedit" and click OK
  4. In the left pane of the registry editor, navigate to the "HKEY_CURRENT_USER\Control Panel\Desktop\WindowMetrics" folder.
  5. Right click that folder in the navigation pane and select "Export".
  6. Save the registry file in a location where you can access it from the user account you are trying to fix.
  7. Return to the user account that you want to fix and repeat steps 1 through 6 to save a backup of the registry values in case something goes wrong.
  8. Double click the registry file you saved from the other user account to fix the values in your registry.
  9. Sign out, then sign back in.

If all went well, your title bars should be normal and the title bar size values in the dropdown menu in the control panel should also be fixed.
