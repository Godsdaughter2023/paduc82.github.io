malkeleah say: The First Law of Driver Updates is "if it ain't broke, don't fix it". Normally if everything is working you want to leave things as they are. The exception is that heavy-duty gamers will usually want to update their video and sound drivers to squeeze every last bit of performance out of the hardware to get the fastest frame rates. If you're not one of those people, you don't need to update your drivers if there are no problems you are trying to solve.

Never get drivers from Windows Update. Get them from:

1. The device mftr.'s website; OR
2. The motherboard mftr.'s website if hardware is onboard; OR
3. The OEM's website for your specific machine if you have an OEM computer (HP, Dell, Sony, etc.).

Read the installation instructions on the website where you get the drivers.

To find out what hardware is in your computer:

1. Read any documentation you got when you bought the computer.
2. If the computer is OEM, go to the OEM's website for your specific model machine and look at the specs (you'll be there to get the drivers anyway)
3. Download, install and run a free system inventory program like Belarc Advisor or System Information for Windows.

Windows 10 Favorites Not Syncing - How To Fix - Microsoft Community

keith elkin say:

Windows 10 Favorites Not Syncing - How To Fix

Working in IT and playing with the Insider builds I find myself frequently wiping out my PC and reinstalling Windows. I have quite a few different desktops and laptops and noticed that even though I'm logged into my Microsoft account, my Edge favorites and history sometimes would never sync. I always got it working but never paid attention to which actions I performed fixed it... until now.

I noticed if you do a fresh install of Windows 10 and manually add a Microsoft account from the Settings --> Accounts --> Add a Microsoft Account, while it does add the account it doesn't seem to initialize the cloud services properly to allow favorite syncing even though the options to sync were enabled under Settings as well as the Edge browser itself.

To work around this, I removed the linked Microsoft account from the Settings --> Your email and Accounts tab, then rebooted.

I then clicked on the Settings --> Update & Security tab, clicked Advanced,  then clicked the option to get Insider Builds.

This launched a wizard that prompted me for my Microsoft account credentials, and initialized my machine properly as once I did this and re-enabled syncing in my Edge browser, the favorites showed up with a minute or two.

Note: Even though you click the button to get Insider builds and enter your Microsoft account credentials, you will not actually start getting insider builds until you then select the Get Started button to chose which ring of builds you want; so if you don't actually want insider builds, don't worry about it. You won't get them  just because you launched this wizard.

If you do want insider builds, then make sure you click the Get started button.

I hope this helps people like me who had this issue multiple times.

I should also note, when experiencing the problem where I had the Microsoft account linked and all sync settings enabled, and my Microsoft account *was* verified, but wouldn't sync my favorites I tried running the Microsoft Accounts Troubleshooter. It yielded the error "Live Provider is not initialized properly with the cloud services" but offered no fix or information on how to fix it. Performing the steps above fixed that error as well.
