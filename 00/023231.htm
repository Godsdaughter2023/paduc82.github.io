dvdmanuk say:

Check if NOW() is Friday

I'm doing a video schedule and my week starts on Saturday from 6:00:00 am, and finishes on Friday at 5:59:59 am, but once it's Friday excel is changing everything to Saturday of the following week.

 

This is fine, but I have now created a Table that shows in real time when I should be videoing so I can double check if it is.

 

So what I need is to have a column that tells me that it is still Friday.  In this column I need to enter a formula that will check using "NOW()" to see if it is from Friday 12:00:00 am to Saturday 6:00:00 am, and if it is between these two times then say it is "Friday".

 

I have this formula but it does not work 100% of the time...

So to break down the Formula I need is...

 

Is NOW() from Friday 12:00:00 am

and

Is NOW() before Saturday 6:00:00 am

 

If between these times then say "Yes"

but

If not between these times then say "No"

 

Any help with this simple question will be a great help to me.

 

Thanks in advance,

Neil

 

Ps. I know this is an easy thing, but I just cannot figure it out...

teylyn say:
Hello,

as far as I can see you formula cannot work since you are ultimately comparing text values, not real date/time values.

You cannot compare a text representation of a date to see if that date is greater than another date. The text representation like "Friday 12:00:00" is always smaller than any other text representation like "Tuesday 12:00:00", since the sort order is alphabetical and does not take into account the date at all. It looks at the text "Friday" and compares it to "Tuesday" and since "F" comes first in the alphabet, any "Friday" will be smaller than any "Tuesday", no matter what the date may be.

For your scenario, I suggest a different approach. 

Evaluate the date/time value based on the weekday, combined with the time value of that cell. So, if it is a Friday, the Weekday() function will return a 6, and if it's a Saturday, the weekday function will return a 7 (out of the box, without any parameters). 

So, get the days that are a 6 or 7. Then narrow down the time frame for these days. For a Friday, you want to include any time, so no further time qualification is required if the weekday is a 6.

If the date is a Saturday, you want to include it only if the time is less than 6 am. The time value 6 am equals 0.25 as a decimal. You can evaluate only the decimal part of a number by using the Mod() function.

So, in pseudo-code:

If EITHER cell A2 is a Friday OR ( cell A2 is a Saturday AND the time of cell A2 is before 6 am), print "yes", otherwise print "no".

In an Excel formula that translates to

=IF(OR(WEEKDAY(A2)=6,AND(WEEKDAY(A2)=7,MOD(A2,1)<=0.25)),"yes","no")

Does that make sense?

cheers, teylyn
