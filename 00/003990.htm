If you are running a preview Windows Insider build that has an expiration date, just upgrade to Windows 10 version 1803. Your device may be permanently licensed for Windows 10, but you are running a preview version that will expire. Expiring does not remove your license for Windows 10 on that device.

Go to the website linked below, to find the Media Creation Tool. With that, you can download an ISO file for Windows 10 version 1803.  Mount that ISO file in Windows and it will create a drive letter, from which you can run setup.exe to upgrade to a non-expiring version of Windows 10.
