mouneshwar r say:

Hi BabaG,

 

To answer your questions:

 

Step 1:Will Windows XP see the disks as foreign disk once I switch back to Windows XP?

Yes. When you move a disk from one computer to another computer, it is considered as a foreign disk. In some cases, a disk that was previously connected to the system can display the foreign status.

 

Step 2: Will i have to re-import into each Operating system until I'm done testing every time i make a switch and need to use the raid?

Yes. You’ve to re-import into each operating system every time you make a switch and need to use the raid.

 

Step 3: Am i correct in the first place that all i need to do is import the foreign disks to get them back online?

Yes. In Disk Management, right-click the disk and then click Reactivate Disk to bring the disk back online. If the disk status remains Offline, check the cables and disk controller, and make sure that the physical disk is healthy.
