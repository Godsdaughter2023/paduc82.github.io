Option Compare Database
Option Explicit

Function bbb(vlr)
Dim i As Long
bbb = ""
For i = 0 To Len(vlr) - 1 Step 2
    If i = 0 Then
       bbb = Mid(vlr, i + 1, 2)
    Else
       bbb = bbb & ", " & Mid(vlr, i + 1, 2)
    End If
Next i

End Function

I like mimmo's function but would probably make it more flexible so you could change the string to stuff into the original string as well as the size of the groupings. I think this modified function provides the flexibility:

Function bbb(vlr, strStuff, intGroupSize)
 Dim i As Long
 bbb = ""
 For i = 0 To Len(vlr) - 1 Step intGroupSize
     If i = 0 Then
        bbb = Mid(vlr, i + 1, intGroupSize)
     Else
        bbb = bbb & strStuff & Mid(vlr, i + 1, intGroupSize)
     End If
 Next i
End Function

Using Left as I suggested does exactly the same thing as the slightly more complicated Mid function when I = 0

As for Duane's version of the function, the strStuff and intGroupSize are arguments to the bbb function.  These arguments allow you to specify the separator and the number of characters in each group.  For your original question you would  call the function this way:
