Hi,

If a Windows application or service uses lots of CPU resources for an extended period, the processor will get warm. Even if your device is on sleep mode or standby mode. We suggest that you use Performance Monitoring Tools to determine what is causing the high CPU utilization. Also, the faster the processor, the warmer it can become.

Here are three ways on how to open Performance Monitor:

Method 1: Open Start, do a search for Performance Monitor, and click the result.

Method 2: Use the Windows key + R keyboard shortcut to open the Run command, type perfmon, and click OK to open.

Method 3: Use the Windows key + X keyboard shortcut to open the Power User menu, select Computer Management, and click on Performance.

To prevent its thermal issue, please follow the resolutions below:

  1. Configure the "put the computer to sleep" setting to something other than never.
  2. Make sure Bluetooth mouse is turned off.
  3. Turn off Windows Hello facial recognition. Under Windows Hello hit the Remove button. You do not need to restart the computer.

ken blake say:

*Everything* on your computer--Windows, application programs, data, etc.--is on your hard drive. If you replace the drive, it will all be lost. You will have to reinstall Windows, reinstall all your application programs, reconfigure everything the way you like it, and restore all your  data from the external  backups that you should create before replacing the drive.

And by the way, if it were me, I would buy a 2TB drive, not a 1TB drive. You get twice as much space for only a little more money.
