I would like the column containing todays date to appear to the right of column D.

Does anyone know of an easy way to do this?

Thanks

hansv mvp say:

Press Alt+F11 to activate the Visual Basic Editor.

Double-click ThisWorkbook under Microsoft Excel Objects.

Enter the following code into the workbook module:

Private Sub Workbook_Open()
    Dim c As Range
    Worksheets("Sheet1").Select
    Set c = Range("F3:NF3").Find(What:=Date, LookIn:=xlFormulas, LookAt:=xlWhole)
    If Not c Is Nothing Then
        Application.Goto Reference:=c, Scroll:=True
        Range("E3").Value = Split(c.Address, "$")(1)
    End If
End Sub

Press Alt+F4 to close the Visual Basic Editor.

Save the workbook as a macro-enabled workbook (.xlsm).

Make sure that you allow macros when you open the workbook.
