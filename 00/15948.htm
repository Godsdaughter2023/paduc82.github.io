Hi there, the problem is that hiding (filtering) the column means vba is trying to change something that is not available. So try this instead...

Private Sub Worksheet_Calculate()
Dim VarPivItem As Variant
    With ActiveSheet
        For Each VarPivItem In .PivotTables("PivotTable1").PivotFields("R_Stat").PivotItems
            If VarPivItem.Visible Then
                .PivotTables("PivotTable1").PivotSelect "'" & VarPivItem.Name & "'", _
                xlDataAndLabel, True
                Selection.Font.Bold = True
                Selection.HorizontalAlignment = xlCenter
                With Selection.Interior
                    .Pattern = xlSolid
                    .PatternColorIndex = xlAutomatic
                    If VarPivItem.Name = "R" Then .Color = 255
                    If VarPivItem.Name = "A" Then .Color = 1030655
                    If VarPivItem.Name = "G" Then .Color = 3381504
                    If VarPivItem.Name = "H" Then .Color = -1
                    If VarPivItem.Name = "NS" Then .Color = -1
                    .TintAndShade = 0
                    .PatternTintAndShade = 0
                End With
            End If
        Next
    End With
End Sub

john_l. say:

When updates for your Windows Phone 7 are available, you'll see a prompt on your phone. You'll need to use the Zune software to install updates.

To install an update for your Windows Phone 7

  1. Start the Zune software.
  2. Connect your phone to your PC with a USB cable.
  3. When Zune prompts you to install updates, clickUpdate Now and follow the on-screen instructions.

You can also check for updates by going toSettings, Phone, Update. If an update is available, clickUpdate.

Note

You must have the latest version of the Zune software before you can update your Windows Phone 7. If an update is available for the Zune software, you will be prompted to update it before you can update your phone.
