ronnie vernon say:


Hi

Libraries are Virtual Folders and they only use very little of the space on the System drive (C:) where they are located.

One of the main reasons that Libraries were created was to save space on the system drive. You can move all of the large data folders off of the system drive, to remote locations, add those folders to a library and you will see that this only adds a small fraction of space compared to the actual size of the folders to the system drive, and you still have access to them locally.

When we were testing this during the beta, we added a 2 Gigabyte folder that was physically located on a remote drive to a Library and it used less than 1 Megabyte of space on the system drive. I also added a 20 Gigabyte folder and it used around 10 Megabytes of space on the system drive.

When you look at the Properties of a folder that has been added to a Library, you are seeing the properties of the actual, physical folder and not the space that it is using on the drive where the library is located.

The only way you can actually see how much space a Library uses on the system drive is to compare the size of the free space on the system drive before, and after, a folder is added to a Library.

Hope this helps.

Thank You for using Windows 7
