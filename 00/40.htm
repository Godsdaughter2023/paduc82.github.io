If you are using SmartArt, the default format for the 1st org chart does not have shadowing.

However, lets suppose you have picked the 2nd org chart style, here is how you would get rid of it if you wanted to (by the way, in the 2nd org chart style, it is really not a shadow but an offset shape):

1. Select the "shadow" for one of the shapes, hold down the Ctrl key and select all the other "shadows".  SmartArt Tools, Format, Shape Fill, No Fill.

2. If the shapes do have real shadows here's what you do - select the border of the entire SmartArt object and choose SmartArt Tools, Format, Shape Effects, Shadow, No Shadow

I'm an Office 365 (OneDrive for Business) client, and you seem to be experiencing something that has being annoying me for some time. I also use OneDrive (personal).

Not only ALL Office files are DOUBLED in the OfficeFileCache directory you've mentioned (you'll notice a bunch of .FSF and .FSD files in there, all created while you sync new files), but the minimum size for .FSD files is 128KB. IE, all your Office files under 128KB will generate doubles with 128K. That's ridiculous! And I have more bad news: latest version of OneDrive (personal) is also doing that.

Messing around after reading a lot of material on-line, after all syncing was done, I've decided to delete all .FSD and .FSF files (JUST THOSE FILES! There's a database file in there that must be kept.). That resulted in absolutely NO IMPACT on both OneDrive (personal) and OneDrive for Business. Both continue to sync properly to this date.

A collateral effect, however, happens when I try to open a synced Office file. As there's no local cache for the file, Excel/Word/PowerPoint opens nothing on first try. It's no a crash, but a blank opening. When I double click the file again, it's downloaded (to cache) and everything opens and runs fine from then. If I delete the cache, that will happen again, of course.

I can live with that while Microsoft does not decide to IMPROVE IT (I guess they never will...), as I definitely don't want my synced files doubled.
