The first issue (and most common) stems from re-installing on top of existing installations. In general, don’t do this. Video | Uninstall Office 2011

Step-by-Step Instructions | How to completely remove Office for Mac 2011

  1. With Finder open, choose Go > Go to Folder...

  2. Type in the following path and then click Go:
    /Library

  3. Find the Automator folder, select it, and then click the name of the folder again to Rename the folder. Name the folder Automator_Old

  4. Attempt to install the 14.2.4 update again.

  5. If the update installs correctly, rename the Automator_Old folder back to its original name: Automator

 

If that solution does not work and you know you have re-installed Office 2011 on top of an existing Office 2011 installation, you’ll need to remove Office 2011 properly and re-install. 

 

The second issue was some users changed the size of the /tmp/ directory on their machine which was too small for the update to fully install. Re-sizing the /tmp/ directory resolved the install issue for these users.

a. user say:

Hi SydneyWyatt14,

Thanks for contacting Microsoft Support.  I understand you're having issues downloading apps to your Nokia Lumia 521.  Just a heads up, this is actually the Windows Phone 7 forums, so for future reference, if you have issues with your Windows Phone 8, we ask that you try to post in the appropriate forum, that way, if other users have the same issue, it'll be easier for them to find.

The reason for this error code is that your Microsoft Account can't sync properly to your phone.  In the meantime, I'd like you to try the following:

  • On your phone, open up Settings>Date+Time.  Turn off Set Automatically.  What are the values set for under Date, Time, and Time Zone?  Are they correct for the region you are in?  If not, update them to the correct values and then turn the phone off for two minutes and the back on.
  • Clear the history for Internet Explorer on your phone. 
  • Try downloading with using only a Wi-Fi connection and just your cellular data as well.
  • On your phone, navigate to Settings>Email+Accounts.  Tap and hold your finger to your Microsoft Account email and tap Sync.

If you still can't sync the account and download apps, then we would have to try the hard reset which would erase everything.  Is there any to content you needed to back up?

Feel free to get back to us when you can and we'll go from there.

Regards,

Chanh S.

ibzie say:

THE FIX IS:-

1. Please press the Windows key + W. This will open the Settings Search.

2. On the Settings Search field, please type Control Panel and open the Control Panel interface on the left panel.

3. On the Control Panel, please choose the Network and Sharing Center option.

If you do not see the Network and Sharing Center on the list, you may need to click on Network and Internet first.

4. Please click on the Change Adapter Settings option on the left panel of the Network and Sharing Center window. This will open the Network Connections window.

5. Please right click on your active network connection and choose Properties.

6. On the Networking Tab, please highlight the option for Internet Protocol Version 4 on the list and click on the Properties button.

7. On the General tab of the Internet Protocol Version 4 (TCP/IPv4) Properties window, choose the option for Use the following DNS server addresses and set the Preferred DNS server as 8.8.8.8 and the Alternate DNS server as 8.8.4.4

8. Click on OK and Close.

9. After setting up the global DNS settings, please check if you can now open websites in Internet Explorer.
