malkeleah say: You cannot fix this without a real XP installation CD. Either borrow one from a friend or take the computer to a reputable local computer tech. Don't use a BigComputerStore/GeekSquad type of place.

The usual fix is:

First make sure there is no floppy in the floppy drive (if you have one). If there is, remove it and reboot the computer. Hopefully all will be fine. Attrib  -H -R -S C:\boot.ini [enter]
  (if not found skip the next line)
DEL  C:\boot.ini [enter]
BootCfg  /Rebuild [enter]

When the process is finished, remove the XP disc and hopefully boot into Windows.

thair57 say:

Recycle Bin Memory Leak Windows 10

I have had issues recently when cleaning up large directories of files.  I copy them from location to location and then delete the originals.  This results in a very large recycle bin (upwards of 20K files, maybe 20GB of data). Occasionally, I find that I got overzealous and have to go back to the recycle bin and retrieve and errant deletion. This has caused, on numerous occasions, low memory alerts and even a total freeze of the machine.  Looking at the Task manager, it show Explorer using as much as 8-10GB of RAM. At first I was not aware of what the cause was as I multi-task frequently and felt that the problem was probably with some old piece of software I was hanging onto, but I have since found the culprit. 

If you access the Recycle Bin when it is very full, Explorer ramps up RAM usage.  If you sort the bin, it ramps up even more.  Closing the bin (not emptying it) does NOT release the RAM.  Repeated access to the bin only increases the RAM consumption to the point of system freeze. Restarting Ecplorer obviously frees up the memory, but why should that be necessary? I've seen this same issue reported more than three years ago and apparently it still had not been corrected.

Before you tell me to do so I've already run sfc and even did a clean install from USB boot media and the problem remains.  I can repeated recreate the issue, and now that I am aware of the cause I can avoid it.  I am reporting this issue because I haven't seen anyone that knew exactly what was causing their memory leak problem. And since Explorer.exe and the Recycle Bin are both part of Windows itself, the problem appears to be with the OS, and not user software/drivers.
