Where's Dropbox - Microsoft Community

robinfinke say:

Where's Dropbox

I just purchased a new computer and Word for Office and Student 2016. I have hundreds of files on Dropbox on my other computer running 2013 but can't figure out how to save my Word Documents and files to Dropbox now.  I don't see that option on add a new place!
jay freedman say:

The "Add a Place" button is misleading; it supports only SharePoint and OneDrive.

Dropbox operates by monitoring the content of the Dropbox folder on your local drive, and uploading new and changed files to their cloud. To save files "to Dropbox" you simply save them into your local Dropbox folder and let the Dropbox agent do the rest.

Unfortunately, a recent update to Office has removed the Recent Folders list from the File > Save As page and the ability to pin folders there. There's a request at uservoice.com to restore that feature (add your vote!), and we have hopes that it will be restored, but in the meantime use this workaround:

Go to File > Options > Save. Check the boxes for "Don't show the Backstage when opening or saving files" and for "Save to Computer by default". Enter the folder path you want for saving most files (such as your Documents folder) in the "Default local file location". Click OK. Use the Ctrl+O shortcut to open files, and F12 to do a Save As. These commands will open the traditional Open and Save As dialog boxes, where you'll find the Dropbox folder in the left column when you want to save there.

I would avoid opening files directly from the Dropbox folder, because Word creates several temporary files in the same folder as the document file, and you don't want the Dropbox agent to interfere with those files or upload them to the cloud. Work on them in the Documents folder or other local folder, and copy the finished files to Dropbox or do an additional Save As there.

graham mayor say:
FWIW my Save in Two Places add-in will facilitate the copy to dropbox as the process saves in one place and copies to the backup location, so no temporary files in the backup folder. Make the backup location the Drobox folder, for the documents that require it, and the process will be transparent. You can choose whether to allow files to be overwritten or renamed when they already exist in the backup folder. http://www.gmayor.com/SaveInTwoPlacesAddIn.htm

use the following macro to delete all hyperlinks in the active workbook:

Sub DeleteHyperlinksWorkbookwide()
Dim shSheet As Worksheet
Dim myHyperlink As Hyperlink
  For Each shSheet In ActiveWorkbook.Worksheets
    For Each myHyperlink In shSheet.Hyperlinks
      myHyperlink.Delete
    Next
  Next
End Sub

Slow Word 2013

I experienced a similar issue with slow typing speed in Office 365 Word. I edit a substantial number of Word and Excel documents for work and found the SkyDrive storage incredibly beneficial for working remotely.

I realized though that documents which are being edited from Skydrive, not a local copy, experience the slow typing speed issue.

The fix:   From word:  File, Options. Word Options has an advanced tab on the left labeled 'advanced'. About 3/4 of the way down, a category for 'Save' options exists. The default setting has 'Allow Background Saves' checked. I uncheck this option and instead check 'Always create backup copy' and 'Copy remotely stored files onto your computer, and update the remote file when saving'.

I assume that if you're storing the files on Skydrive, or Dropbox, or web, or maybe even some external drive or flash memory this background save could be lagging and result in typing slowness. Either way, I changed the advanced options, and no more problem. Maybe it'll work for you.

Excel didn't have the option, and I usually save a local copy anyways due to the working file size.

*I run windows 7 with office 365 and windows surface pro with win 8 and office 365. 4gb memory each.
