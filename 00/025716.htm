How to create a different Microsoft account - Microsoft Community

georgelaing say:

How to create a different Microsoft account

I have two computers both have the same Microsoft account how can I change one to a different account

a. user say:

Hi George,

To create a new Microsoft account

  1. Click on Windows. Select Settings.

  2. Click Accounts, and then Click Other accounts.

  3. Click Add an account.

  4. Enter the account info for this person to sign in to Windows. There are four ways to do this:

    • If the person you're adding already has a Microsoft account, enter it now.

    • If the person you're adding doesn't have a Microsoft account, you can use their email address to create one. Enter the email address that person uses most frequently.

    • If the person you're adding doesn't have an email address, tap or click Sign up for a new email address. It's free.

    • If the person you're adding is a child, tap or click Add a child's account.

  5. Follow the instructions to finish setting up the account.

You can simply delete the old account by clicking on it and select remove.

Hope this helps. Get back to us for more assistance on Windows.
