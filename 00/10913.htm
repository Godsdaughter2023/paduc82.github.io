mitch rodelco say:

Convert pages of document to files

How can I convert each page of a multi-page document to individual documents?
doug robbins - mvp office apps & services (word) say:

Use the following macro:

Sub splitter()
'
' splitter Macro
' Macro created 16-08-98 by Doug Robbins to save each page of a document
' as a separate file with the name Page#.DOC
'
Dim Counter As Long, Source As Document, Target As Document
Set Source = ActiveDocument
Selection.HomeKey Unit:=wdStory
Pages = Source.BuiltInDocumentProperties(wdPropertyPages)
Counter = 0
While Counter < Pages
    Counter = Counter + 1
    DocName = "Page" & Format(Counter)
    Source.Bookmarks("\Page").Range.Cut
    Set Target = Documents.Add
    Target.Range.Paste
    Target.SaveAs FileName:=DocName
    Target.Close
Wend
End Sub

Don't try to do this with columns. Instead, use the "2 pages per sheet" option under "Multiple pages" on the Margins tab of Page Setup. Better still, use "Book fold," which will create the pages in the correct order for a folded booklet. While there is a limit to the number of pages that can be effectively folded, there is no practical limit if you're going to cut them in half and bind them, and using "Book fold" will result in the cut pages being in the correct order; you need only cut the pages in half (with the center of the book facing up), then flip the pages on the left over the pages on the right.

When you use either "2 pages per sheet" or "Book fold," Word creates two logical pages on one physical sheet, and you can add page numbers, page borders, or any other page-level formatting just as you would for any other page.
