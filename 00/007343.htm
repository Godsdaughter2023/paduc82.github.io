Find specific text characters in a cell - Microsoft Community

arjay12345 say:

Find specific text characters in a cell

Hi.  I have a column of data and need to highlight fields that contain any non-alphanumeric characters (i.e. _+{}:@~<>?, etc).  What's the easiest way of doing this?

The following works but surely there's a more cleaner and condensed version, as there are approx 30 chars I am interested in?

=OR(SEARCH("+",A5),SEARCH("=",A5),SEARCH("=",A5),SEARCH("=",A5),SEARCH("=",A5)

Ideally, I like to find and replace any of these chars with a space

Thanks, Arjay

ronrosenfeld say:

Arjay,

Glad to help.  Thanks for the feedback.

=OR(ISNUMBER(FIND({"_","+","{","}",":","@","~","<",">","?"},A1)))

We'll pick this apart by looking at the FIND function first.  FIND works the same as SEARCH except for case sensitivity.

Instead of looking at each unwanted character individually, we put them all into an array constant:  {"_","+",",",  etc

FIND then looks to see if each of those characters is in A1, and returns either a #VALUE! error if the character is not present, or a number representing the position of the "found" character.

ISNUMBER will then return an array to TRUE or FALSE 's depending on if the unwanted character was found.

OR will return TRUE if any element in the array is TRUE.

If you step through, using the EVALUATE FORMULA wizard, you can also see what is going on.

Note that with the Replace UDF, the characters to add to the string are those you want to keep; all others will be replaced with the <space>.

Best,

  --Ron

To use an email message template, use the following steps:

Select New Items > More Items > Choose Form.

In the Choose Form dialog box, in Look In, click User Templates in File System.

The default templates folder is opened. The folder location (in Windows 7 and later operating systems) is c:\users\username\appdata\roaming\microsoft\templates. If your template is saved in a different folder, click Browse, and then select the template.

Select the template, and then click Open.

Make any additions or revisions to the recipients in the To, Cc, or Bcc boxes and any changes to the subject and message body.
