Is There Windows Mobile 6.5 Disk Defragmenter? - Microsoft Community

dhspncr527 say:

Is There Windows Mobile 6.5 Disk Defragmenter?

Is There Windows Mobile 6.5 Disk Defragmenter? Thanks.
david a matson say:

Greetings dhspncr527.  Welcome to the Windows Phone forums!

Windows Mobile is generally used on devices without Hard Disks, so one of the biggest reasons for defragmenting is rendered moot.   (when you had a physical read-head that moved across the platter back and forth to find the 100 or so pieces of a file caused the entire system to slow down; now that SD cards are replacement for hard drives, there is no physical moving read/write head to slow the system down)

Additionally, memory management sub-routines within the OS automatically defragment running processes in memory as best they can, so it's best not to interfere with that.  You'll likely just cause more damage than improve device efficiency.

If you have a SD card that is mostly full and want to defrag the contents of the card, then you're better off just making a folder on your PC desktop and then just copy the contents of the SD Card over to the folder.   Then delete everything on the SD Card, then simply copy the files back from the folder to the card.   This will have the same effect as defragging, except it will only take a fraction of the time, since you are not asking a 3rd party program to create thousands of temporary files during each file copy.

rich007a say:

OK, I think I've got it.

The file to be downloaded should be cleared from the cache first by using the DeleteUrlCacheEntry function:

 'Declaration for deleting a file from the cache:
Private Declare Function DeleteUrlCacheEntry Lib "wininet.dll" Alias _
    "DeleteUrlCacheEntryA" (ByVal lpszUrlName As String) As Long

.....

 'Function to download file from URL:
Function DownloadFileFromWeb(URL As String, SavePath As String) As Boolean
'Returns True if download was successful.
'
    Dim Ret As Long
   
     'First delete the file from cache:
    DeleteUrlCacheEntry URL
   
     'Download the file and return result:
    DownloadFileFromWeb = False
    Ret = URLDownloadToFile(0, URL, SavePath, 0, 0)
    If Ret = 0 Then DownloadFileFromWeb = True
End Function
