Error 403 Forbidden means that you are not authorized to view information on that particular web server. Internet Explorer is able to connect to the website, but Internet Explorer does not have permission to display the webpage.

 

Follow the suggestions listed below for a possible fix:

 

Method 1: First try to change the proxy server settings.

 

Refer this link for instructions:

http://windows.microsoft.com/en-US/windows7/Change-proxy-server-settings-in-Internet-Explorer

 

Method 2: If that doesn’t fix the problem, then you may try these steps and check if it fixes the problem:

 

Open an elevated command prompt.

 

a.       Go to start and type in cmd in the search box.

b.      Right click on the command prompt and click run as administrator.

 

Now reset your TCP/IP stack.

Type the following commands in the command prompt, each followed by pressing enter.

 

ipconfig /flushdns

nbtstat -R

nbtstat -RR

netsh int reset all

netsh int ip reset

netsh winsock reset

Exit.

 

Restart the computer.

 

Now open Internet Explorer and check if you can access the website.

    hansv mvp say:

    Here is a macro you can use:

     

    Sub Insert3Rows()
        Dim r As Long
        Dim m As Long
        m = Cells(Rows.Count, 1).End(xlUp).Row
        Application.ScreenUpdating = False
        For r = m To 2 Step -1
            Cells(r, 1).Resize(3, 1).EntireRow.Insert
        Next r
        Application.ScreenUpdating = True
    End Sub

     

    This assumes that column A will be filled.
