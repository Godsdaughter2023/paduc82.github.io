How to delete a created number list? - Microsoft Community

hongli05 say:

How to delete a created number list?

I created a new number list but now i want to delete it. How can i do that?
john mcghie say:
Hi Hongli:

Basically, you can't; but there is no need to do so.

Word stores all formatting, including numbering, in a database at the end of the document.  (It's a table, like an Excel spreadsheet).  This database is accessed sequentially, and each numbered list is a row.  If you simply remove a row, the document corrupts and Word can't open it.

However:  You can remove the list from "Display".  To do that, go to Format>Bullets and Numbering, select your list, and click "Reset".  That resets the selected display position to the default for that position.  The list remains in the document, the document continues to open properly, and any text that list was applied to continues to have that numbering.  You just don't have to look at the list.

If it's a list STYLE you have created, go to Format> Bullets and Numbering>List Styles and select it.  You can click Delete there.

If you wish to use VBA, you can accomplish the same things, and in VBA, you can delete lists if you are careful.  But you have to do it exactly correctly, or you corrupt the document.

Of course, I could explain why all this is so: post back if you want to know.  But that's all you need to know to work with the mechanism.

Hope this helps

ray gao msft say:

Hi Jessica_V,

 

I would suggest you follow the steps below. (on each user’s computer)

  1. Stop the current syncing and exit all Office applications including OneDrive for Business.
  2. Removing all the stored credentials in the credentials manager (Control Panel > User Accounts > Credential Manager > Windows Credentials). (NOTE: This will remove your stored passwords.)
  3. Run regedit as administrator.
  4. From Registry Editor, browse to:
    HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\Common\Identity
  5. Delete the Identity folder.
  6. Open an Office application like Word and sign in with the user’s Office 365 account.
  7. Re-establish the sync.
