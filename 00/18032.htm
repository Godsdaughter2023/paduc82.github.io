You can use the COUNTIFS function with dates - not sure what you are looking for exactly, but this would show how many employees were hired in 2012:

=COUNTIFS(Q:Q,">="&DATE(2012,1,1),Q:Q,"<="&DATE(2012,12,31))

You could also use formulas like this to do the same thing:

=SUMPRODUCT((YEAR(Q:Q)=2012)*1)

You could also use a Pivot Table, with the hire date as the row and data field, grouping the row fields by year, and you would get a report of the counts for all years.

Create your message and then select Menu > Copy/Paste > Select All > Copy

Then while you are in a call:

select Menu > Open Contact 
select Menu > Send Text Message 
select Menu > Copy/Paste > Paste

On a windows mobile phone (with a touch screen) :

Then while you are in a call:

select Menu > Send Text Message
