I am selecting the word I want to change her line spacing value and right click and tapping into Paragraph and changing the line spacing value, but this is changing the value of the whole paragraph.

Best Regards.

rohn007 say:

Hussein:

Suzanne's reply is the only work-around that will work for you. 

Even though the font size is set the same, each different "font" has has different attributes defined by the creator of the font. As you have seen, one of those attributes is vertical space used by the fonts.  Automatic line spacing will set spacing for the "tallest" of the 3 fonts used in the paragraph, making the spacing look too wide for the other 2 fonts. If you use exact spacing, you can set a value that is slightly wide but works for the majority of the paragraph, but does not cause the text in the tallest font to run into the lines above or below.

Good luck.

suzanne s. barnhill say:
As I said, the only way to have uniform spacing is to set an "Exactly" amount instead of using Multiple spacing. It will require some trial and error to find out the exact line spacing (in points) that will be equivalent the 1.5 spacing in your base font. Then use that for all the text.

t. valko say:

 

I want the cells A:P to have bold font if the cell in column C ="0"

Thoughout the database-- so going row be row, from row 2 to to about row 1000.


Ok, let's assume the range of interest is A2:P1000.

Select the *entire* range A2:P1000 starting from cell A2. Cell A2 will be the
active cell. The active cell is the one cell in the selected range that is
not shaded. The formula will be relative to the active cell.


• Goto the Home tab>Styles>Conditional Formatting>Manage rules>New rule>
• Use a formula to determine which cells to format
• Enter this formula in the box below:
• =AND($C2<>"",$C2=0)
• Click the Format button
• Select the Font tab
• In Font style, select BOLD
• OK out
