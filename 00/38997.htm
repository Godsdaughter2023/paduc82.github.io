Failed to launch UI Automation Server - Dragon NaturallySpeaking 12 - Microsoft Community

eanderson2 say:

Failed to launch UI Automation Server - Dragon NaturallySpeaking 12 & Windows 10

After the latest Windows 10 update, I started getting the message "Failed to Launch UI Automation Server" whenever I tried to launch Dragon NaturallySpeaking 12. After a bunch of research (and a ton of frustration), I was finally able to get the issue resolved. Below is what I had to do, in case there are others facing the same issue.

Put simply:

1. Turn on User Account Control

2. Run the DNS setup.exe file on your installation CD/DVD and select "Repair"

Steps to turn on User Account Control:

1. Click "Start"

2. Type "Control Panel" and open Control Panel

3. Type "UAC" in the search box in the top right corner of the Control Panel window

4. Click "Change User Account Control settings" under Security and Maintenance

5. If the slider is all the way at the bottom, slide it up to turn User Account Control back on

6. Restart the computer, and see if Dragon will open correctly.

Note: Dragon  still did not work for me after turning UAC back on. For some people, this will work. For me, I had to run a repair using my installation media. For this, do the following:

1. Insert your DNS installation disk and look for the file named "setup.exe"

2. Run "setup.exe" and select "Repair"

3. After Dragon has finished repairing, restart your computer.

4. Dragon should now be working again

At this point, I turned User Account Control back off because I get tired of the constant confirmation boxes that pop up. Of course, if I start getting the error message again, I'll just be forced to turn it back on and deal with it.

Anyway, with this recent update, I'm sure I'm not the only one that faced this problem. Hopefully this information will help someone  get their issue resolved without going through all of the frustration that I went through.

Cheers!
