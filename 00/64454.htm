¡firedog say:
I just want to navigated to the folder where the incoming .doc files are stored

When you receive a mail message with an attachment, the 'attachment' is just a block of code at the end of the message you can read. At the top of the block will be a Content-Type header line telling the program what the block of code contains. For a Word document, this will often be application/msword. You can see this for yourself if you select the message in the message list and press Ctrl-F3. This will reveal the whole of the message source. Attachments will be at the bottom; if you start at the bottom and scroll up until you come to a blank line, you'll see the header lines for the attachment.

The attachment doesn't become a file until you ask the mail client program to open it or save it as a file. The program then converts the code into the appropriate format - in this case a Word document - and produces a file. If you selected Save, the file will have the name you choose and be stored in the location you choose. To do this in Windows Live Mail, right-click on the attachment icon in the message header area and select Save as. You can either accept the filename suggested if there is one or enter a new name. Once saved, the attachment is just another Word document like those you produce yourself and will be treated as such by any program you use to open it.

If you receive a message like this in a Hotmail-type account and read it in your web browser, you will be given the option to view the attached document online, where you will be able to use an online version of Word to read, edit and save the document. Read more here: Download a Word, PowerPoint, or Excel attachment in Hotmail

    a. user say:

    @ Smittychat [Community Moderator] You did not answer my question!!!  


    I believe I did  but if you truly want an answer to this question:

     

    What's the best thing to do, To make Windows Free & Clean...

     

     

    Manually clean the registry:

    1. Click on the "Start" button, then select "Run...".

    2. Type regedit inside of the text box.
    3. Press "Enter" or click on "Ok".

    Backup Windows Registry

    1. Click on the "File" menu and select "Export".

    2. Select "All" in the "Export range" panel.
    3. Choose a location for the backup and then enter a name.
    4. Click on "Save".

    For applications that have been uninstalled

    1. Expand the "HKEY_CURRENT_USER" key (looks like a folder) by clicking on the plus (+) sign next to it.

    2. Expand the "Software" key.
    3. Look for keys that contain either the application's name, or more commonly, the application's developer (company name).
    4. Highlight a key for the application.
    5. Press "Del" to delete it.

    You can search for applications by their name, executable/name and or folder name

    1. Press "Ctrl" and "f" to open a find dialog.

    2. Enter the text used to search for the application's listings.
    3. Click on "Ok" to search. The key or value should be highlighted when found.
    4. With the proper key or value highlighted, press "Del" to delete it.
    5. Press "F3" to find the next result and repeat.

     

    Remove Unwanted Startup Items

    1. Expand keys (as done previously) to get to the following location:  HKey_Local_Machine\ Software\ Microsoft\ Windows\ Current Version

    2. Highlight the "Run" key.
    3. Look for values inside of the left pane. These will be shortcuts to executable files.
    4. Highlight a value to remove. If you're not familiar with these, or are unable to guess which ones correlate to which applications, Bing it, Google for it or use a process search engine such as Process Library
    5. Press "Del" to delete it. To select multiple entries to delete, hold down "Shift" or "Ctrl" while clicking.
    6. Repeat this process for the same key path inside of the HKEY_CURRENT_USER location as well. If an application installs for "all users", it will put these startup items inside of the the local machine folder. For current user or single user software installs, they will be put into the current user folder.
