The newer versions of Office (365/2016/2013) don't require product keys for second time installation. When you install this first time and activate, the product key gets stored in digital format on Microsoft activation servers. For any subsequent downloads and activation, you need to login here and install and activate.

You will have to first deactivate Office license which was installed on old computer otherwise it will be counted as a license. Hence, deactivate first and then install and activate.

I would first stop Windows Update from running:

You can create a stop and start script for Windows Update.

Create a Notepad file with each of the following and save them with the .bat extension.

stop-updates.bat:

sc config wuauserv start= disabled
net stop wuauserv

sc config bits start= disabled
net stop bits

sc config dosvc start= disabled
net stop dosvc

pause

start-updates.bat:

sc config wuauserv start= auto
net start wuauserv

sc config bits start= auto
net start bits

sc config dosvc start= auto
net start dosvc

pause

Now you can simply right click on stop-updates.bat and select “Run as Administrator”
