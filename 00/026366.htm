gord dibben say: 
This macro from Ron de Bruin will insert a pagebreak at 15 rows
To improve it you could add an InputBox to ask for RW number
Sub Insert_PageBreaks() 
    Dim Lastrow As Long 
    Dim Row_Index As Long 
    Dim RW As Long 
    'How many rows do you want between each page break 
    RW = 15
    
    With ActiveSheet 
        'Remove all PageBreaks 
        .ResetAllPageBreaks 
        
        'Search for the last row with data in Column A 
        Lastrow = .Cells(Rows.Count, "A").End(xlUp).Row 
        
        For Row_Index = RW + 1 To Lastrow Step RW 
            .HPageBreaks.Add Before:=.Cells(Row_Index, 1) 
        Next 
    End With 
End Sub

Kia ora,

the Calendar View sheet has conditional formatting set in cells C6 to AR17. There's a conditional format for each leave type. You need to add a new rule for each new leave type. You can copy the formula from an existing rule and just change what it refers to. It is easy once you see how the formula works. 

=COUNTIFS(lstEmpNames,valSelEmployee,lstSdates,"<="&C6,lstEDates,">="&C6,lstHTypes,'Leave Types'!$B$4)>0 for blue Annual Leave

=COUNTIFS(lstEmpNames,valSelEmployee,lstSdates,"<="&C6,lstEDates,">="&C6,lstHTypes,'Leave Types'!$B$5)>0 for orange bereavement

=COUNTIFS(lstEmpNames,valSelEmployee,lstSdates,"<="&C6,lstEDates,">="&C6,lstHTypes,'Leave Types'!$B$6)>0 for green Lieu

See how the only difference between the formulas is the cell it compares with on the Leave Types sheet. 

Copy one of the formulas, then create a new rule with a formula and paste in the formula you copied and adjust the row number in the leave types reference. Click the format button and select the colour you want, then confirm all dialogs. Change the color in the statistics box to the respective one you used in the conditional formatting.

  • Start Windows AND Office application in safe mode

  • uninstall from control Panel > Program features

ken yung say: Thanks for answering! But since this document is just a part of the entire document, we must use Heading 3 as the page number so as to align with other files... Any idea of how to make the TOC display the correct number??

doug robbins - mvp office apps & services (word) say:
Thanks for answering! But since this document is just a part of the entire document, we must use Heading 3 as the page number so as to align with other files... Any idea of how to make the TOC display the correct number??


I do not believe that it is possible for the reason given in my response - the Heading 2 appearing before the Heading 3 that is the trigger for the new chapter page numbering.

Consider the situation where you have a whole page or more of text after a Heading 1 or Heading 2 style paragraph before a Heading 3 styled paragraph.

Or, if you never have a Heading 3 style in the document.  In that case, the number on the page will not even show any heading level numbering.

The only way around the issue would be to select the Table of Contents and then use Ctrl+Shift+F9 to convert it to ordinary text and then edit the page numbers.

That should be done on a copy of the file so that if subsequent edits are required, the Table of Contents can be updated.  However, once you do that, you will no longer be able to use Ctrl+left click to go to the related part of the document.  That however may not be an issue if the document will ultimately be converted to pdf.
