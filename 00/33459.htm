      I found how to fix it:

      1. Go to C:\ProgramData\Microsoft\ and delete Search folder.

      2. Open Windows + R > regedit    Go to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Search\

          Choose SetupCompletedSuccessfully and change value from 0 to 1.

      3. Go to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Search\FileChangeClientConfigs\

          and create new String Value with name: {A010C594-BAB6-4690-952C-28A0C0365307} and     value: ScopeToMonitor.

      It helped me!
