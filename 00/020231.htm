beth melton say:

If you're using an Excel number format that uses _) at the end, such as the default Accounting format, then it will add an extra space at the end of the number if using a paste link in Word. It’s added to offset the value one character from the right to accommodate parentheses around negative numbers to align the decimals. Try changing the number format in Excel to see if it resolves the issue.

~Beth Melton Microsoft Office MVP

How did you select all the content? You must uncheck the Widows & Orphans box in Paragraph panel for each style you use.

1. Open Word, click the Home tab so that it is the active tab in Word.

2. Click the small square with an arrow on the bottom right corner of the Paragraph group on the Ribbon to open the options dialog box.
