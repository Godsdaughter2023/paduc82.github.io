no.compromise say:

The dumps are varied enough to suggest that some driver is not behavior properly and perhaps causing memory corruption or trying dynamic code execution while the code to be executed is paged out - there is no smoking gun sticking out in any of the dumps.

You do have a few stale drivers, some of which have been involved in similar behavior before.

 

RimSerial_AMD64.sys 1/9/2007  (RIM / Blackberry?)

PxHlpa64.sys 2/2/2007  (Sonic software)

Rt64win7.sys 2/26/2009 (Realtec NIC driver)

gdrv.sys 3/12/2009  (Gigabyte Dynamic Energy Saver Advanced program)

I'd suggest updating the above drivers and then checking the behavior.  If no update is available, remove the driver or the software that it came with.

There should be an update available for the Realtek NIC driver - that version has been involved in a number of blue-screen scenarios with Windows 7.

gdrv.sys probably needs to be removed by uninstalling the Gigabyte software on your system - there is evidence that gdrv.sys has also been involved in blue-screen scenarios, and that the Gigabyte software will recreate the driver file gdrv.sys if it is removed.  So if you don't know which of potentially multiple pieces of Gigabyte software may be using the driver, the suggestion would be to uninstall all of them.

Once those four drivers are out of the way or updated, then check the issue, and if it persists strategy can be adjusted, probably focusing on the AV software installed (McAfee) and other third-party drivers.

Hi Cathy,

Receiving an Error 0x80070103 means that Windows Update tries to install a driver that is already installed in your system or Windows Update tries to install/update a driver to an unsuitable version of that driver.

To resolve your concern, you can contact your device manufacturer for the most recent available driver or hide the update so that it is not offered again.

To hide the update, follow the steps below:

  1.  Click Start, click All Programs, and then click Windows Update.
  2.  Click Check for updates.
  3.  After the scan for updates is complete, click the Optional updates are available link.
  4.  In the Select the updates you want to install section, right-click the update that you want to hide, and then clickHide update.
  5.  Click OK.
  6.  Click Install Updates to install any additional updates that are selected. If no other updates are selected, close Windows Update.
