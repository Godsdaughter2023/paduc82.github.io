gord dibben say:

You can use a macro to save each sheet as a seperate PDF file.

Edit the "C:\Gordstuff" to a folder of your choosing

The original file must be an Excel workbook.

Sub Make_Sheet_PDF()
'produce one pdf per sheet
Dim w As Worksheet
    Application.ScreenUpdating = False
    Application.DisplayAlerts = False
    For Each w In ActiveWorkbook.Worksheets
        w.Copy
        With ActiveWorkbook
        .ExportAsFixedFormat Type:=xlTypePDF, _
             FileName:="C:\Gordstuff" _
                  & "\" & w.Name & ".pdf"
            .Close
        End With
    Next w
    Application.DisplayAlerts = True
    Application.ScreenUpdating = True
End Sub

Customizing file folder label template (3-tab) to put in a - Microsoft Community

jsh_nh say:

Customizing file folder label template (3-tab) to put in a custom color strip using Word

I want to find out how to customize a Word file label template (used for 3-cut manila file folders) and put my own custom color strip on top.  I have numerous files requiring more colors (or color combinations) than the standard colored labels offered on the office supply market.

I'd also like to know if I can get color strip to cover entire top of label.

Any help is greatly appreciated.

suzanne s. barnhill say:

Yes, you can do this. Let's take Avery 2181 labels as an example.

  1. On the Mailings tab, in the Create group, click Labels.
  2. On the Labels tab of the Envelopes and Labels dialog, click Options...
  3. For "Label vendors" choose "Avery US Letter."
  4. Under "Product number," scroll down to 2181 and select it.
  5. Click OK.
  6. Back on the Labels tab, click New Document. You'll get a page that looks like this (if you have table gridlines and nonprinting characters displayed):
  7. Click the table handle to select the entire table.
  8. On the contextual Table Tools | Layout tab, in the Merge group, click Split Cells.
  9. In the Split Cells dialog, set the number of columns to 1 and the number of rows to 12. This will split each of your file folder labels into two.
  10. You can now fill the top half of each label with color using the Shading dropdown in the Table Styles group on the Table Tools | Design tab:
  11. You probably won't want the colored portion to take up half the label space, so you can adjust the size of the cells. Each row is 0.33" high, so on the Row tab of the Table Properties, I set the first row to 0.22", the next to 0.44", and continued through the table alternating heights. The heights are set to "At least," so you will probably need to set them to "Exactly" to prevent the text cells from expanding.

This is just an example; if you're using some other label stock, you'll need to adjust accordingly. You may want to make all the tabs the same color. Whatever you do, you can save the document for future use, and you can change the colors on any tab anytime you use it.
