suzanne s. barnhill say:

You can restore the default formatting by deleting the current styles.

  1. Click in a List Bullet paragraph.
  2. In the Styles pane, find the List Bullet style, right-click (or click the down arrow), and choose Delete List Bullet...
  3. You'll be asked, "Do you want to delete style List Bullet from the document?" Click Yes.
  4. The List Bullet paragraph will become Normal.
  5. Reapply the List Bullet style.
  6. Repeat for List Bullet 2 and List Bullet 3.

Unfortunately, this process means that you will have to reapply the style to all the List Bullet paragraphs in the document. A better approach is to modify each style manually. Note that you cannot set indents for numbered/bulleted styles in the Paragraph dialog, and you cannot set numbering/bullets for a style through the Modify Style dialog. Instead, you must use the Define New Multilevel List dialog to set up the numbering and indents, with each level linked to a style.

The easiest way to start is to place the insertion point in a List Bullet paragraph and proceed as follows:

  1. On the Home tab, in the Paragraph group, click the arrow beside Multilevel List.
  2. From the gallery, select the preview that shows assorted bullets. This will apply a bullet to the List Bullet paragraph.
  3. Open the Multilevel List menu again and choose Define New Multilevel List.
  4. Click More>> to expand the dialog.
  5. If you like the bullets used, you can leave them alone. If not, select each level in turn and choose a small round bullet instead.
  6. With Level 1 selected, in the Link level to style dropdown, choose List Bullet.
  7. Repeat for Level 2 (List Bullet 2) and Level 3 (List Bullet 3) and as many more levels as you want.
  8. Click OK. This will firmly link the List Bullet styles to the bullets and indents you have selected.

petrula say:

After carefully reading your suggestion, I discovered an interesting way that seems to work so one does not have to reapply the style to all the List Bullet paragraphs in the document.

 

Create a new Style as follows:

Name: List Bullet Temporary

Style type: Paragraph

Style type based on: Normal

 

Bring up Style panel.

Right mouse on List Bullet.

Select All Instance(s).

With all the instances selected, you can click on the List Bullet Temporary style.

Now those instances have the List Bullet Temporary style.

Right mouse on List Bullet.

Select "Delete List Bullet..." . Note: It may say "Revert to List...".

This resets it back to the built-in factory List Bullet style.

The previous instances should still be selected so click on the List Bullet style.

Your original List Bullet instances should now have the factory List Bullet style.

Repeat this process for all List Bullet related build-in styles.

*****

Unfortunately, this did not work. Specifically, only "Use Destination" retains the styles when you paste. The other choices like Merge or Use Source do not preserve the styles. Apparently, during the paste, the List Bullet was modified for some unclear reason. This could be due to the nature of the corruption to the file.

marco - msft say: Hi, jlmwrite (and other folks on the thread with the same issue).

There are certain Windows Mobile phones which use custom SMS applications which store data in a format and store not expected by the service. My Phone service will not sync SMS from these phones.  For example if you use My Phone service on Palm Treo 800 on Windows Mobile 6.1, Palm 750 on Windows Mobile 6.0, you will not see your SMS synced to the online account even if that setting is enabled in the application.

Video files are a combination of 2 formats: the container format, which is AVI in your case, and the CODEC, which is the algorithm that compresses and replays the video in the container. PowerPoint has told you the problem: you don't have a 64-bit CODEC that can play file.

The simplest solution is to convert the video to a format that PowerPoint can understand. Zamzar.com site may be able to convert your file for free.
