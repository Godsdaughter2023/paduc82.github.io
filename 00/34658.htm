nicksdbomb say:

Working cable "USB 3.0 Cable E119932 AWM 20276 80C 30V VW-1 COPARTNER" 

Working solution found on a self built computer using a MSI Z370 SLI Plus motherboard latest BIOS, Windows 10 64bit. 

Problem: Not all USB 3.0 cables work on a USB 3.1 port.

Using the same USB 3.0 A to B cable in the USB 2.0 port the SansDigital external RAID is seen and all the folders and files in them are present. Plugging the same cable into the USB 3.1 port the RAID is not seen. When plugged into the USB 3.1 port with the USB 3.0 A to B cable after about 5 minutes, a notice pops up saying that the USB device is not recognized.

Printed on the blue cable that only works on the USB 2.0 port is:

 "USB 3.0 Cable E318233 AWM 20276 80C 30V WW-1 KAIBAO"

I have tested another USB 3.0 A to B cable that works in both USB 2.0 and USB 3.1 ports. Printed on the black cable that works on both USB 2 and USB 3.1 ports is:

"USB 3.0 Cable E119932 AWM 20276 80C 30V VW-1 COPARTNER" 

Conclusion:

THE RAID is working. The MSI Z370 SLI Plus USB 2.0 port is working. The MSI Z370 SLI Plus USB 3.1 port is working on some cables and not working on others in a backward compatible mode that supports a USB 3.0 cable and USB 3.0 device.
