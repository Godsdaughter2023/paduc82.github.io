Windows Basic Theme: more than baby-blue! - Microsoft Community

nathan warawa say:

Windows Basic Theme: more than baby-blue!

Currently, the Windows 7 “Basic” theme is limited to one theme only: a baby-blue theme. The reasons posted for a single Basic theme are listed as two fold:

(1) A product line differentiator between Starter and Basic, and more advanced versions that support full Aero.

 

(2) For battery life and performance benefits on netbooks that the feature-limited versions of Windows are usually found on.

I find these reasons fair and logical.  As an extension of that reasoning, I find it logical to include a more robust, customizable Basic theme in the more advanced versions of Windows:

(1) As a product line differentiator: those who pay more for, say, Home Premium, and want a more attractive theme, may simply want a classier version of the Basic theme (I am one of those, for the reasons expressed in “(2)”)

            (2) For battery life and performance benefits, while maintaining the more advanced feature set of a more                         expensive Windows (such as XP mode).

The problem with Aero, and in particular, “Desktop Composition”, is game compatibility. It seems almost any new game I test, Windows will within a few minutes of play bring up a dialog box suggest disabling desktop composition. It has become so frequent that by default I now just set the game exe’s properties to disable desktop composition upon install. Even that workaround isn’t foolproof, though, as often desktop composition does not restore upon game exit, and I am stuck with the, in my opinion, hideous, baby-blue theme, until restart.

My suggested fix is one or both of the following two additions:

(1)   Add a color selector and intensity slider for the Basic theme like there currently is for the Aero themes

(2)   Create a handful of preset Basic themes that can accessed via the same “Get more theme online” link under the Aero themes.

* Remove any discs from the disc tray.

DMZ

Login Information

  • Gateway - 192.168.1.1 or 192.168.254.254
  • Username - admin
  • Password – admin

DMZ

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.254.254 into the address bar. (Or click the link) When prompted enter admin for the user name and admin for the password.
  3. From the menu on the left, select Advanced Setup.
  4. Select NAT from the submenu on the left.
  5. Select DMZ Host on the next submenu.
  6. Enter the IP Address of the Xbox.
  7. Select Save/Apply.

Disable DMZ

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.254.254 into the address bar. (Or click the link) When prompted enter admin for the user name and admin for the password.
  3. From the menu on the left, select Advanced Setup.
  4. Select NAT from the submenu on the left.
  5. Select DMZ Host on the next submenu.
  6. Clear out the box.
  7. Select Save/Apply.

      My biggest issue is that my computer is constantly running the fan because it has high CPU and disk usage. This is killing my battery life, slowing down my computer, and making it constantly burn my hand to touch the back of the computer.

      There appears to be no way to turn off wsappx. It continues to do the same thing even after a Clean Boot. Please help me to make it stop. I don't understand how a computer with lower specs would even be able to run like this.

      rogarc say:

      I believe I have solved my own problem. I had selected to defer Windows Updates because the constant updates and random reboots and computer slow downs that plague me whenever my computer downloads an update were getting in the way of my work flow (for example, in the middle of work I've had it reboot and take 30-60 minutes to get done updating and cost me progress in unsaved work).

      My solution:

      1) Settings --> Update and Security --> Advanced Options

      2) Uncheck "Defer Feature Updates"

      3) go back to Update and Security page

      4) Search for and install updates. Then keep repeating this until all of the firmware and updates are installed.

      5) Make sure to do a final reboot.

      When all that was done my computer seems better using just 2% at idle and not burning up anymore. After the update, I had a couple of unwanted apps I had to uninstall like "Getting Started" again.

      I hope this helps someone else. Until my next problem!
