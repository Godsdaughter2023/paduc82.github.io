Use a Directory type mail merge main document in which you insert a one row table and insert the required mergefields into the cells of that table.

rich michaels - mvp (greatcirclelearning.com) say:

Hi Sam,

If something was copied into the document from another document, without the styles being cleared, that could have impact.

If you haven't already done this... select everything in the document (Command + A) and then take a look at the spelling checkbox on the language dialog.

If there's a dash and not a checkmark, then something in the document is marked to not check spelling.

If the last empty paragraph in the document is marked to not check spelling, the above test will not detect the problem. However, as soon as you type something into that empty paragraph the setting will apply and the text won't be spellchecked.

Hope this helps uncover the problem
