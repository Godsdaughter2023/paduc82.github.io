The reason Rodney has recommended trying to open the file on a PC running Windows is because the encryption methods being used on the Mac are different than on a PC with Windows... at least they have been in the past, and I have not been informed if Microsoft has ever changed it.

charles kenyon say:

See Where can Building Blocks be stored?

To have AutoText available in any document, it must be stored in a Global Template. To share it, it must be in a template other than the Normal template.

Once AutoText is inserted in a document, it is no longer connected to the AutoText entry. You can use an AutoText Field or an AutoTextList Field to maintain the connection to the AutoText entry.

To have it update in documents when changed centrally, it must be in those documents using an AutoText field and that field must be updated. I use AutoText fields in my templates. They update when a new document is created from the template. Those templates are in the Workgroup Templates Folder.
