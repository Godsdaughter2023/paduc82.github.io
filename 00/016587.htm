Your QUESTIONS:

1.Will the legacy license (7/8) remain valid after upgrading to Windows 10? People may have to fall back due to driver or program compatibility issues.

REPLY - No the license for windows 10 will be consumed (replace) windows 7.sp1 or window 8.1 license.

To get the windows 7.sp1 or 8.1 key back you will need to fully reinstall and phone activate the pc with Microsoft (run command slui.exe 4).

"If you upgrade from a OEM or retail version of Windows 7 or Windows 8/8.1 to the free Windows 10 upgrade this summer, the license is consumed into it. Because the free upgrade is derived from the base qualifying license, Windows 10 will carry that licensing too.


If you upgrade from a retail version, it carries the rights of a retail version.
If you upgrade from a OEM version, it carries the rights of a OEM version.

Full version (Retail):

- Includes transfer rights to another computer.
- Doesn't require a previous qualifying version of Windows.
- Expensive

Upgrade version (Retail):

- Includes transfer rights to another computer.
- require a previous qualifying version of Windows.
- Expensive, but cheaper than full version

OEM :

OEM versions of Windows are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system"


2.Are Windows 10 licenses user or machine based? That is, if I build a new machine in 2 years will I be able to transfer the license to the new machine?

REPLY - OEM licenses (codes & OEM bios embedded) are machine based and motherboard details are saved to MS activation servers. Yes you can transfer RETAIL licenses to a new machine and are retail licence based. But is still linked to

your original motherboard so for retail licenses you may need to phone activate it.

"What happens if I change my motherboard?
As it pertains to the OEM licenses this will invalidate the Windows 10 upgrade license because it will no longer have a previous base qualifying license which is required for the free upgrade. You will then have to purchase a full

retail Windows 10 license. If the base qualifying license (Windows 7 or Windows 8.1) was a full retail version, then yes, you can transfer it."

From the end user license agreement:

"UPGRADES. To use upgrade software, you must first be licensed for the software that is eligible for the upgrade. Upon upgrade, this agreement takes the place of the agreement for the software you upgraded from. After you upgrade, you

may no longer use the software you upgraded from."

"TRANSFER TO ANOTHER COMPUTER. a. Software Other than Windows Anytime Upgrade. You may transfer the software and install it on another computer for your use. That computer becomes the licensed computer. You may not do so to share this

license between computers."
 
3.Can we create a dual boot configuration in which both Windows 7/8 and Windows 10 are on the same machine, active (one at a time), and receiving updates?

REPLY - The initial windows 10 upgrade free offer does not allow dual boot only an update within windows from windows 7.sp1 or windows 8.1. If you want to dual boot you need to purchase windows 10. If you activate windows 10 the license for windows 7 or 8 will eventually expire later as it checks the MS authorising server (monthly or few months later).

4.Will we need the original media and / or original key if we perform a clean install from an ISO or will Microsoft provide a new Windows 10 key (based on our current validated installation) that can be used during the installation?

REPLY -

Microsoft recently noted the following options available:

If things go wrong and you are unable to refresh or reset your device successfully, you can boot the device using recovery media and reset to the prior pristine state.

Source:  http://blogs.windows.com/bloggingwindows/2015/03/16/how-windows-10-achieves-its-compact-footprint/"

Either way you need to run setup.exe in windows not boot off the image. By default, Windows 10 preview builds are pre-keyed, meaning, you do not have to enter a product key and should not be prompted to enter one even

doug robbins - mvp office apps & services (word) say:
The protection resulting from a document opening in Protected View in the case of documents that are received as email as email attachments is a different form of protection to that applied via the Restrict Editing facility and the only way to disable it is via File>Options>Trust Center>Trust Center Settings>Protected View.  It is not possible to disable it or change that setting via VBA code.

jay freedman say:
Before opening the document in Word, use Windows Explorer to go to the folder where you stored the document. Right-click the file and open its Properties dialog. If there is an Unblock button in the dialog, click it. That will remove the file property that tells Windows and Word that the file came from another computer, so you can now open it in Word without Protected View.
