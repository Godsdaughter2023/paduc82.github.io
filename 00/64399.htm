justin dunn say:

Just wanted to point out that deleting Known Folder Definitions from the registry is a bad idea.

Known Folders are available to any application that wants to use them so it's possible that in the future a third-party application may decide to use that folder "Camera Roll" or "Saved Photos" just as any application can use the "Documents" or "Pictures" folders and there maybe unexpected behavior in the application when the Known Folder cannot be found.
There might even be unexpected problems from the Photos app if it can't find that folder.

Also if you have multiple users on the computer then none of the users can use the folder.

One nice thing about Known Folder is that they can usually be redirected to a different location.

So you could just tell Windows you want to store "Camera Roll" in the location C:\temp\CameraRoll for example and then any application wanting the "Camera Roll" folder will find it and be happy and you won't see the folder in the regular location.

To change the location of a Known Folder like "Camera Roll" you can use a couple different ways:

1. Right click on the folder, select Properties, go to the location tab and then can select a new location for that Known Folder.

2. Just drag the folder to another location and it will become the new location of that folder.
