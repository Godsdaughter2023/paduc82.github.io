Rename the false advertised 'Tablet mode' (build 10159)

As a 8" Windows 8.1 tablet user I recommend to rename the 'Tablet mode' to 'Touch PC mode' because it is false advertised now. It does not have anything with excellent UI optimization for tablets introduced in Windows 8.1 modern (aka Metro) interface. I fully agree with arguments posted here. The word 'tablet' is associated with usually a 7" to 10" size device that has touch screen only and no keyboard.

There are also touch display equipped 11" to 15" laptops or 2-in-1 devices with detachable keyboard. Current 'tablet mode' might work for them but definitely not for small tablets without keyboard mentioned above.
