andreas killer say:
This will fail if the sheet contains a comment, not every shape has a TextFrame2 property.

The best way is to use the Type property of the shape object to get the class of the object inside that shape. After we know the type, we can get closer, depending on the type of the object.

Andreas.

Option Explicit
Option Compare Text

Sub Test()
  Dim S As Shape
  Dim Contents As String
  For Each S In ActiveSheet.Shapes
    Select Case S.Type
      'Comment
      Case msoComment
        Contents = S.TextFrame.Characters.Text
      'Form control label
      Case msoFormControl
        Select Case S.FormControlType
          Case xlLabel
            Contents = S.TextFrame.Characters.Text
        End Select
      'ActiveX controls
      Case msoOLEControlObject
        Select Case S.OLEFormat.progID
          Case "Forms.Label.1"
            Contents = S.OLEFormat.Object.Object.Caption
          Case "Forms.TextBox.1"
            Contents = S.OLEFormat.Object.Object.Text
        End Select
    End Select
  Next
End Sub

Tim S08

in your criteria, you need to use two cells.  You put any text string in the first cell of the criteria that is not a column label.  In the second cell put in your formula.  The formula would represent a formula against a single row in the data, the first row.

MyCriteria

=(YEAR(TODAY())-YEAR(A2))<=4

the A2 refers to the first cell in that column that has data.
