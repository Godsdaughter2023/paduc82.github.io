a. user say:

You need to get the firmware updater that can handle the .rom file type from the same place you got the firmware file.  From what I can see, they seem specific to the product being updated. http://www.fileinfo.com/extension/rom  If they don't have it, talk to their technical support people and ask them what program is required to install the firmware provided in .rom format and where you can get it.  Vista does not have the ability to read a .rom file type.  Some software has to be added to give it that capability.

Here's something I read: "Various hardware upgrades and emulators use the ROM file extension. A user needs a flash program usually available from the website of the motherboard manufacturer. The user must not flash BIOS unless the user knows that he has the image for his motherboard. An error or the computer might be disabled when the wrong BIOS image is flashed."  So be VERY, VERY careful here or you could completely disable your system.

I hope this helps.
