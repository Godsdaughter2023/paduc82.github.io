    Hi Scottish,

    I realize the inconvenience you are facing. Let me help you in answering the question.

    I like to inform you that system restore takes your PC back to an earlier point in time, called a system restore point. Restore points are generated when you install a new app, driver, or Windows update, and when you create a restore point manually. Restoring won’t affect your personal files, but it will remove apps, drivers, and updates installed after the restore point was made.

    1. Right-click (or press and hold) the Start button, and then select Control Panel.
    2. Search Control Panel for Recovery.
    3. Select Recovery > Open System Restore > Next.
    4. Choose the restore point related to the problematic app, driver, or update, and then select Next > Finish.

    Note: If you’re not seeing any restore points, it might be because system protection isn’t turned on. To check, go to the Control Panel, search for Recovery, and then select Recovery > Configure System Restore > Configure and make sure Turn on system protection is selected.  

    Also refer to Restore from a system restore point from the link below:

    http://windows.microsoft.com/en-us/windows-10/windows-10-recovery-options 

    Keep us posted on Windows related queries and we will be happy to assist you further.

    The only solution to FIX GameDVR Missing/Deleted - Microsoft Community

    [dano] say:

    The only solution to FIX GameDVR Missing/Deleted 'Captures' Folder, or change the location of the folder to another HDD!

    Follow these steps to fix a missing/deleted 'Captures' DVR folder, or to also change the location of the folder if you need to 
    because of low HDD space.

    1. Go to "Search" or "Cortana" and type "regedit"

    2. Run "regedit" as administrator and go to "Computer\HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\User Shell Folders"

    3. Right click on the {EDC0FE71-98D8-4F4A-B920-C8DC133CB165} key and then click the "modify" option to edit it.

    3.1. If the key doesn't exist right click inside the registry window and click "new" -> "expandable string value" -> and name it
    as follows {EDC0FE71-98D8-4F4A-B920-C8DC133CB165} then do STEP 3.

    4. Change the path to %USERPROFILE%\Videos\Captures (%USERPROFILE% stands for your windows username) if you want to restore the default location in the "Videos" folder.

    4.1. If the folder doesn't exist, or you've deleted it, go to C:\Users\%USERPROFILE%\Videos (for the default location) (%USERPROFILE% stands for your windows username) and create it using the "Create New Folder Option". 
    Then name the new folder "Captures" and then do STEP 4.

    5. If you only want to change the location of the folder, using the default method to change the location has a higher chance 
    of bugging it than fixing it so simply repeat STEP 4.1. with the following changes:

    5.1. Create a new folder in your bigger on size HDD (for example "E:")
    5.2. Follow STEP 3
    5.3. Follow STEP 4 by changing the location. (for example "E:\Captures" (to mark the new folder you've just created in the new HDD) 
    instead "C:\Users\<YourUsername>\Videos\Captures")

    6. Save and close the registry.
    7. Restart your PC.
