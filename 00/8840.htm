cliffgu_msft5984 say:

Hi Shashi,

For your questions, the answers are as follows.
1. When a user updates their profile photo through SharePoint Online profile, does this get copied over to the thumbnail photo attribute in Azure AD?
If you point to Microsoft Azure AD, the answer is no. We can’t get user profile photos synced to Azure AD because SharePoint Online receives profile information from Office 365 Azure Active Directory through one-way synchronization.
2. What would be the best source of this photo information?  Should we get it from SharePoint Online or from Azure AD (if the photos are available in Azure AD)?
Photos uploaded in SharePoint Online will not be synced to Azure AD, so you need to pull the photos directly from SharePoint Online.
All photos uploaded are stored in the User Photos library in the my site host.
You can access my site host via https://tenantdomain-my.sharepoint.com/_layouts/15/viewlsts.aspx.

Thanks,
Cliff
