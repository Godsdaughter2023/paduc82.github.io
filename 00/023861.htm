a. user say:

Saving mixed layout documents as PDF (again)

Could others please check out the following with a mixed-layout document (e.g. some portrait sections, some landscape sections):

 a. pick a page number that is larger than any (adjusted) page number in the document. e.g. for a 5-page document, numbered 2-6, pick 7.

 b. Click File->Print...

 c. Enter that page number into the "Page range" box in the dialog

 d. Click the PDF dropdown, and select Save as PDF...

 e. save the output.

 f. Open it in whatever PDF reader you have, and see if it contains the complete document.

Previous month in VBA - Microsoft Community

marcusmackaku say:

Previous month in VBA

How do I get the previous month in VBA?
amit tandon say:

Please see below code which also explains each code line in comments (whether previous month name or month number from NOW or from specified date):

Sub PreviousMth()

Dim dt As Date
dt = "11/23/2015"

'returns previous month name (previous month from current date)
'returns "January"
MsgBox Format(DateSerial(Year(Now), Month(Now) - 1, 1), "MMMM")

'returns previous month name (previous month from specified date)
'returns "October"
MsgBox Format(DateSerial(Year(dt), Month(dt) - 1, 1), "mmmm")
MsgBox Format(DateAdd("m", -1, dt), "mmmm")

'returns previous month number (previous month from current date)
'returns 1
MsgBox Month(DateSerial(Year(Now), Month(Now) - 1, 1))

'get date of one month back from specified date - returns "10/23/2015"
MsgBox DateAdd("m", -1, dt)

End Sub
