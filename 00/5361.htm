roady [mvp] say:

If you get the category assigned, then you can use Conditional Formatting to apply different colors to the message header in the message list view. It is not the same as a color category, but at least recognizable;
tab View-> View Settings-> button Conditional Formatting...

The use of it is explained here;
http://www.howto-outlook.com/howto/coloremailadvanced.htm
(the feature was called Automatic Formatting in previous versions of Outlook but still works the same)

The main reason why it is disabled is because the assigned categories won't be synced back to the IMAP server and thus also not to other (Outlook) clients which you use to access that same account.
