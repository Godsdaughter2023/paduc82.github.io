Help with Macro - Microsoft Community

jon o. say:

Help with Macro

Is it possible to assign a macro to an image in an Excel Spreadsheet that will allow me to click on it to enlarge it and then click on it again to bring it back to it's original size?
rich007a say:

This should do what you need (note, there was no need to do all that you had done...):

Sub PictureBigSmall()
    Dim shp As Shape
    Dim big As Single, small As Single
    Dim shpH As Double, shpOriH As Double
   
    big = 2.5   'Set large scale, 2.5 = 250%
    small = 0.5 'Set small scale, 0.5 = 50%
   
    Set shp = ActiveSheet.Shapes(Application.Caller)
   
    shpH = shp.Height
    shp.ScaleHeight 1, msoTrue, msoScaleFromTopLeft
    shpOriH = shp.Height
   
    If Round(shpH / shpOriH, 2) = big Then
        shp.ScaleHeight small, msoTrue, msoScaleFromTopLeft
        shp.ScaleWidth small, msoTrue, msoScaleFromTopLeft
        shp.ZOrder msoSendToBack
    Else
        shp.ScaleHeight big, msoTrue, msoScaleFromTopLeft
        shp.ScaleWidth big, msoTrue, msoScaleFromTopLeft
        shp.ZOrder msoBringToFront
    End If
End Sub

Sub Copy_Things()
Dim ws As Worksheet, LastRow As Long
Dim MyRange As Range, c As Range
Dim CopyRange As Range
Set ws = Sheets("Sheet1")
LastRow = ws.Cells(Cells.Rows.Count, "A").End(xlUp).Row

For Each c In MyRange

        c.Offset(, 2) = c.Offset(, 1)

hansv mvp say:

A subform is not part of the Forms collection. You must refer to it in the form

 

Forms!MainFormName!SubFormName

 

where SubFormName is the name of the subform as a control on the main form. This is not necessarily the same as the name of the source object of the subform.

So if your main form is named DataForm, you'd use something like

 

Forms!DataForm!Est_sub!cbo_Manu

 

(I assume that the apostrophe before Est was a typing error)

djsteele, mvp say:

While often it will work without referring to the Form property of the subformcontrol, it really should be included. (Some versions of Access are pickier than others!)

Forms!DataForm!Est_sub.Form!cbo_Manu
