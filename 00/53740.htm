Hello Gilles,

For you to be able to know if a program is running 32bit or 64bit, follow the steps below:

1. Right click on the application or program then select Properties.

2. Click Compatibility then check the "Run this program in compatibility mode for" box.

3. Open the drop-down list. If the list starts with Windows Vista, then the application you selected is a 64-bit application. If not, it's a 32-bit application.
