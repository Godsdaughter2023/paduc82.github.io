Microsoft Word 365 Table of Contents Issues

I also can't get the wrap to work correctly for titles that don't push the tab feature to the next line.

suzanne s. barnhill say:

There are two reasons for having all caps in the TOC:

  1. The TOC styles are formatted as All Caps (apparently not the case here).
  2. The headings were typed using Caps Lock. If you want the headings in the document to be in All Caps and the TOC not, you should apply the All Caps property to the heading style instead of typing in caps.

As for the wrapping, you need to set a right indent at half an inch or so from the right margin--far enough to clear the longest page numbers.

For more on formatting TOCs, see http://wordfaqs.ssbarnhill.com/TOCTips.htm
