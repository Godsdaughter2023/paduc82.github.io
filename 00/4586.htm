graham mayor say:
Office 365 installs using click to run technology (which is undoubtedly why so many people are having problems with it). Click to run allows an existing version of Office to remain installed. However only one version of Office can have control of the system, and with two versions it can become confused as to which is activated, and some recent Windows updates have managed to screw up the file associations for many Office users.

If you had already purchased Office 2007, then open Word 2007 from the Windows Start Menu. Click the Office button > Resources > Activate. If it is activated it will tell you, if it is not it will require the Office 2007 key.

If you didn't buy Office 2007, then what you have is almost certainly a trial version and because of the aforementioned saga of the update faux pas, it has control of the registry rather than Office 365. You will need to re-associate your Office files with 365. Repairing 365 might be the simplest way to achieve that.
