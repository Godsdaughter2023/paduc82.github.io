david marcovitz say: OK. So what you want is to trigger an animation based on mouse over. This is not possible. However, it is easily faked. Create two identical slides, one without the picture and one with it. Have the action settings on the trigger on the first slide be set on mouse over to hyperlink to the other slide. If you also want the picture to disappear when you get off the slide, you will then need a transparent shape on the second slide. This shape covers the entire slide, but is positioned behind the picture. This shape has its mouse over action setting to hyperlink back to the first slide.

It's a little more work than a simple trigger animation, but your users won't know the difference. It will behave just like the rollover effect you seek.

stefan blom say:

When you change the defaults for a text box in Word, the new settings apply to future text boxes that you create in that same document. You can change the defaults in your template, of course; the new defaults would then apply to all documents based on that template.

Note that the defaults you can save for a text box in Word does not include text formatting, only settings for the text box itself (borders, shading, fill, etc.).

What you can do is create an example text box with the desired settings and save it as an AutoText entry for easy re-use.
