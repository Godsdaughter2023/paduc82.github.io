stefan blom say:

If the language is not visible in the status bar. Right Click the bar and select <Language> is will display the default language in the document.

Please note that the Language control on the status bar does NOT display the default language; instead, it displays the language at the insertion point (or for the current selection).

Language is a character attribute, similar to bold or underline. You can change it for a selection by going to the Language dialog box (Review | Language | Set Proofing Language), pick a language and click OK (as opposed to clicking Set as Default).

If the "Detect language automatically" option is selected in the Language dialog box, Word will try to guess which language you are typing in and, consequently, may change it at the insertion point as you type. Once that has happened, you often have no other choice than to manually select all the text and (re-)apply the correct language to text.

For "everything" on language formatting in Word, see MVP Suzanne S. Barnhill's article on the subject at http://wordfaqs.ssbarnhill.com/MasterSpellCheck.htm.

Step One: Calculate the percentage of weight for each person. In this case, the sum of entries is 30 entries. You can use no. of entries of that person to divide total entries. For example, The percentage of Joe is: 5/30= 0.16667 (B2/30)

Freehand drawing on a slide for emphasis - Microsoft Community

bill w9 say:

Freehand drawing on a slide for emphasis

I want to add freehand drawings on a slide during presentation, such as encircling a word or subtitle.  I've used the Drawing Toolbar for other purposes.  How can I change linewidth and color when drawing a "scribble"?   or is there another tool better than a scribble for my purpose?
lucy thomson say:

Hi Bill

You could either add this line while you are editing (which will give you lots more options in colour & thickness) then add an animation to it so it will to the audience as if you are drawing it, or you can actually add it by hand directly onto the slide as you are presenting - that''s the difference I was trying to get at!

It sounds to me like you want option 1 as you are mentioning the drawing tools (which aren't available in slide show mode):

  1. Draw your line using the line, curve, scribble or freeform drawing tool
  2. With the line still selected, go to the Drawing Tools -> Format tab
  3. From the Shape Outline drop down select Weight and choose a pt thickness or 'More Lines' to set your own thickness
  4. Again from the Shape Outline drop down select a colour you want or 'more outline colours'. Note if you opened the 'more Lines' dialogue in the above step you can also change the colour there
  5. Now to animate: with the line still selected go to the Animations tab -> custom animations
  6. Select a suitable entrance animation (say wipe for a line, or wheel for a circle)
  7. Change direction, speed, spoke number (if using wheel) and start (i.e. with or without mouse click) as desired.

I hope that helps. Let us know if you have any more questions :-)

Lucy

Excel 2010 Vertical Scroll Bar - Split? - Microsoft Community

ryanbaldy say:

Excel 2010 Vertical Scroll Bar - Split?

I created a dashboard page witin a workbook which rolls up data from the rest of the sheets. On the dashboard page, my scroll bar has a "slider" of sorts that allows me to drag it up and down. When it is moved, it relveals a copy of the same page behind it.

 

What is the purpose of this?

How do I turn it off and on?

 

This workbook will be utilized by many front end users on my team and I would like to make this as user friendly as possible.

 

Thanks,

 

Ryan

e.j. gun say:

The slider allows you to "split" the active window into two "panes".  The two panes can be scrolled vertically independant of one another.  You can also use the "View/Freeze Panes" option to "freeze" the upper pane so that it cannot be scrolled.  This allows you to have something at the top that is always visible, while enabling the bottom part to be scrolled vertically.

 

To turn the split panes off, either drag the slider back up to the very top of the scroll bar, or go to "View" and click on the "Split" button.

 

BTW - you can also do this horizontally!

 

HTH,
