burrwalnut say:

Unless I’ve missed something, there seems little point in renaming a Windows operating system as Linux.

However, that information can be changed as it is contained in a file named oeminfo.ini  in \WINDOWS\system32. If the file currently exists on the computer, it might be prudent to rename it with an extension of old.

To create a new oeminfo.ini file, open Notepad and copy the text below into it:
[general]
Manufacturer=John built it last week
Model=Cray Supercomputer
[Support Information]
Line1=John Smith
Line2=Palo Alto
Line3=California
Line4=United States

There must be no breaks between the lines and no spaces between the "=" and start of the text. Amend it accordingly and use 'Save as' so that it doesn't become a text file, i.e. not the Notepad .txt default. Save it in the \System32 folder.
