jan karel pieterse say:

The compile error is due to a missing space character:

.SaveAs wbThis.Path& "C:\District3.xlsx"

should be:

.SaveAs wbThis.Path & "C:\District3.xlsx"

But that isn't all; wbThis.Path already contains a drive letter so you're adding another, making the entire path and filename invalid. Moreover, the Path property does NOT have a trailing path separator.

Therefore, the correct code would be:

.SaveAs wbThis.Path & "\District3.xlsx"
