At this point in time, we suggest that you delete the SoftwareDistribution folder. The SoftwareDistribution folder is where the system stores data for Windows Updates. Under normal conditions, you shouldn't touch this folder. That being said, we still suggest that you go through deleting the folder because doing so can fix Windows Update issues.

To delete the SoftwareDistribution folder, here's how:

  1. Press Win+X, and then select Command Prompt (Admin).
  2. Stop the Windows Update Service and the Background Intelligent Transfer Service by entering the following commands, one after the other:
    net stop wuauserv
    net stop bits
  3. Navigate to the following location:
