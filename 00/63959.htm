greg carmack - windows mvp 2010-2020 say:
Hi Matt. I'm Greg, an installation specialist and 9 year Windows MVP here to help you.

OneDrive is a folder under C:\Users|Your User Name that you can drag or copy files into to store them safely in the cloud and/or on the PC at the same time. So check that folder first to see what's there. It's possible entire User folders may be there. If so continue reading. If not and it's just ordinary files, you can move them elsewhere.

OneDrive introduced some features recently that were set as default for some users without their knowledge, so be aware and adjust as necessary

If the new Folder Protection is chosen in OneDrive Settings > AutoSave Tab, then Documents, Desktop and/or Pictures folders will be moved into the OneDrive folder. See Folder Protection details here to change this or read more:
https://www.tenforums.com/tutorials/116029-turn...

Another setting designed for smaller hard drives removes files from the hard drive and stores them only in the cloud if Files On Demand is chosen in OneDrive Settings. This feature is described here:
https://www.pcworld.com/article/3233488/windows...

If you want to disable OneDrive and it's presently signed in because there's a cloud icon in the System tray at right end of task bar, right click on it to open Settings, Then on Account tab choose Unlink from this PC and it will be gone. Any files in the OneDrive folder on the PC and in the cloud will stay put until you sign in again.

I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.

joseph agu say:
Thanks for getting back to us. It seems that this is most likely caused by corruption with the components that run the Windows Store. You can try these troubleshooting steps:

1) Reset the Windows Store cache.
    > On Windows RT 8.1, press the Windows key then search for "command prompt". Once found, click 'Run as administrator'.

    > Type and enter "wsreset.exe" once the command prompt pops up. This should reset the Windows Store cache and you'll receive a prompt saying so.

2) Download and run the App troubleshooter
    We suggest that you  download and run “Windows 8 UI App Troubleshooter” to diagnose the exact issue. Refer to this link to get started.

3) Delete files within the Windows Store folder
Head to this folder: C:\Users\user_name\AppData\Local\Packages\Microsoft.WindowsStore_8wekyb3d8bbwe\LocalCache and delete everything in it. Once done, restart your PC.

Additionally, please change "user_name" to your computer's actual username.

As for the changing of language for your device, you can check Ben_Mk' suggestion here.

Keep us updated on the outcome of the troubleshooting.
