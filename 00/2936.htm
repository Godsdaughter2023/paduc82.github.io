nikhil_s_k say:

selecting a range based on a cell value in another formula

Hi. I have a worksheet with names in Col D & month in Col E(entered as text i.e. Aug, Sep and so on). In Col G I have a formula which is =ROUND(SUMPRODUCT(Weightage,Aug!T3:T166)*10/SUMIF(Aug!T3:T166,">=0",Weightage),2). I have different named ranges like Aug, Sep, Oct..... If the cell in Col E is aug, the formula uses the range Aug. If the cell in Col E is sep, the formula should use the range Sep.

Also I want the columns to be dynamic. Here in the formula its T3:T166. This column depends on the value in Col D. So if the name in Col D changes, the column range in the formula should also change...For the lookup purpose i check if cell in ColD matches to any value in the worksheet called Aug (row G1:Gx1) and returns the col no. In the above formula the match happens in the 14 column (i.e. T Column) . The row nos is constant all across for all sheets & ranges - 3:166.

How should my above formula be modified to include dynamic values based on Cell D & Cell E

Plz help

Nikhil

max, singapore say:

Essentially you're seeking a way to dynamically capture this range:  Aug!T3:T166
where Aug (the sheetname) is defined in a cell, say in cell E1,
and the col range, ie row 3 to 166 is defined off a column number housed in say, cell D1
(The phrase "Cell D" in itself has no meaning, a cell must also have a row ref, eg cell D1)

Assume E1 houses the text:  Aug
and D1 contains:  20  (ie column "T" = 20, the column number) 
then this OFFSET expression below will return the range:  Aug!T3:T166
OFFSET(INDIRECT("'"&$E$1&"'!A3:A166"),,$D$1-1)

You could plug the above OFFSET into your posted formula, replacing "Aug!T3:T166"
and that should achieve the dynamic features that you seek
