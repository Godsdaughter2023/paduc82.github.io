How can I hide columns based on a empty cell? i.e. C1 is empty so I want column C hidden. I only want this to apply to columns not rows.

 

thank you.

 

jllatham say:

What is in C1 at this time?  Is there a formula that returns a result of some kind, presumably text or ""?

C1 actually does not contain anything (no value you have typed in and no formula) then how would column C get unhidden?

If there is a formula in C1, then this worksheet event code will may do the job for you.  When C1 = "" then column C will be hidden, but if C1 contains some visible text or value, then column C will be made visible.  For testing I put this formula into C1 and then changed the value in A1 from 0 to other numbers greater than zero , and back to 0 a few times.

=IF(A1>0, "has data","")

To put this code into the workbook in the proper location, open the workbook, select the page with the column C you need to work with.  Right-click on the worksheet's name tab and choose [View Code] from the list.  Copy the code below and paste it into the code module shown to you.  Save the workbook.  Press [Alt]+[Q] to return to the worksheet and enter values or data into other cells that would affect the results in C1 to see if it is going to work for you or not.

 

Private Sub Worksheet_Calculate()
  If Range("C1") <> "" Then
    Range("C1").EntireColumn.Hidden = False
  Else
    Range("C1").EntireColumn.Hidden = True
  End If
End Sub

jeeped say:
Try this event macro triggered by a change in values for a cell within a worksheet.

Right click the name tab of the worksheet and select View Code. When the VBE pops up paste the following into the pane on the right side titled something like Book1 - Sheet1 (Code),

Private Sub Worksheet_Change(ByVal Target As Range)
    Application.ScreenUpdating = False
    On Error Resume Next
    With Range("A1", ActiveCell.SpecialCells(xlLastCell))
        .SpecialCells(xlCellTypeConstants).EntireColumn.Hidden = False
        .SpecialCells(xlCellTypeFormulas).EntireColumn.Hidden = False
        .SpecialCells(xlCellTypeBlanks).EntireColumn.Hidden = True
    End With
    Application.ScreenUpdating = True
End Sub

Press Alt+Q when you are done to return to your worksheet.. Vacating (e.g. making blank) any cell within your used range will cause that cell's column to become hidden. This looks at your entire data range for blank cells. If you only need a specific row to be considered for blanks, post back with more details.

You can Unhide the columns manually through selection and right-click, Unhide or press F5 to GoTo a hidden cell and supply a value which will expose the column if all other cells in the column are non-blank.

I don't usually like to use the On Error code, but Excel VBA really doesn't like referring to SpecialCells when there are none of that type.
