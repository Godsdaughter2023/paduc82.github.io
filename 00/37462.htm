frederik long say:

Since System Restore does not work, you need to do this:

  1. Back up your data files, including your E-Mail files.
  2. Perform a few sample restores on a different machine, to be sure that the backup is readable.
  3. Restore the old image.
  4. Restore your data from your backup.
  5. Restore any application that might have disappeared.
  6. Ask the Avast help desk about the misleading advice you were given.
  7. Review your backup policy. Lots of people feel that regular backups to an external medium are a waste of time. They usually change their minds after suffering a major disaster.

Here is an alternative method. It is more demanding but also more secure because you cannot possibly lose any of your data files.

  1. Create a new image. Store it on an external disk while making sure not to overwrite the old image.
  2. Mount the new image on a different machine as a Virtual Hard Disk (VHD).
  3. Do a few spot checks to ensure that your data is readable.
  4. Restore the old image.
  5. Mount the new image VHD.
  6. Manually restore your folders from the VHD.
  7. Restore any application that might have disappeared.

_null____ say:

You will need to repartition your hard disk, as Windows does not recognize the file system that Ubuntu Linux uses by default (which is ext3). You can do this within Ubuntu Linux using several different tools, or if you are no longer interested in using Ubuntu Linux you can repartition the disk within Windows setup.

PLEASE NOTE: The following steps will delete any data that is contained on your hard disk. I can not be held responsible for any data loss that may be incurred. Please ensure that you back up your data before executing the following commands. Use at your own risk.

To repartition the disk using Windows Setup:

1. Insert the Windows 7 installation DVD into your DVD drive.
2. Reboot your computer and boot from the DVD drive when asked ("Press any key to boot from CD or DVD...")
3. Windows Setup will load, give it a few moments.
4. When you see the Windows 7 installation screen with "Install Windows" press Shift-F10.
5. In the Command Prompt window that appears, type diskpart and then press Enter.
6. At the diskpart prompt, type select disk 0, and then press Enter.
7. At the diskpart prompt, type clean, and then press Enter.
8. The DiskPart utility will remove any partitions on the hard disk as well as the master boot record (MBR).
9. At the diskpart prompt, type exit and then press Enter.
10. In the command prompt window, type exit.
11. Click on Install Windows and follow the on-screen instructions.

lemp say:

Bluetooth accessories like this don't need specific drivers. 

On the other hand, newer Bluetooth devices, specifically including the JBL Charge 3, use Bluetooth version 4 and Windows 7 does not natively support Bluetooth 4.  See, e.g., https://docs.microsoft.com/en-us/windows-hardware/drivers/bluetooth/bluetooth-support-in-previous-windows-versions

Fortunately, you do not have to abandon your new speaker.  You do, however, have to buy a new USB-connected Bluetooth 4 adapter -- and this device will come with the required driver.
