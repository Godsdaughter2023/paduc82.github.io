You can create the text file you are after without opening NotePad or fooling around with SendKeys... just supply the path/filename for output where indicated in the macro below and the used range on the active worksheet will be sent out as a text file directly.

Sub SaveUsedRangeToTextFile()
  Dim R As Long, C As Long, FF As Long
  Dim TextLine As String, TextOut As String
  Dim Data As Variant
  Data = ActiveSheet.UsedRange
  For R = 1 To UBound(Data, 1)

kwroblewski say:

Hoopla! I have located the solution, you can create a shortcut (Something.lnk) and then use it in whatever launcher app you may use. (Facade, Orchid, etc).

To create a shortcut to SMS (or MMS) a specific phone number, create a text file and put in the following:

 

20#\Windows\tmail.exe -service "SMS" -to "888-888-8888" -body ""

 

where "888-888-8888" is the phone number of the person you want to SMS or MMS. then save the text file as Something.lnk and put it where you want to access it.

 

As you could guess, if you want to also automatically insert a body, put it in the body string.

 

Note: I don't think this works on WM6 from what I have read.

The 502 Bad Gateway error is an HTTP status code which means that one server received an invalid response from another server that it was accessing while attempting to load the web page or fill another request by the browser.
