doug robbins - mvp office apps & services (word) say:

Almost certainly, the space on either side results from your printer not being able to print "full bleed", that is right from one edge of the paper to the other. 

While not many printers are capable of doing that, you may be able to trick your printer into printing right across the sheet that you are using by setting the paper size wider by the sum of the widths of the two small spaces.  To prevent other changes to the layout of your document, you will need to increase the width of the side margins so that the space between them remains the same.

Note however, this may result in your printer depositing ink\toner off the edge of the paper which may later cause problems.

hi lorraine,

about “i didn't want to have access to all the folders”, i would like to confirm if these folders you don’t want him/her to access are in the same library with the folder you shared as below:

to remove someone from a shared file, we can check the user permissions for the file and then remove them.

1. open the library and select the file you shared.
2. click the ellipsis > share > share with > advanced. if the user displays in this page, we can select it and click remove user permission. 
  
3. if not, please check if the user have permissions through some groups. please click check permission in this page and type the user into the box.

4. remove the user from the group.

if the user can access all folders in this library, we need to check his/her permission at the library level via: library > library settings > permissions and management > permissions for this document library > check permissions. then remove his/her permissions as above.

if there is anything misunderstood or unclear, please let us know at your convenience.
