cmdrkeene say:

What you are seeing could be an effect of DPI scaling.  Because your monitors have different number of pixels per inch (dots per inch, DPI), the size of items and text on the screen is adjusted to try to keep the same size on each monitor.  This adjustment in size can cause fuzziness.


To turn this scaling off, try this:

  1. Right-click your desktop background and choose Screen Resolution.
  2. At the bottom, choose "make text and other items larger or smaller" link.
  3. Uncheck the "use one size for all displays" checkbox, then click apply.
  4. Test for better text now.

Did you try upping the Zoom or DPI?  ClearType needs room to do its work.  Characters aren't rendered using just the old 9x5 set of pixels.  Unfortunately, they spoiled us by using a feature called sub-pixel color interpolation but that caused problems with color fringing and so has now been changed to grey-scale interpolation but that means it is no longer "sub-pixel" and that means you need to give up more pixels to help it out, hence the need to use slightly larger Zoom or DPI settings.
