philipradler say:

Word will show just a thin strip of the bottom of the image if your paragraph spacing is set to "Exactly" (and depending on your font size, it's usually at something like 12 points—which is just 1/6 of an inch). This procedure will fix that problem:

  1. With the image selected or the cursor in the affected paragraph, go to Home>Paragraph and open the dialog box by clicking on the arrow in the bottom-right corner of the "Paragraph" group.
  2. Make sure you're on the "Indents and Spacing" tab. The "Spacing" choices are about midway down, and the "Line Spacing" choices we're interested in are to the right. Change spacing to "Single" or "Double" (or just about anything except "Exactly") and your graphic should magically appear. 

Hope this helps.

Macro for Increasing/Decreasing Decimal - Microsoft Community

ashoo1711 say:

Macro for Increasing/Decreasing Decimal

Very often I need to increase/decrease decimal in large tables which have percentages and absolute numbers.

If I select all and make the change, percentage get converted in number format. Selecting numbers one by one to increase/decrease decimal is inefficient as I need to do this in various such tables.  Is there a way to inc/dec decimals without screwing the number formats in the cells selected?

***Post moved by the moderator to the appropriate forum category.***

bernie deitrick say:

Select your cells and run one of these

Sub IncreaseDecimal()
    Dim rngc As Range
    For Each rngc In Selection
        If rngc.NumberFormat = "0" Or rngc.NumberFormat = "0%" Then
            rngc.NumberFormat = Replace(rngc.NumberFormat, "0", "0.0")
        Else
            rngc.NumberFormat = Replace(rngc.NumberFormat, "0.0", "0.00")
        End If
    Next rngc
End Sub

Sub DecreaseDecimal()
    Dim rngc As Range
    For Each rngc In Selection
        If InStr(1, rngc.NumberFormat, "0.00") > 0 Then
            rngc.NumberFormat = Replace(rngc.NumberFormat, "0.00", "0.0")
        Else
            rngc.NumberFormat = Replace(rngc.NumberFormat, "0.0", "0")
        End If
    Next rngc
End Sub
