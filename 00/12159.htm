rich007a say:

Or....

 

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim rng As Range
    If Target.Cells.Count > 1 Then
        Exit Sub
    End If
    On Error GoTo ErrHandler:
    If Not Application.Intersect(Target, Range("A:A")) Is Nothing Then
        If Target.Text = "M" Then
            Set rng = Target.Offset(0, 5)
            rng.Value = "NULL"
            With rng.Interior
                .Pattern = xlSolid
                .TintAndShade = -0.149998474074526
            End With
            With rng.Font
                .ThemeColor = xlThemeColorDark1
                .TintAndShade = -0.149998474074526
            End With
        End If
    End If
ErrHandler:
End Sub

 

(if you think you might want to do other things to Target.Offset(0,5)....)

Note there's no need to select a cell or range to do things to it.

Also note that Target is a range representing the range that was changed (when the event is fired).  After you make a change and hit enter, usually the cell below becomes the ActiveCell, so if you use ActiveCell for your .Offset(0,5) you end up a row down on where you wanted to be!

Hope that helps.
