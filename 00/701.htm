chriss_83 say:

Insert Excel data into multiple word files

We have over 75,000 Word documents and what we want to do is insert information from and Excel file into each Word document. We want the name, date of birth, gender and account number at the top of each file.

jeeped say:
You are not going to want to run any macro created on 75K DOCX files without thorough testing on a smaller sampling. Create a folder and copy a few (3-5) DOCX files into it along with a copy of the workbook containing the worksheet with the account numbers and other information. Edit the copied workbook so the worksheet only references the DOCXs in the new folder.

With Word closed, open the workbook and tap Alt+F11 to open the Visual Basic Editor (i.e. VBE) and immediately use the pull-down menus to Insert, Module. Start by pasting the following into the new pane titled something like Book1 -Module1 (Code), 

Sub mcrAppendWordDOCX()
    Dim sFn As String, r As Range, i As Long, x As Long
    Dim wdApp As New Word.Application
    Dim wdDoc As Word.Document
    For Each r In Range("A2:A" & Cells(Rows.Count, "A").End(xlUp).Row)
        sFn = ActiveWorkbook.Path & "\" & r.Value & ".docx"
        If Len(Dir(sFn)) > 0 Then
            Set wdDoc = wdApp.Documents.Open(sFn)
            With wdDoc
                .Content.InsertBefore "Gender: " & r.Offset(0, 3).Text & Chr(10)
                .Content.InsertBefore "DOB: " & r.Offset(0, 2).Text & Chr(10)
                .Content.InsertBefore "Name: " & r.Offset(0, 1).Text & Chr(10)
                .Content.InsertBefore "Account: " & r.Text & Chr(10)
                .Close
                i = i + 1
            End With
            Set wdDoc = Nothing
        Else
            x = x + 1
        End If
    Next r
    Set wdApp = Nothing
    MsgBox i & " DOCX files appended." & Chr(10) & x & " DOCX files not found."
End Sub

Edit the column and offset references to properly reflect your own data layout. The above uses a data layout like the following:

          

Tap Alt+Q to return to your worksheet and then tap Alt+F8 to open the Macros dialog and Run the macro. After it has completed, you will be presented with a dialog box noting the number of DOCX documents modified and the number of DOCX documents that could not be located and opened.

Open one or more of the DOCX files and observe what changes have been made to the beginning of the document and make appropriate changes to the Excel macro if necessary.

That should be enough to get you started. Post back with any questions you might have. There is also a wealth of information on transferring text directly from Excel to Word via a VBA  programming conduit available online for general information on the subject.

It's always been something that COULD be fixed; gapless playback is a problem that's been solved for years, by pretty much all products that do music playback, including lots of Microsoft ones.

It's been announced that Windows 10 Mobile, shipping later this year, will support gapless playback.  I haven't heard anything specific about whether it will be supported at launch, or added later - but still, it's the first time that Microsoft has said anything even slightly definite about it, so I'm cautiously optimistic.

Spotify with a premium subscription kinda/sorta supports gapless playback today - most of the time, if I'm listening carefully with headphones, I can still hear the gap - but sometimes I can't, and it's always smaller than the built-in Music app's gaps.  (I don't think that gapless is specifically a premium-only feature - but with the free membership, you can't play albums, so gapless doesn't apply...)
