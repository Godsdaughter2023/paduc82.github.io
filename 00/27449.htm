cjlim say: The CA-101 is a data cable which also charges when used with a phone that supports USB charging. The E71 does not support USB charging so is only works as a data cable. The same applies to the N96 which also comes with the CA-101 cable.

The CA-100 cable on the other hand is a cable which has a USB port one end and a power charger jack the other end and can only be used to charge a phone. See link below.

http://europe.nokia.com/A4397382

The CA-126 does both. See

http://europe.nokia.com/products/accessories/all_accessories/memory-cards-and-cables/cables/CA-126

Message Edited by cjlim on 17-Nov-2008 12:07 PM

psychomania---01 say: The description on the CA-101 cable is misleading. On most models the cable is for data only, not charging. The E71 is compatible with that cable for data only.

If you had looked at the CA-100 & CA-126 in the accessories section you would have noticed that they connect to the phones charger port, not the USB port.

The CA-100 connects to a PC's USB port then to the phones charger port.

The CA-126 is like the CA-100 & CA-101 combined. It connects to the computers USB port but it has two ends. One plugs into the power port and the other into the data port.

I hope this simple explanation clears things up for you.

If not here it is again:
The E71 cannot be charged through it's USB port. It is not possible.
