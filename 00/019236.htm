suzanne s. barnhill say:

If you're referring to using Alt+0xxx to insert special characters, you can do this even though Word has popped up ScreenTips (since those all require an alphabetic character rather than a number). The number must be typed from the numeric keypad with NumLock enabled.

If you're referring to keyboard shortcuts you've created, note that this is why it's not a good idea to create shortcuts with just Alt (Alt+Ctrl and Alt+Shift and Alt+Ctrl+Shift are okay). The Keyboard dialog will not tell you that those commands are already assigned, but in a sense they are.
