rajesh govind say:

 

Hi,

 

 

The Microsoft Compatibility Center shows that Microsoft Wireless Mouse 5000 is completely compatible with Windows 8. So, this does not seem to be an incompatibility issue with the mouse. As you have mentioned, there should be something which is conflicting with the mouse driver software causing this issue.

 

Method 1: Run the Hardware Troubleshooter

 

The Hardware Troubleshooter is an automated tool which checks the hardware connected to the computer for any known issues and provides the details on how to fix them. Follow these steps to run the troubleshooter.

 

a)       Press “Windows Logo” + “W” keys from the keyboard.

b)       Type “Troubleshooting” in the search bar and press “Enter”.

c)       In the “Troubleshooting” window, click on “View All” on the left pane.

d)        Click on “Hardware and Devices”.

e)       Click on “Advanced” and then click on “Run as Administrator”.

f)        Click “Next” and follow the on-screen instructions to complete the troubleshooting process.

 

 If the issue persists, then try the next method to check if there is any other conflicting mouse software installed on the computer.

 

Method 2: Check if the issue is caused by any other conflicting mouse software.

 

a)       Press “Windows Logo” + “X” and select “Programs and Features” from that list.

b)       Check if you have any old mouse software in the program list and then select it

c)       Click “Uninstall” and follow the onscreen instructions and restart the computer if prompted.

 

Now check if you still face the same issue with the mouse. If the issue still persists, then let’s also perform a clean boot and check.

 

Method 3: Perform a Clean Boot and find out which application/service is causing this issue.

 

A clean boot helps to eliminate software conflicts and to determine what is causing the problem. Perform a clean boot on the computer by following the steps given in the link:

 

http://support.microsoft.com/kb/929135

 

Check if the issue persists. If the issue does not happen in clean boot, it does not mean that the issue is resolved completely. It means that we have a possible identification of the root cause and how to confirm it is as described in the above given article. Follow all the steps from the link and try to figure out the application or service causing this issue.

 

Note: After you have finished troubleshooting, refer the article “How to reset the computer to start as usual after troubleshooting with clean boot” for Windows 8, given under the section “More Information” to reset the computer to start as usual from the above link.

 

If this issue still persists, then it would be better to get in touch with the Microsoft Hardware Support for further assistance. You may contact them from this link:

 

http://www.microsoft.com/hardware/en-us/support/contact-us

 

Hope this information helps you in resolving the issue.
