monique tang msft say:

Hi MOL,

Based on my research, this error could occur if the mapi32.dll file is corrupt. Please run MAPI Repair Tool to repair the mapi32.dll file by taking the following steps:

  1. Quit all programs.
  2. Rename the Mapi32.dll file in the C:\Windows\System32 folder to "Mapi32.old" (without quotation marks).
  3. Locate the Fixmapi.exe file in the C:\Windows\System32 folder, and then double-click Fixmapi.exe to run the program.
  4. Restart your computer.

If the issue persists, let’s then identify if the issue is related to add-ins. Please start Outlook in safe mode to see if it helps. (Click Start > Run> type outlook.exe /safe> click OK.)

If safe mode works, try disabling add-ins to see if it works:

Click File> Options> Add-ins> Select “COM Add-ins” in the Manage drop-down> click Go> Uncheck any add-ins that you are not using> click OK.

And if the error occurs in safe mode too, I’d like to confirm the following information:

  1. The OS version.
  2. The detailed version of your Outlook client. The detailed version of your Outlook client. Please navigate to File> Office Account> Office Updates to check it.
  3. Which account are you using, Office 365, Outlook.com, Hotmail.com or others? If it’s Office 365, is it an Exchange Online mailbox or an Exchange hybrid one?
