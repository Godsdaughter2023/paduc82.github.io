alex chen msft say:

Hi Kate,

 

Per your description, you want to separate the original pivot table and the pasted one. Since it is related to data cache, changing the name of the pivot table won't achieve the effect you want it to be. Here is an official article for your reference.

If you want to separate the existing  pivot table, you can try this VBA code.

---------------------------------------------------------------------------------------------------------

Sub DataCache()

 

Dim PivTbl As PivotTable

Dim ws As Worksheet

Dim wsTemp As Worksheet

Dim pt As PivotTable

 

For Each ws In ActiveWorkbook.Worksheets

 

For Each PivTbl In ws.PivotTables

 

c = PivTbl.RowRange.Column

r = PivTbl.RowRange.Row

ws.Activate

Cells(r, c).Select

Set wsTemp = Worksheets.Add

ActiveWorkbook.PivotCaches.Create(SourceType:=xlDatabase, SourceData:=PivTbl.SourceData).CreatePivotTable TableDestination:=wsTemp.Range("A3"), TableName:="PivotTableTemp"

PivTbl.CacheIndex = wsTemp.PivotTables(1).CacheIndex

Application.DisplayAlerts = False

wsTemp.Delete

Application.DisplayAlerts = True

Set PivTbl = Nothing

Next PivTbl

 

Next ws

 

End Sub

---------------------------------------------------------------------------------------------------------

 

Pivot tables' name won't be duplicate, every time you copy & paste a pivot table, it will be automatically assigned a new name.

jay freedman say:

First go to Office button > Word Options > Display. Check the box in the middle group (Always show...) for "Hidden text", and uncheck the box in the bottom group (Printing options) for "Print hidden text".

Select a group of pages you don't want to print, and apply the Hidden font format to the selection (most quickly with the shortcut Ctrl+Shift+H). Repeat for each such group, leaving only the "do print" pages unmarked.

When you view the document on screen, you'll see everything (the hidden text will have a dotted underline), but only the unmarked text will print.

You may need to mark the first paragraph of each printing page with the "Page break before" option in the Line & Page Breaks tab of the Paragraph formatting dialog to prevent the pages from collapsing together.
