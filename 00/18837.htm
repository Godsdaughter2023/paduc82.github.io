Hi Klaro

If you mean files that you download are disappearing after a few days, this is being reported since the release of version 1809 of Windows 10

The downloads folder was only ever meant as a temporary storage folder, once a file is downloaded, it should be moved to a more permanent storage location

Previously this was never really enforced by Windows . . .

The Windows 10 1809 update has been reported as deleting the entire contents of the downloads folder when it is installed. That update also added the Downloads folder to disk cleanup and that is turned on by default. The downloads folder has also been added to Storage Sense, so if Windows thinks you are running low on Space, it will delete the contents of your Downloads folder

So if you regularly run Disk Cleanup, you need to untick the Downloads folder. If you run any software like CCleaner, check that it is also not deleting the contents of Downloads and finally, open the Settings App, go to System Storage - Change how we free up space automatically and make sure Storage Sense is turned off
