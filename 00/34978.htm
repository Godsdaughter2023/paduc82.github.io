Method 1:

It is possible that the names of the computer are same on the local network. You may try renaming the computer name.

The issue can occur for one of two reasons:
- You actually have two machines on the network with the same name
- One of the computers has the same name as the name of the workgroup

a. You can go to Start then click on run then type “CMD”
b. In the command prompt type in “nbtstat –n” and it’ll show you a conflicting workstation if there is one.

Make sure to look at the one for network connection you are currently using, i.e. Local Area Connection or Wireless Network Connection and check.

You can rename a computer by following the below steps -
a. Right-click on the computer icon on the desktop or computer in the start menu.
b. Click on Properties and check for the section Computer name, workgroup and domain settings.
c. Now under this section click on Change settings and then under theComputer Name tab you can change the computer name. Click on Apply and then OK.

Method 2:
The problem may also arise due to some old network adapter settings–

a. Click onStart menu, type in “devmgmt.msc” to open the Device Manager.
b. Click on View and choose Show Hidden devices.

c.Now scroll down to Network Adapters and right click to uninstall those that are showing with transparent icons. If nothing is grayed out, then your problem is also not with an old network adapter.
d. Restart if you have uninstalled any old network adapter and then check if the issue persists.

Even if you change the display language, some text in menus and dialog boxes for some programs might not be in the language that you want. This happens because the program might not support Unicode.

The system locale determines the default character set (letters, symbols, and numbers) and font used to enter and display information in programs that don't use Unicode. This allows non-Unicode programs to run on your computer using the specified language. You might need to change the default system locale when you install additional display languages on your computer. Selecting a different language for the system locale doesn't affect the language in menus and dialog boxes for Windows or other programs that do use Unicode.

Follow the steps in the article below to change the system locale.
