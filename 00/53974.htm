Hi Phil,

Thank you for writing to Microsoft Community Forums.

I'd suggest you to check whether the system is meeting Windows 10 requirements or not. Refer this article on How to Find Windows 10 Computer Specifications & Systems Requirements.

I'd suggest you to create a bootable media by downloading the latest ISO and to perform the following steps.

Refer this article on How to create Windows 10 installation DVD/USB  flash drive Step-by-Step Guide.

Method 1: Perform Automatic Repair or Startup Repair.

1. Set up your BIOS to boot from the Windows 10 installation DVD/USB.

2. After booting into setup, choose Repair your computer option.

3. Click on Troubleshoot > Advanced options > Start up Repair.

Don't restart the device during this process. If the issue persists, follow method 2.

Method 2: Rebuild BCD.

1. By following the above method, open Command prompt from Advanced options.

2. Type the following commands one by one and press Enter.

              - bootrec.exe /FixMbr

              - bootrec.exe /FixBoot

              - bootrec.exe /Rebuild BCD

Once it is done, restart the system and check. If the issue persists, follow method 3.

Method 3: Use DISM command and check.

1. Open Command prompt.

2. Type the following command and press Enter: DISM /Online /Cleanup-Image /RestoreHealth

Wait for the operation to complete. Restart the system and check. If the issue persists, follow method 4.

Method 4: Perform a clean installation of Windows 10.

I'd suggest you to perform clean installation of Windows 10 using the bootable media that you've created. I'd also suggest you to unplug all the external devices and install it. You can refer this article on How to Clean Install Windows 10 on a PC Step-by-Step Guide.
