Sub MoveEmOutR1()
'Jim Cone - Portland, Oregon USA - September 2017
Dim rngRw As Excel.Range
Dim x As Long
Dim Y As Long
Dim Z As Long
Dim N As Long

Set rngRw = Selection.Cells
If rngRw.Rows.Count > 1 Then
VBA.MsgBox ("Select One row only.        "), vbInformation, "Julie"
Exit Sub
ElseIf Application.WorksheetFunction.CountA(rngRw) <> rngRw.Columns.Count Then
VBA.MsgBox ("No blank cells allowed.     "), vbInformation, "Julie"
Exit Sub
End If
Application.ScreenUpdating = False
N = 1
Do
N = N + x + 1
Y = VBA.InStr(1, rngRw(1, N - 1), ")", vbBinaryCompare) - 2
x = VBA.CLng(VBA.Mid(rngRw(1, N - 1).Text, 2, Y)) - 1

If x > 0 Then
For Z = 1 To x
rngRw(1, N).Insert Shift:=xlToRight, CopyOrigin:=xlFormatFromLeftOrAbove
Next 'Z
End If
Loop Until VBA.IsEmpty(rngRw(1, N + x))

Application.ScreenUpdating = True
End Sub

Method 1:

 

 

Check for the service DHCP Client and DNS Client Services if they are started or not.

 

1. Click on Start

 

2. In the start Search box type Services

 

3. Right Click on - RUN AS ADMIN

Check if the below mentioned services has the mentioned status and start up type.

 

DHCP Client Started Automatic\Manual.

DNS Client Started Automatic\Manual.

 

Also check the dependency services that are required for DHCP Client and DNS Client Serviceto run.

To check that double click on the service and click on the dependency tab.

 

 

Method 2:

 

You can get a new IP address for your computer. You might need to do this if you are having connection problems.

a. Click Start. Type cmd in the Start Search box, and then press ENTER.

b. At the command prompt, type the following command, and then press ENTER:
ipconfig /all (there is a space between ipconfig and /)

The ipconfig /all command displays Windows TCP/IP settings for all your network adapters.

c. ipconfig /release in the command prompt and press ENTER. This releases your current IP address.

d. ipconfig /renew in the command prompt and press ENTER.

The DHCP configuration for all adapters will be renewed. To renew the IP address for a
specific adapter, type the adapter name that appears when you type ipconfig at the command prompt
