myall_blues say: The =RAND function in Excel is one of a group of what are known as 'volatile' functions. Among others in the group are =NOW() and =TODAY().
Being volatile means that they recalculate every time the workbook recalculates, which includes both when it opens and when it saves. The status of the file being read only or whatever does not affect this (but as long as you don't save the base file doesn't change, but unfortunately this doesn't help you as you can never see it).

The only way to overcome this is to convert the random values to fixed values by selecting and copying all the =RAND cells, and then 'Paste Values'.
