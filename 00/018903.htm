doug robbins - mvp office apps & services (word) say:
Adjust the spacing before or after the paragraph via the Paragraph dialog (Hold down the Alt key and type ppg).  You may also want to change the default line spacing to Single.  Then click on the Set as Default button and when the dialog appears asking if you want to set the default alignment, indentation and spacing for the paragraphs for This document only or All documents based on the Normal.dotm template, select the latter.
 
Note, you should probably leave the Space after setting alone so that when you press the Enter key to start a new paragraph, a space will automatically appear between it and the preceding paragraph.  If, for some reason you want to start a new line in a paragraph, use Shift+Enter.  Also note, that things have moved on from the days of typewriters and when you get to the end of the line, a new line will be started automatically without you having to do anything, but continue typing.
