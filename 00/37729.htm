Deleting Folders/Files from Backup - Microsoft Community

otisg1 say:

Deleting Folders/Files from Backup

How can a folder or file be deleted from the backup file?

a. user say:

It depends on what backup method you used.  If you did a Complete Backup (available only with Vista Business and Vista Ultimate) then you can't delete a file or folder since the entire backup is tied together to bring the computer back into the same exact state it was in at the time that the backup was made.  The best way to deal with this is to format the backup drive and then do another complete backup after deleting the file or folder from the system - then the backup will not have the file or folder in question.  If you did a full backup or an incremental backup, all you need to do is find the zip file that corresponds to the file you want to delete (or the folder containing the zip files that you want to delete) and delete the file or folder.

If the system says you don't have permission (or gives you an access error), give yourself permissions as follows (including taking ownership):

To see your permissions, right click the folder/file, click on properties, and check out the security tab.  Check to see what permissions you have by clicking on your username (or user group).  Here are the types of permissions you can have: http://windows.microsoft.com/en-US/windows-vista/What-are-permissions.  You need to be an administrator or the owner to change permissions (and sometimes being an administrator or even an owner isn't enough - there are ways to block even them from access (though a smart administrator knows these ways and can get around them - but usually shouldn't because they were denied access usually for a very good reason).  Here's how to change folder permissions in Vista: http://www.online-tech-tips.com/windows-vista/set-file-folder-permissions-vista/.
 
To fix this problem with the folders, take ownership of the folders or the drive (as an administrator) and give yourself full rights.  Right click the folder/drive, click on properties, go to the security tab and click on Advanced and then click on the Owner tab.  Click edit and then click the name of the person you want to give ownership to (you may need to add it if it isn't there - or it may be yourself). If you want it to apply to sub-folders and files within that folder/drive, then check the box for replace owner on sub-containers and objects and click OK.  Back out and now there's a new owner for the folders/drive and files who can change permissions as needed.  You can now change them from read-only (even if the main folder says they still are read only - you can have access yourself as the owner).  You can keep them as read-only for other users and guests and even administrators (though they can take ownership themselves and gain access if they so desire and there really isn't much you can do to stop this except protect the folder with a password using a 3rd party product.  Here's more information on taking ownership of a file or folder: http://www.vistax64.com/tutorials/67717-take-ownership-file.html.

If the file or folder still won't delete, try Unlocker at: http://www.softpedia.com/get/System/System-Miscellaneous/Unlocker.shtml which is sometimes successful at deleting files and/or folders when the normal Vista delete process doesn't work correctly for whatever reason.

I hope this helps.

Good luck.

P.S. It is VERY unusual to want to delete a file or folder from a backup set since all you need to do to accomplish the same thing is restore the backup (if necessary) and then delete the file and/or folder after it has been restored.  You can also do it by deleting the file/folder in the system and then doing another full/complete backup and it won't be there.  What exactly are you trying to accomplish?
Lorien - One - MCSE/MCSA/Network+/A+

If you are running Windows 8.1, please follow these steps to check for and set the correct audio device as the default.

  1. Swipe in from the right edge of the screen, and then tap Search.

  4. If the wrong audio device is listed as the default audio device, tap or click  the correct device,  tap or click Set Default, and then tap or click OK.

  5. Changing the audio device might require you to restart your PC.

More information can be found here.

If you are running Windows 10, please follow these steps to check for and set the correct audio device as the default.

  1. Open Start, enter Sound, select it from the list of results,
  2. On the Playback tab, select a device, then Set Default.
