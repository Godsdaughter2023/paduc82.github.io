              ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
              ren C:\Windows\System32\catroot2 catroot2.old

Step 4: Restart the BITS, Cryptographic, MSI Installer and the Windows Updates Services by entering the following command prompts in the Command Prompt (Admin) window.


              net start wuauserv
              net start cryptSvc
              net start bits
              net start msiserver

Step 3: Wait for the process to complete.
Step 4: Close the Command Prompt window.

Step 5: Reboot the PC and check if the changes are effective.

armand cos say:

Hi,

To resolve the issue, we would suggest that you follow the steps below:

  • Press “Windows key”, right click on Photo app
  • Click “Turn Live tile off”
  • Now open the “Photo app”, click on “settings” in the left pane
  • Add photos folder in the sources that you want to view on the tile
  • In the “photo tile” option, click on “recent photos”
  • Close the current window, again press “Windows key”
  • You should be able to see the selected images now

Hope this helps. Do get back to us should you need further assistance.

Regards.
