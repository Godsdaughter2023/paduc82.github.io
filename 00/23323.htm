ossiemac say:

Question. Why do you want to make all sheets visible when the workbook closes?

My reason for this question is that the user only has to hold the Shift key down during open and the Workbook_Open code will NOT run and it will leave all of the worksheets visible when it is opened without the need to enter a User Id and password.

The Workbook_BeforeClose event is not suppressed with the shift key so worksheets can be hidden on close but unfortunately it then asks if you want to save and if the user answers "No" then it is closed without re-saving and the sheets are not hidden and will still be visible on the next open. To AutoSave is dangerous because sometimes a User messes up and wishes to close the workbook without saving and re-open and start again and Autosave prevents them from doing this.

You need to understand that it is virtually impossible to totally secure the workbook. Like locks can be picked, the protection can be by passed. However, having said that, if you have users that can be trusted to do the right thing and they understand that the protection is basically to prevent them inadvertently performing incorrect operations and you can accept the limitations of the protection, then I am happy to work with you to achieve your end result.

    If you use Paste Special to paste as a picture, you should have Save as Picture on the shortcut (right-click) menu.

    Another approach is to “print” the page as a picture and save. You can get a free trial of Snagit, which installs a virtual printer:
