a. user say:

Hi there,

The spell-check function in Skype is based on your keyboard input language (not the display language or the keyboard layout). First make sure that spell-check is enabled in Windows 10:-

1. Right click the windows button, and Select 'Settings'.
2. Select 'Devices'
3. Select 'Typing'
4. Make sure that 'Autocorrect misspelt words' and 'Highlight misspelt words' is turned on.

To add words to the dictionary, please do the following:-

1. Windows Key + R
2. Type the following path:- %APPDATA%\Microsoft\Spelling
and hit Ok
3. Inside the Spelling folder, you'll see one or more folders depending on the languages you have installed. Select the folder that corresponds to your language.
4. Open the 'default.dic' file with notepad
5. All you have to do is add new words, line by line for each word, and then hit save!

I hope this helps!

jeovany cv say:

Hi Cheyenne

Here the macro for the 6 sheets

************************************************

Private Sub Worksheet_SelectionChange(ByVal Target As Range)


Dim MyRng As Range


''' This sets the ranges where the check mark symbol goes

Set MyRng = Range("D10:F39")


If Not Application.Intersect(Target, MyRng) Is Nothing Then

      With Target

      .Value = Chr(252)

      .Font.Name = "Wingdings"

      .Font.Size = 11

      .Font.Bold = True

      .HorizontalAlignment = xlCenter

      End With

End If


End Sub
