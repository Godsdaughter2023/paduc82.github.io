A Product Key consists of 25 digits divided into 5 groups of 5 digits.

bill smithers say:

If Windows 8/8.1 came preinstalled on your machine (Dell, HP, etc) which is an OEM License, you are allowed to reinstall it on the same machine for the life of that machine.

It makes no difference about you using Linux for a year.

You still own the Windows 8.1 OEM License for that machine until that machine dies.

There was no need to buy a Retail 8.1 License to use for a reinstall on that machine.

Reinstall using the original OEM License, then you are free to use the Retail License elsewhere.

If the retail License is activated, you may need to do a phone activation after it is removed and installed elsewhere
