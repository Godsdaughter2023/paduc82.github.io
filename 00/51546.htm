lemp say:

All versions of Windows XP are described as "Version 2002."  I'm not sure why that is, considering that XP was released in late 2001, but that's what Microsoft does.

You didn't buy an XP sp 2 CD in 2002 because Service Pack 2 for XP wasn't released until 2004.  I asked when you bought it because in my opinion, the longer after April 2014 a copy of Windows XP was purchased, the greater the chance that it is counterfeit.

Let's assume that you have a valid retail copy of Windows XP including sp 2.  You didn't mention whether your CD is XP Pro or XP Home, but that doesn't matter for installation purposes.

Use the computer that you're using to ask questions here and go to

http://www.toshiba.co.uk/innovation/download_drivers_bios.jsp?service=UK

Set the drop-down menus to look like the following and click "Search" --

The search should result in 21 files found.  Note that you can also download the User Manual for your laptop at the link at the bottom of the page.  You should do that.

You're going to have to do some guessing and some detective work in order to determine what driver files you need to download and install.  The User Manual may help, but you'll have to read through it carefully.  At minimum, you should download the following files:

  1. Chip Set Utility
  2. LAN Driver
  3. Sound Driver
  4. Touch Pad Driver

You'll also want one of the display drivers. After you install Windows, you can use Device Manager to determine if your unit has an Intel or ATI graphics chip.  You can either download both drivers now or wait and download the correct one later.

You can find out more about each of the other files by clicking the "more info" arrow at the right of each line.  Some may be useful.  You might consider the PC diagnostic tool, which is supposed to "display the basic information of a computer and diagnose its built-in devices."  This may be an easy way to determine what drivers to install.

With the same computer, use Internet Explorer (only) and go to http://catalog.update.microsoft.com/v7/site/Search.aspx?q=XP%20Service%20Pack

The first line of the search results should be Windows XP Service Pack 3 (KB936929).  Click the "Add" button at the far right and click the link at the top right of the page to "view basket."  Download the file.

Copy the files you download to a USB flash drive.  After Windows is installed, connect the flash drive to the laptop, copy the files to the laptop, and install the drivers.  Install the chip set drivers first, then reboot, then install the others.

There are several step by step guides to installing Windows XP.  Here are two reasonably good ones:

http://www.theeldergeek.com/xp_pro_install_-_graphic.htm

http://pcsupport.about.com/od/operatingsystems/ss/instxpclean1.htm#step1
