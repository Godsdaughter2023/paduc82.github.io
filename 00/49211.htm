I'm suprised that the same problem now also happens with discrete amd graphic cards. Before I thought it is APU specific. However, this might help:

This was a problem that occured on systems with an AMD APU (e.g. A8 7600, A10 7850k, Athlon 5350 ...). Since that, every driver (called "omega" drivers) made problems with JPEGs. First there were broken (e.g. black) thumbnails in Windows. The photo gallery crashed upon creating any thumbnails. You could use a workaround: deleting or renaming the JPEG-decoder files of AMD. Then Windows used automatically its built-in JPEG-decoder (a part of the so called "windows imaging component"). The last remaining problem was (and still is) the issue with the white or chess-board-like blocks in the image viewer.

Recently, the JPEG-decoder-bug (for APUs) was fixed by the last drvier-update (15.7 WHQL driver). But the issue with the white or chess-board-like corner remains. For discrete graphic cards, it seems to be a new problem. Apparently, you have to go back to the last stable catalyst driver (for APUs even before the omega-drivers!).

Solution 1. Run the troubleshooter for Windows Apps on your computer by clicking the run troubleshooter button on this link. 

Solution 2. Reset Windows Store to clear store cache. 

  1. Press Windows key + X on your keyboard.     
  2. Click on Command prompt (Admin).     
  3. Type wsreset.exe, and then hit Enter.     

Solution 3. Uninstall and reinstall Xbox app to update it. 

  1. Click Start, and then type Windows PowerShell on the search box.    
  2. Right-click, and then select Run as administrator.    
  3. Type the following command to uninstall Xbox app: Get-AppxPackage *xboxapp* | Remove-AppxPackage    
  4. Hit Enter.    
  5. Launch Windows Store and download Xbox App.    
  6. Open the game through Xbox App.    

If the issue persists, you need to register Windows Store again on your computer by following the steps below:

  1. Press the Windows key + X on your keyboard, and then select Command Prompt (Admin).
  2. Type the following command: PowerShell -ExecutionPolicy Unrestricted -Command "& {$manifest = (Get-AppxPackage Microsoft.WindowsStore).InstallLocation + '\AppxManifest.xml' ; Add-AppxPackage -DisableDevelopmentMode -Register $manifest}"
  3. Restart your computer, and then launch test your game again.
