Have a surface PRO 4, cannot get my attached keypad to work? - Microsoft Community

jamesbator say:

Have a surface PRO 4, cannot get my attached keypad to work?

is there a on off setting ?

ruz san say:

Hi James,

We may need to perform a number of troubleshooting to determine if the problem is within Windows or if it could already be the Type Cover/keypad itself. Were any changes done on the device before you encountered the problem? Does the LED light for the Caps or Fn key come up when pressed? Is the touchpad also affected?

First, disconnect the Type Cover and clean the pins and connectors using a soft, cotton cloth wet with rubbing alcohol. Once dry, reconnect the Type Cover and see if the accessory will communicate properly with the Surface.

If there are no changes, please download and run the Surface Diagnostic Toolkit. It will get the recommended driver and firmware udpates for your Surface and check for other software issues that could be affecting the device's performance. For more information and to download the tool, go here.

After running the Surface Diagnostic Toolkit, perform a forced shutdown. While the device is on, simply press the power button for a full 10 seconds or until the screen goes off. When you restart the Surface, the updates will be installed and the software will be reloaded properly.

It would also give us a better idea about the problem if you could boot into the Surface UEFI. To do that, switch off your Surface. Once the device has completely shut down, press and hold the Vol + button then press and release the power button. See if the Type Cover will work in this environment. Try using the touchpad or pressing the Caps and Fn keys again to see if the LED will come up. If it still does not work, go to Devices using the touchscreen. Toggle Type Cover port On/Off, then go to Exit and choose Restart now. Boot back into the Surface UEFI and enable the Type Cover port, then restart the Surface once again.

If the Type Cover works in the UEFI but not when you log in to Windows, the problem is the software. We would need to perform further troubleshooting. If it does not work in the UEFI, the problem is the accessory itself. It would be best to get a replacement. Please contact Microsoft Support or take the device to a Microsoft Store for better assistance.

Let me know how it turns out by responding to this post. I'll be notified when you reply and I'll get back to you the soonest I can in case you need more help.
