ricw-nw say:

OK Thanks.... unfortunately it is the wav. file format that is your problem. The reason for this goes back to the early days of home computing - wav files having been developed mainly for use for very small sound files (such as alerts - key-strokes etc.) and they were not designed to support extra data as a result. So when you add info in WMP it only goes into WMP's database... it doesn't attach to the file itself. When you rip a CD with the correct info a limited amount is attached (although again, it cannot be edited on the file itself) so this is why some appear OK and others do not.

What OS do you have on your present pc? (I'm looking for other possibilities!) - Cheers, Ric.
