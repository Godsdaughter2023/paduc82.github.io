a. user say:

Hi Aric,

Windows will handle multiple page files by maintaining a table where each memory page is located. Where a page winds up, it depends largely on which volume was least active when it got paged out.

You should ideally have your page files on your fastest drive. However, if you have a lot of RAM and rarely use your page files, you may want to put it on a mechanical hard drive to save space on an SSD.

Hope this information helps.

Windows Update not working and .NET 4 installation giving error code - Microsoft Community

colanzo say:

Windows Update not working and .NET 4 installation giving error code HRESULT 0xc08000222

Having tried everything on the Windows site and utilising live support (Ref 1225074352) to no avail the eventual solution was simplicity itself so I am sharing here in case other people have a similar problem.

Despite Windows Update telling me it was not working and showing no updates it was actually running in services and the help desk command line suggestion to stop it net stop wuauserv was just giving me an error. The eventual solution was as follows:-

  • Go to Services and Stop the Windows Update service. Type "services.msc" in the search program box to locate your Services, scroll down to find Windows Update, right click and select Stop
  • Open the Windows folder (via Start/Computer/Local Disc - normally C:) Rename the folder \SoftwareDistribution within Windows as \SDOld
  • Go back to Services and Start the Windows Update service.
  • Restart the PC
    • Having done this Windows corrected itself on bootup and Windows Update then showed 145 updates which happily installed okay.

    Hope this helps someone

    What will I lose if I did factory reset on my computer? - Microsoft Community

    faithefranklin say:

    What will I lose if I did factory reset on my computer?

    If I do a factory restore on my computer will I lose all of my programs, pictures, music, and other files?

    mike hall... always say:

    A factory reset wipes out EVERYTHING on the hard drive EXCEPT for the recovery partition.

     

    So, if your programs, pictures, music and other files were not on the computer when you first bought it, they will NOT be present after a factory reset.

     

    You must save all of your own data onto external media and then re-instate it after the factory reset. Programs you installed previously will have to be re-installed from their original media..

    andre da costa say:

    You will basically lose everything that did not come preinstalled with the computer, that includes your personal files and settings, applications etc.
