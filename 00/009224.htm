Am 19.11.2010 03:57, schrieb Kylie4me:

I prefer a formula that can go in a cell, if not this then a VBA program that can be used as a function.

A formula can be very huge and complicated, an UDF is easy for this. I've used this data from your example

B1 Deposit
B2 500000
B3 600000
C1 Withdrawals
C4 650000
E1 FXRate
E2 0.85
E3 0.88
E4 0.9
F1 Gain/Loss
F4 =Loss(B2:B4,E2:E4,C4)

and the UDF below in a normal module.

Andreas.

Function Loss(Deposit As Range, FXRate As Range, Withdrawals As Range) As Double
  Dim i As Long, Amount As Double
  'Get the value from first cell
  Amount = Withdrawals(1, 1)
  'Walk through all rows we have
  For i = 1 To WorksheetFunction.Min(Deposit.Rows.Count, FXRate.Rows.Count)
    'Decease amount
    Amount = Amount - Deposit(i, 1)
    'Is it negativ or the row above Withdrawals reached?
    If Amount < 0 Or Deposit(i, 1).Row + 1 = Withdrawals(1, 1).Row Then
      '+ (150,000 / 0.88)
      Loss = Loss + (Amount + Deposit(i, 1)) / FXRate(i, 1)
      '- (650,000 / 0.90)
      Loss = Loss - Withdrawals(1, 1) / Intersect( _
        Withdrawals(1, 1).EntireRow, FXRate(1, 1).EntireColumn)
      'Done
      Exit Function
    Else
      '+ (500,000 / 0.85)
      Loss = Loss + Deposit(i, 1) / FXRate(i, 1)
    End If
  Next
End Function
