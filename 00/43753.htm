We are volunteers willing to help you try to resolve this issue.

No, that is where all your data is stored. You can delete the files in Documents that you are sure you do not need however not the entire folder.

Try moving some of the contents to external USB drives.

=====

Start - Settings - System - Storage - double click the drive and it will display what files have been stored and the space used. Then you can double click those entries and continue to drill down for more results. For most of those entries drilling down will eventually open File Explorer to an individual folder.

=====

Run DiskCleanup - WinKEY + X - Command Prompt (Admin) OR Windows PoweShell - type in --> Cleanmgr.exe

Or on the Start Screen - type Disk Cleanup

Or Cortana Search - type Disk Cleanup OR type in cleanmgr.exe

=====

If that does not free up enough space then you need to move some data to other media (such as other hard drives/partitions, CD's, DVD's, or USB drives) and/or uninstall programs you don't use.

--------

These will show you how disk space is used.

WinDirStat - Free - use this to make sure you know what is taking up space.
http://windirstat.info/

http://www.jam-software.com/treesize_free/

http://www.jgoodies.com/freeware/jdiskreport/

Hi

Unfortunately, the older version of Skype v7.xx is no longer possible to download as there is a security issue in the program that installs the Skype software – the issue is not in the Skype software itself. That is the reason why we have removed this older version of Skype for Windows desktop from our website skype.com.

And with regards to contacts status, the green dot on the contacts is an indication that they are active or online. It is still visible on the other Skype platforms such as Skype for Windows 10 v12.xx and the new Skype v8.xx.
