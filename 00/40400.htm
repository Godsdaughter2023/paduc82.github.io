Windows 10 updater stuck at 45% - Microsoft Community

jonandie say:

Windows 10 updater stuck at 45%

Installed Win X with no problems.  When I checked Windows updater, it was stuck on 45% and has been for several hours.  Everything else seems to working properly.  Any suggestions?

a. user say:

Hi Jon,

Thank you for posting your query in Microsoft Community.

* Which update is under progress?(I.e. KB305....or name of the update)

In general updates will take more time if they are downloading high content files or due to slow internet connection. If you see it taking beyond the maximum time frame, then try the below steps:

Method 1:

Try to run the Windows update troubleshooter and see if it fix the issue:

* Press Windows Key then type ‘troubleshoot’ and hit Enter

* Select “View all” on the left panel and run “Windows update” troubleshooter from the list

* Click Advanced, ensure to check Apply repairs automatically and also select Run as administrator

* Click Next and finish the troubleshooting

Method 2:

Step :1

a. Press Windows Key + X.
b. Click on ‘Command Prompt (Admin)

Note: Click ‘Yes’ if it prompts for UAC(User Account Control).
c. Type following commands into the command prompt one after the other 

net stop wuauserv

net stop bits

net stop cryptsvc 


Keep Command Prompt window opened.

Step :2

Rename Windows Update folders 

At the command prompt, type the following commands, and then press Enter after each command:


ren %systemroot%\System32\Catroot2 Catroot2.old

ren %systemroot%\SoftwareDistribution SoftwareDistribution.old

Then type the following, commands and then press ENTER after each command:

net start wuauserv

net start bits

net start cryptsvc

Close the Command Prompt window and check if you can install the updates.

Hope this will address your query.

Moving Windows Live Mail and folders onto new computer - Microsoft Community

cbrophy say:

Moving Windows Live Mail and folders onto new computer

I have my email organized into folders.  I have a new windows 7 64 bit computer and want to move Mail live (with the email in its separate folders) to my new computer from my old.  Both computers are Win 7 - 64 bit and both have Mail Live 2012 installed.

Thanks!

Chris

¡firedog say:

Both computers are Win 7 - 64 bit and both have Mail Live 2012 installed.

If it's not too late, this is what I would have done in your situation: copy the entire store folder from the old computer and paste it into the equivalent location on the new one

%userprofile%\AppData\Local\Microsoft\Windows Live Mail

where %userprofile% is the folder called C:\Users\{Windows username} 

You could do the same with the program registry key to transfer program settings to the new machine. The key is

HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows Live Mail

so you could export that on the old machine and import it on the new one.
