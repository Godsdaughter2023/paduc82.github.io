Hi Moritz,

I'd like to make a summary for this case so that we are on the same page for communication. 

Now, you have created 2 accounts (with the Exchange Online licenses assigned). As a result, the each of the 2 accounts has the Exchange Online mailbox. Then, you got an error "Target user ‎'Moritz Bust' already has a primary mailbox" during the Hybrid move (AAD Connect).
