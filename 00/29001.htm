a. user say:

Hello Seasmus,

Thank you for posting your concern here in Microsoft Community and welcome to the Windows 10 Family,

We appreciate the effort you have put in to resolve the issue. I would be happy to assist you further from here.

If the computer beeps while you are typing, Toggle keys and/or Sticky keys are active. The Toggle Keys can emit a beep when the Lock keys are pressed. The Sticky Keys function can emit a beep when the Ctrl, Alt, Shift, and Lock keys are pressed. Use the following steps to disable Toggle Keys:

1. Type Ease into the Start Search field and select Ease of access from the populated list.
2. Scroll down the page and select Make the keyboard easier to use.
3. If selected, remove the selection from Turn on Sticky Keys.
4. Click Set up Sticky keys.
5. Remove the selection from Turn on Sticky Keys when SHIFT is pressed five times and click Save.

6. Remove the selection from Turn on Toggle Keys.
7. If selected, remove the selection from Turn on Toggle Keys by holding down the NUM LOCK key for 5 seconds and click on Save.

Additionally, I would also suggest you to follow the below steps and check if changes to the keyboard properties helps to resolve the issue.

1. Right click on the Start Menu and select Control Panel.

2. Click on Keyboard to open Keyboard properties.

3. Check if the provided options are set to your preference.

  • Set Repeat Delay to long.
  • Set Repeat rate to fast.

Hope the above information was helpful. If you need further information, feel free to write to us and we would be happy to help you.

Thank You.

balaji kundalam say:

The features you're referring to are called virtual machines and not virtual desktop. Virtual desktop refers to more than one desktop screen at a time in a single physical monitor. Virtual desktop feature is available in Windows 10. With built-in Hyper-V virtual machine manager, you can create virtual machines of Windows XP (not recommended as support ended), Windows 7 and Windows 8.1 in Windows 10. There is no support for Windows XP Mode from Windows 8 onwards, however.

So the user can install older version Windows as needed in the Hyper-V and the user has to manage the licenses, unlike Windows XP Mode.

Hi DM. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

Are you accessing Gmail in a browser? Try another browser. I would have several browsers on your task bar so there's always one that's best for the task. I use Firefox, Edge and Chrome in that order of preference. It's easy to share bookmarks between all of them in their settings under Import Favorites from another Browser. Set the preferred one as Default in Settings > Apps > Default Apps.

You can also add your Gmail and other web and Office mail accounts to Windows 10 Mail app which is a syn service that allows merging them all into one Inbox like on a phone, with a huge ad-free reading pane. You can email from that app too.

If it's the Mail app that's not working then try resetting it in Settings > Apps > Apps & Features > Mail and Calendar > Advanced Settings > Reset and Repair.

Restart PC. If that doesn't help then re-register app by typing CMD in Start Search, right click Command Prompt to Run as Administrator, copy and paste the following command into the box using the Paste tool from top left dropdown menu:

PowerShell -ExecutionPolicy Unrestricted -Command "& {$manifest = (Get-AppxPackage *windowscommunicationsapps*).InstallLocation + '\AppxManifest.xml' ; Add-AppxPackage -DisableDevelopmentMode -Register $manifest}"

I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.
