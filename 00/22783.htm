arvin meyer say:

You have fallen into the trap of using an autonumber for something meaningful. Record numbers in a relational database have no meaning other than to the record sets of the database.

So if you want to make a meaningful number, you'll need to ditch the autonumber and use another datatype such as a Long Integer. One that I've used is DMax, as in:

=DMax("[RecordID]","tblYourTable")+1

Add that expression to the DefaultValue property of the RecordID textbox. That will allow you to manually fill in deleted records with new ones, as well as make manual changes. The DefaultValue will fill in the next record as long as you don't make a manual change.

brian tillman [outlook mvp 2007-2019] say:

Send all and Send/receive all refer to accounts, not messages. They mean that Outlook either sends from all accounts where a message is waiting in the Outbox or receive from all mailboxes referenced by the accounts configured in Outlook.

Every draft message is a separate message. Each has its own recipient list. You can't automatically populate the same recipient to multiple outgoing messages in the manner you wish, at least not without scripting.

While you can't send multiple messages to the same recipient (without programming), you can certainly send the same message to multiple recipients. Mail merge is one way.

samwang_msft say:

Hi Dennis,

Based on your description, I understand the Outlook performance is reduced after switching from On-premises Exchange server to Exchange Online email service.

Generally, Cached Exchange Mode was introduced in the Outlook client to give users a seamless online and offline Outlook experience. While Online Mode works by using information directly from the server, and, as the name implies, it requires a connection. During the process, if the data file is too big, it will be affected by network bandwidth(or others) which is different from On-premises Exchange server that uses Internal network. Also, it seems you actually understand using Outlook cached mode could improve the experience, so I’d like to confirm if you have any concern about cached mode. For instance: out of disk space or others.

About the shared mailboxes, frankly, too many shared mailboxes would affect the Outlook performance as well. Therefore, I’d like to confirm if you have several shared mailboxes connected to your primary email account.
