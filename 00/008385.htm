jay freedman say:

The Page Color > Fill Effects > Picture feature has always had a bug that causes it to print small tiles. (The Gradient fill effect has a similar bug that causes multiple thin stripes.) There is a workaround but not a fix.

Instead of using the Page Color button on the Page Layout ribbon, open the document's header or footer and insert the image with the wrapping set to Behind Text. Drag the image to the desired size and location. If you want it to be tiled, insert multiple copies of the image and position them as needed. Otherwise, you can drag the image to the size of the entire page.
