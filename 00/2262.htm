bernie deitrick say:

    1) Copy this code.
    2) Press Alt-F11 to open the VB Editor
    3) Select "View"  then "Project Explorer"
    4) Double click the ThisWorkbook under your file name (expand your file's project if the project has a + in the box)
    5) Paste the code into the window that appears.
    6) Save the file as a macro-enabled .xlsm file.
    7) Make changes if needed

Then, when you enter a yes into any single cell in column K of any sheet, the entire row will be copied to Warranty Info (at the bottom).  This assumes that column A is filled with information.

Private Sub Workbook_SheetChange(ByVal Sh As Object, ByVal Target As Range)
    If Target.Cells.Count > 1 Then Exit Sub
    If Sh.Name = "WARRANTY INFO" Then Exit Sub
    If Intersect(Target, Range("K:K")) Is Nothing Then Exit Sub

    If UCase(Target.Value) = "YES" Then
    'Turn off events to keep out of loops
    Application.EnableEvents = False
    With Worksheets("WARRANTY INFO")
    Target.EntireRow.Copy .Rows(.Cells(.Rows.Count, "A").End(xlUp).Row + 1)
    End With
    'Turn events back on to get ready for the next change
    Application.EnableEvents = True
    End If
End Sub
