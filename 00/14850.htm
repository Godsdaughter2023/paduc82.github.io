sanwin2 say:

You have run foul of the problem which is the reason MS does not recommend or support saving to removable media or 'the cloud'.

When you save to USB there may well be sections of the open files which do not get saved/closed properly unless you correctly eject the USB stick after closing windows.  If a USB file is open and windows crashes, the buffered parts of the file may well not get saved and your file will be corrupt or possibly even lost.

Always save to hard disc and then copy to USB.

There are recovery tools available on the 'net which will try to recover files from USBs, so you may get lucky that way.

In future I suggest you set up Word's options to always save a backup copy.  This actually preserves a copy of the file as it was last saved.  The usefulness of that depends on the frequency of your saves.

  4. Click on the None button to deselect all styles.

  6. Save your document and distribute.
