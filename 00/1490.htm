stefan blom say:

Is the problem with double-clicking the icons for *.doc files? In other words, can you open the files if you start Word first? In that case, it's a file association issue, which can be resolved as follows: Exit Word if it is running. Hold the Windows logo key as you press R; in the Run dialog box, type winword /r and press Enter (note the space before the slash). Wait for the configuration to complete, and then try double-clicking *.doc files again. If it still doesn't work, take a look at http://www.gmayor.com/document_not_found.htm.
