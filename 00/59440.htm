thahaseena m say:

Hi,

¬∑        What is the exact error message that you receive and in what phase?

¬∑        What do you mean by computer jams?

¬∑        Are you able to boot to the boot to the desktop?

 

It might be that the files that are present in the disc are corrupted. I would suggest you to use a different disc and check whether you are able to repair. You can use your product key even if you use a different disc.

 

If you are able to boot to the desktop then, I would suggest you to follow the steps mentioned below.

 

Note:This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry, click the following link.

http://support.microsoft.com/kb/322756

 

Step 1: Open command prompt and type in regedit to get into the registry edtior. Navigate to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Class\{4D36E965-E325-11CE-BFC1-08002BE10318} and see if any Upperfilters or LowerFilters keys exist. If they do, back up the key and delete the upper and lower filters components.

 

Step 2:Close the registry editor and open device manager by typing in devmgmt.msc. The CD ROM drive(s) will still have an exclamation point. Right click on the drive (one by one if needed) and go to remove. Right click on the computer name at the top and go to "Scan for hardware changes". This should re-enumerate the drives. You should now be able to see them during setup.

 

Step 3:Type notepad at the command prompt and press enter. This will bring up a notepad window.

When notepad opens, click file and go to open. The open dialog box appears. Switch the file type selection to all files. Navigate to the CD ROM drive through the My Computer icon on the left side of the dialog box. When you get to the contents of the CD ROM drive, you will find the i386 folder.

 

Step 4:Right click on the folder and go to "copy". Navigate to the root of the C drive using the My Computer icon. Right click within the C Window and go to paste. This should begin the copying of the files from the CD ROM drive to the C drive.

 

Step 5:Wait for the files to copy. When this completes, close the dialog box, close notepad and the command prompt. When the next file copy error occurs, you can browse to the i386 folder on the C drive. You will have to do this with every file copy error that occurs.

 

This will of course not work if the disk is damaged. If the customer gets file copy errors during this process as well, consider that the CD may be damaged in some way.
