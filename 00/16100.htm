bernie deitrick, excel mvp 2000-2010 say:

SUMPRODUCT is a great way to string together conditionals - the trick is to figure out which conditions are needed ;-)

So, we want to only show TRUE on lines where the student has a pass, and only if it is their second pass.  Changing this:

=SUMPRODUCT(($B$2:$B2=B2)*($D$2:$D2="Pass"))=2

to this

=SUMPRODUCT(($B$2:$B2=B2)*($D$2:$D2="Pass")*(D2="Pass"))=2

will return TRUE only on the row where the second Pass for that student appears.

 

Bernie

bernie deitrick, excel mvp 2000-2010 say:

You need to add a column of formulas to your data table. For example, if your table is really only 4 columns wide, in cell E2:

=SUMPRODUCT(($B$2:$B2=B2)*($C$2:$C2=C2)*($D$2:$D2="Pass"))=1

copied down to match your data, will return TRUE for the first unique value, and FALSE for any repeated data. Then you can use that field, set the TRUE, as a page filter.

Then, to create a table of how many times each student has passed an exam, simply drag email to both the row and data areas of a pivot table.

You could also count how many types of exams a student has failed using this formula to return TRUE or FALSE for the first failure of each exam type.

=SUMPRODUCT(($B$2:$B2=B2)*($C$2:$C2=C2)*($D$2:$D2="Fail"))=1

To count the number of exam types a student has taken

=SUMPRODUCT(($B$2:$B2=B2)*($C$2:$C2=C2))=1

Then the difference between the last and first is how many tests on which each student can improve from fail to pass

Bernie.
