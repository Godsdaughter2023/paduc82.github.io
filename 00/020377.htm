rick rothstein, mvp - excel say:

I am going to assume the your actual file names will never have "dots" in their names (that is, when the code is done, the only dot will be in front of the extension). If that is the case, then give this macro a try...

Sub RemoveExcessFromFileNames()
  Dim FileName As String, Parts() As String
  Const Folder As String = "c:\my file\"     'Note the required trailing backslash
  FileName = Dir(Folder & "*.*")
  Do While Len(FileName)
    Parts = Split(FileName, ".")
    Name Folder & FileName As Folder & Parts(0) & "." & Parts(UBound(Parts))
    FileName = Dir
  Loop
End Sub

NOTE: The directory path that you are going to search in is assigned to the Folder constant (in the Const statement)... just make sure the path has a trailing backslash when you change it to your actual directory path.

a. user say:

Hi olderperson,

 

I can see that you wish to see the email addresses that you swept and to retrieve them. Let me help you with it.

 

Please be informed that the email addresses of the messages which you have blocked in the Sweep feature go to the blocked senders list. What I advise you to do is to go to the blocked senders list and remove them there. I have the steps below for you to do this:

 

1. Sign in to your account.

2. Click on the clog wheel located at the upper-right side of the page.

3. Select More mail settings.

4. Under Preventing junk email, click Safe and blocked senders.

5. Click the email address in the blocked senders list.

6. Click Remove from list.

 

If you wish to add them again in the contact list, perform the steps below:

 

1. Go to People page.

2. Click on New.

3. Fill in the information afterwards.

4. Click Save.

 

Let us know if issue persists and if you have other questions.
