How do you know if a scheduled scan is running? - Microsoft Community

shaggy10 say:

How do you know if a scheduled scan is running?

kosh vorlon say:

Hi Shaggy,

 

Here's a couple of things about scheduled scans. To run, the PC must be on and connected to the internet (it cannot be shutdown, in sleep mode, in hybrid sleep mode, or in hibernation). The power saving settings must be such that the PC does not shut down or go asleep during the scan (or it will be as if it never happened). Scans do not begin precisely when scheduled - some can start an hour or so earlier or later than the officially set time (depending on Microsoft's servers and the loads at that time). There is a way to use Task Scheduler rather than MSE to awaken sleeping computers, run the scans, and then shut them down when done, but it is somewhat complex. More on that shortly.

 

Also, go to settings and check scheduled scans and uncheck the box that prevents them from starting unless the system is on and idle (if you're using the system when it's scheduled or it otherwise isn't idle - perhaps downloading Windows Updates or doing something else - then the scan will not start and will be missed). If a scan is missed it does not start at the next available time, but waits for the next scheduled scan.

 

Go to Start / Control Panel / Administrative Tools / Services / scroll to Task Scheduler and make sure it is started and automatic. If not set it to automatic and restart it. If it won't start, click on the dependencies tab and do the same thing for each of the ones listed. If any of those won't start, continue with it's dependencies and do the same thing. When you get to the end, reverse course and Start the ones you couldn't because of the dependencies as you go until you get to Task Scheduler and hopefully it will now go to automatic and start. If you changed nothing read on. If you changed anything, reboot and see if that helps.

 

There is a way to use Task Scheduler rather than MSE to awaken sleeping computers, run the scans, and then shut them down when done, but it is somewhat complex. You can also use it to create different scheduled scans of different types since MSE only allows one choice.

 

Please see http://answers.microsoft.com/en-us/protect/forum/protect_start/running-mse-full-scan-as-a-scheduled-task/f627f5ce-0b74-4b1d-be64-d6ceeb4d9b02 - especially the later corrections from GreginMich.

 

The post from GreginMich here http://answers.microsoft.com/en-us/protect/forum/protect_start/scheduled-scans-not-working/fd769043-f67e-4990-8805-a02427f22dff will also be useful.

 

I'd also suggest the post from GreginMich here http://answers.microsoft.com/en-us/protect/forum/protect_start/i-would-like-to-schedule-mse-to-scan-pc-with/58474560-4cca-49c1-a51a-76ca9bc3b9f2

 

I hope this helps.  If not, post back with as much more detail as you can offer and we'll try a more comprehensive troubleshooting effort.

 

Good luck!

 

Kosh

vaseem ahmed say:

Hi Chad_s,

Internet Explorer's protected mode is a feature that makes it more difficult for malicious software to be installed on your computer.

In addition to helping protect your computer from malicious software, protected mode allows you to install wanted ActiveX controls or add-ons when you are logged in as an administrator.

Protected mode is turned on by default in the Internet, intranet, and restricted sites zones and an icon appears on the status bar to let you know that it's running.

In addition to warning you when web pages try to install software, Internet Explorer will warn you when web pages try to run certain software programs. You are warned when a software program would run outside of Internet Explorer and outside of protected mode, because that program might have more access to your computer than you prefer. This usually happens when a website is using an add-on to run a software program on your computer. If you are viewing more than one website at a time, check all open websites to be sure you know which one is trying to run the program before granting your permission. If you trust the program and want to allow it to run on any website, select the Do not show me the warning for this program again check box

For more information regarding protected mode check out Microsoft article given below:
http://windows.microsoft.com/en-US/windows-vista/What-does-Internet-Explorer-protected-mode-do

I strongly suggest that you don't turn protected mode off, but I will tell you how to turn it off.

1. Go to Tools and Internet options.

2. Click on the security tab and make sure to de-select 'Enable Protected Mode' and restart Internet Explorer.

It’s always safer to browse with “Protected Mode: On”. If you visit a page whose zone has Protected Mode enabled and you see the status is "Protected Mode: Off", you will want to close and restart a new instance of Internet Explorer to visit the page.

Hopefully, this post gives you a better understanding about when Protected Mode is enabled or disabled.

Hyperlink opens in browser

HI,

When I create an hyperlink to a picture coming from a file server and then run the slide show and click the hyperlink, the image is shown in my internet explorer rather then the Picture viewer (default program).

Is there a way to make PP open in default browser?

Tried in both PP 2010 and 2013 and in different Win 7 64bit laptops with Office 2010 32bit.

Any suggestions?

Thanks a lot in advance

steve rindsberg say:
You could use a Run Program action setting and supply the full path to the EXE file of the program you want to use to open the image, and also the full path to the image file; it's generally best to put both paths in quotes.

Or perhaps simpler, convert the image file to a format that the browser doesn't recognize; then PPT will be forced to use whatever your default settings are for that file type.  BMP, TIF, etc are reasonable choices.

PowerPoint will launch PNG, GIF and JPG files in MSIE.
