Private Sub Worksheet_SelectionChange(ByVal Target As Range)

This macro launches the userform Userform1 if cell A1 is clicked.

bob jones aka: cybertaz say:

Simply put, Object Linking & Embedding [OLE] primarily is a Windows OS technology which never has been fully implemented on Macs. In fact, it is severely restricted.

The only file types that can be embedded as objects [other than image files inserted as Pictures] are those listed on the main pane when you go into the Insert> Object dialog window... Essentially, those created by Word & Excel - note that even PowerPoint files are excluded. Unfortunately, if you click the From File button not only does it give the impression that you can embed any existing file regardless of file type, it further allows you to check the boxes for Link & Embed as Icon. However, neither of those options work unless the file is of one of the types listed on the prior pane of the dialog window.

Likewise, files which contain objects/icons embedded by Windows Office users will display when the document is opened on a Mac, but the OLE object cannot be opened & will not update unless it is one of the Office file types referred to above.
