daniel g. grau say:
Label templates are just tables with fixed-size cells, but Word only fills up the address. What you need to do is make one cell like you want it, then copy it over the rest of the cells. To do this:

  1. Place your cursor on the first cell before adding the picture with Insert > Photo > Picture from file or drag a picture onto the cell.
  2. Make sure the picture is attached to text in that first cell. To check this, display picture anchors with Word > Preferences > View > Object anchors.
  3. Select the complete cell with Table > Select > Cell and copy it to the clipboard
  4. For each of the columns, do Table > Select > Column and paste.

Hi,

When creating a Data Table, one should never link any cell in the row/column labels to the cell which is the change variable else this causes a circular reference.  Therefore the entries in the row/column labels should always be hard coded.  At max, they may be linked to each other i.e. if you have the row labels in D7:G7, then E7 can be a formula like =D7+0.01 (D7 is a hard coded entry).  This will not cause any problem.
