To go into break mode, press the CTRL key +PAUSE/BEAK

To monitor the progress of you sub, make use of the StatusBar.

 

Example adapted from Help

Sub DisplayStatus()
Dim oldStatusBar As Boolean
'Check is the statusbar is displayed
oldStatusBar = Application.DisplayStatusBar
'Display the statusbar
Application.DisplayStatusBar = True

'Write own info to statusbar
Application.StatusBar = "Please be patient..."
Workbooks.Open Filename:="LARGE.XLS"

'Give the control back to Excel
Application.StatusBar = False
'restore the previous view of the statusbar (on/off)

End Sub

Your sub becomes then

 

Sub Convert()
  Dim wshIn As Worksheet
  Dim wshOut As Worksheet
  Dim r As Long
  Dim m As Long
  Dim c As Long
  Dim n As Long
  Dim t As Long
Dim oldStatusBar As Boolean
'Check is the statusbar is displayed
oldStatusBar = Application.DisplayStatusBar
'Display the statusbar
Application.DisplayStatusBar = True

  Application.ScreenUpdating = False
  Set wshIn = ActiveSheet
  Set wshOut = Worksheets.Add
  Application.StatusBar = "Searching rows m ..."
  m = wshIn.Cells.Find(What:="*", SearchOrder:=xlByRows, SearchDirection:=xlPrevious).Row
  Application.StatusBar = "Searching columm n ... "
  n = wshIn.Cells.Find(What:="*", SearchOrder:=xlByColumns, SearchDirection:=xlPrevious).Column

paul de jong (slim applications) say:
The root cause is that SharePoint Online no longer automatically imports the EXIF properties into the image properties.
See https://support.microsoft.com/en-gb/help/4493033/automatic-exif-property-import-is-not-supported-in-sharepoint-online

If you need to extract EXIF metadata from images you need to find alternative ways to upload the content. There are PowerShell based solutions and browser-based solutions. My company has developed a solution named SLIM Companion Explorer that is browser-based. It extracts EXIF metadata like GPS coordinates, Date Picture Taken, ... and stores the values in SharePoint columns.
