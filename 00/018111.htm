guidors83 say:

Hi 

I would like to set a default font per each hierarchical level of activity

eg.

wbs 1  - Activity xx Font Arial 10

wbs 1.1 - Activity xx Font Arial 9

wbs 1.1.1 - Activity xxx Font Arial 8

john - project say:

GuidoRS83,

Well yes, it is possible but it will require some work. Other than summary lines versus non-summary lines, to do what you want you will need to custom format each of the WBS levels. To do that select all tasks at a given wbs level. Then go to Task/Font group and set the desired font. Repeat this for each wbs level.

The above could be automated with some VBA but that may be more work than simply doing non-contiguous select and set.

Keep in mind that the font characteristics are unique to a view, so if you set this all up for the Gantt Chart view and then switch to the Task Sheet view for example, you will have to set it all up for that view also.

One more thing. If you add any new tasks or switch WBS levels you will need to re-format the new/moved tasks.

Hope this helps.

John
