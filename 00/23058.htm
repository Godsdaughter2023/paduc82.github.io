ossiemac say:

What you can do is declare a public variable for the worksheet object and then set a constant for the worksheet name.

Then from within a sub you can set the object variable using the constant as the worksheet name. If you are calling subs from a main sub then you only need to do this in the sub that calls the other subs or you can set the object in individual subs. Effectively you will only need to change the worksheet name in the global Constant which I understand is what you want to achieve.

Public WS_Data As Worksheet
Const strShtName = "NEW DATA"


Sub Example()

    Set WS_Data = Worksheets(strShtName)
   
    MsgBox WS_Data.Name

End Sub

andreas killer say:
It's giving me a Run-time error '1004':
Unable to get the Transpose property of the WorksheetFunction class

I made a typo (vbCrLr should be vbCrLf), sorry for that, and I think I have not considered all situations.

Please try the code below.

Andreas.

Option Explicit

Sub ImportEmails()
  Dim olA As Outlook.Application
  Dim olNS As Outlook.Namespace
  Dim olF As Outlook.MAPIFolder
  Dim olM As Outlook.MailItem
  Dim Data
  Dim lRow As Long
  Dim CloseIt As Boolean

  'Get access to an already opened outlook
  On Error Resume Next
  Set olA = GetObject(, "Outlook.Application")
  If olA Is Nothing Then
    'It was not open, try to open
    Set olA = CreateObject("Outlook.Application")
    If olA Is Nothing Then
      MsgBox "Outlook is not accessable!"
      Exit Sub
    End If
    CloseIt = True
  End If
  On Error GoTo 0

  Set olNS = olA.GetNamespace("MAPI")
  Set olF = olNS.Folders("Personal Folders").Folders("Temp")
  lRow = 1
  For Each olM In olF.Items
    Data = Split(olM.Body, vbCrLf)
    Select Case UBound(Data) - LBound(Data)
      Case -1
        'Body is empty
      Case 0
        'Only one line
        Cells(lRow, 1) = Data(0)
        lRow = lRow + 1
      Case Else
        'More lines
        Data = WorksheetFunction.Transpose(Data)
        Cells(lRow, 1).Resize(UBound(Data), 1) = Data
        lRow = lRow + UBound(Data)
    End Select
  Next
 
  'Close outlook if it was not open
  If CloseIt Then olA.Quit
End Sub
