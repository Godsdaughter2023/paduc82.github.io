Cortana: Expectations, Usage, Tips, Work Arounds.

Technical Level : Basic

Summary
Cortana is a word in progress in the Windows 10 Technical Preview Build 10041. Currently only available for the US. As of Build 10041, Cortana now also works on the desktop in China, the UK, France, Italy, Germany, and Spain – This article includes some tips and information courtesy of Jason D and janetbarrie
Details

Hey Cortana and what you should expect

Hey Cortana General – What works what you should expect

Hey Cortana is a new feature that allows users to talk to give Cortana one shot commands without having to touch their computer. Hey Cortana works by waking up when the user calls its name instantaneously followed by an instruction. The one shot commands that a user can give Cortana, don’t allow for the user to wait for Cortana to respond to them before issuing a command. Hey Cortana will only work if Cortana knows what the user wants her to do when the user wakes her up. Below are some examples of what we mean by one shot, and what won’t work.

  1. Hey Cortana Play Music -> one shot all said at one time, no waiting period
  2. Hey Cortana what’s the weather in Seattle  ->  one shot gave Cortana a direct command
  3. Hey Cortana <let two seconds pass> tell me a joke -> NOT ONE SHOT WILL NOT WORK
  4. Hey Cortana <let two seconds pass> what is the weather -> Not ONE SHOT WILL NOT WORK

Having problems with “Hey Cortana” hearing me

Build 9926 does not automatically set the gain for microphones.  This issue will be addressed in a future release.  For now, you can manually adjust the gain by following these steps: 

  1. Right click on the speaker icon in the lower right hand corner of your desktop.
  2. Select "Recording Devices".
  3. Select your main input microphone.
  4. Speak into the microphone
  5. Note the sound level meter.  You want the level meter to peak every so often at the top of the bar but still show good bounce during your spoken text. 
  6. Ensure the microphone input is still selected and click properties.
  7. Click the Levels tab.
  8. If you have a microphone boost set it in the middle.
  9. Now adjust the microphone gain by moving the slider. 
  10. Click “Apply” and verify that the level meter is responding as described in step 5.

Changing the size of or hiding the Cortana/Search box

The search box on the taskbar can either be shrunk to an icon or completely hidden. In both of these cases, the search box will reappear when you press the windows key or the start button, allowing you to type to enter your search. To change the setting, please do the following:

  1. Right-click on the taskbar
  2. Mouse over “Search”

The settings for “Disabled”, “Show search icon” and “Show search box” can be found here.

Changing the default music player for “Hey Cortana play music” Changing the default music player for “Hey Cortana play music”

In build 9926, by default Cortana will play music in the Windows Media Player, rather than the player selected in the Defaults settings page.
The first time you use this command to start playing music you will be presented with a notification to change the default player.  If you miss this notification, there is no way to see it again.  To work around this issue, you can do the following:

  1. Launch Windows Media player.
  2. Click Create playlist, enter a name for the playlist, and hit enter.
  3. Under the Playlists section, right click the name of the playlist you just created and then select “Open file location.”
  4. In file explorer, right click the playlist file and select “Open with” and then select “Choose default program…”
  5. Make sure the “Use this program for all .wpl files” is checked, and then select the program you want the “Hey Cortana play music” command to use. (The selected program must support .wpl files)

Cortana says “I wasn’t able to connect to get you set up. Make sure you have a network connection, or try again in a little bit.”

This message is displayed when your Microsoft Account needs to be refreshed. A bug in build 9926 causes the Cortana application to be unable to refresh the account. To work around this issue, please do the following:

  1. Load the Store application. It also requires the use of the Microsoft Account.
  2. If the Store app prompts you to update your account information, please do so.
  3. Once the Store is loading properly, you can close it and relaunch Cortana.

Array-enter (press

=IFERROR(INDEX($D$12:$D$15, MATCH(A2 & D2, $B$12:$B$15 & $C$12:$C$15, 0), 1), "")

or

=IF(D2="elpd", IFERROR(INDEX($D$12:$D$15, MATCH(A2 & D2, $B$12:$B$15 & $C$12:$C$15, 0), 1), ""), "")

The first formula assumes that you misspoke, and Rohn at MSU should get the OT amount in the lookup table ($40).

But if you truly want to limit OT to the ELPD agency, the second formula ensures that.

Your VLOOKUP formula does not work for a couple reasons.

First, VLOOKUP(...,0) will match only the occurrence, if there is more than one match.  So VLOOKUP("Rohn",...,0) always matches the "Rohn ... $36.12" row.  (See my comment about "Rohn ... MSU" below.)

Second, there is nothing in your VLOOKUP or IF expression to limit the match to ELPD, if that is truly your intent. It worked for Smith and Jone only by coincidence because they are in the lookup table only with ELPD.

My first formula searches for the pair "RohnELPD" in the table.  I intentionally allow for matching "RohnMSU" because there is an overtime amount ($40.00) for the combination, so I assume Ronh should get that OT amount.

My second formula disallows matching "RohnMSU", if you truly intend not to recognize that as a match.

My formulas assume you are saving as "xlsx" or "xlsm".  Let me know if you are saving as "xls".  I depend on IFERROR, which used in an "xls" file.
