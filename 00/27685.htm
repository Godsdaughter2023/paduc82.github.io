dhookom say:

I would stop all work on the system and attempt to change the table structure to be normalized. There are lots of great resources on the web on what this should look like.

In the mean time to get your charts (I much prefer vertical bar charts to pie charts) you can use a normalizing union query to transpose your data:

Select [Job Name], "Total Job Price" as CostType, [Total Job Price] as Amount
FROM YourUnnormalizedTable
UNION ALL
Select [Job Name], "Total Labor Costs", [Total Labor Costs]
FROM YourUnnormalizedTable
UNION ALL
Select [Job Name], "Total Materials Costs", [Total Materials Costs]
FROM YourUnnormalizedTable
UNION ALL
Select [Job Name], "Total Profit", [Total Profit]
FROM YourUnnormalizedTable
UNION ALL
Select [Job Name], "Department 1 Labor", [Department 1 Labor]
FROM YourUnnormalizedTable
UNION ALL
Select [Job Name], "Department 2 Labor", [Department 2 Labor]
FROM YourUnnormalizedTable
UNION ALL
Select [Job Name], "Department 3 Labor", [Department 3 Labor]
FROM YourUnnormalizedTable
UNION ALL
 - etc -
