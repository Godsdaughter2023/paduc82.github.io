I don't have any question about whether or not the devices you are using should work.

The "renaming" is a Vista issue but it should not affect the sync.

 

The names PC1 and PC2 were the names for your computers.  You can change the names right from the device or from the PC.

 

From the PC:

From the Device

With the device NOT connected

Open Active Sync, press Menu then Options

 

Highlight the NAME of the computer, Press Settings and rename the device

So I would do it ... you really want to know if it will work.  Sync one computer then go in and change the name of that computer to reflect work or laptop.

http://social.technet.microsoft.com/wiki/contents/articles/31962.create-onedrive-for-business-usage-report-using-powershell.aspx

Regarding Exchange Online, if you would like to check the usage of all users' mailbox, for your references, please, also use powershell,

 Get-MailBox | Foreach {Get-MailboxStatistics -Identity $_.UserPrincipalName | select DisplayName,TotalitemSize} | Export-csv D:\mailboxusage.csv

Hi KERATL

Are you setting the foreground color on a chart object?

If so, you can set colors with their RGB values, if known

.Format.Fill.ForeColor.RGB = RGB(0, 176, 80)

Or use a preset color index:

.Format.Fill.ForeColor.ColorIndex = 5
