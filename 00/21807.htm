Word 2003 document icon files - Microsoft Community

alex35.w say:

Word 2003 document icon files

What is the location of the word 2003 document icons in a windows 7 pc? Like when changing the icon for a folder or a new internet shortcut, i'd like to know where i can find the folder containing all the Office icons. Plz help!

Thanks, Alex :-D
jay freedman say:
The icons aren't in .ico or other graphics files. They're stored as resources in an .exe file. You can find the location by looking in the registry at the key HKEY_CLASSES_ROOT\Word.Document.8\DefaultIcon. The value of the Default item looks like this (it may be different on your computer):

C:\windows\Installer\{91140000-0011-0000-0000-0000000FF1CE}\wordicon.exe,1

The folder C:\Windows\Installer is hidden, but you can get there by typing or pasting the path (up to the closing curly brace) into the address bar of Windows Explorer. The same folder also contains .exe files for the other Office applications.

The ",1" at the end of the value means that the icon for Word 2003 document files (known in the registry as the class Word.Document.8) is the first one stored in the wordicon.exe file, which contains about a dozen different icons.

You can see these icons if you use Visual Studio to open wordicon.exe (it opens only the resource part of the file) and double-click each numbered entry under Icon to open it in the icon editor window. Each icon comes in a series of different sizes and color depths. I wouldn't advise making any changes, but you can copy any particular icon, paste it into IrfanView (www.irfanview.com), and save that as an .ico file.

If you don't have Visual Studio, you can download the free Express version at http://www.microsoft.com/visualstudio/en-us/products/2010-editions/express. I think that version can open the resources of an .exe file. IrfanView can also open the .exe file, but it will show only the first icon in the smallest size.

Contractions are (or used to be) discouraged in formal academic writing, so this is a "style" issue. Go to Office Button | Word Options | Proofing. Under "When correcting spelling and grammar in Word," you have several options:

  • You can opt out of grammar checking entirely by clearing the check box for "Mark grammar errors as you type" and/or "Check grammar with spelling."

steve rindsberg say:

There are some new commands in 2011 to do this sort of thing. 

Select two shapes, Ctrl+Click and look at the options on the Grouping menu flyout.

For example, by selecting a rectangle, then shift-clicking to select a circle that overlaps the corner of the rectangle and then choosing the Subtract option, you Subtract the shape of the circle from the rectangle where it overlaps.

The first selected shape is the one that remains; the second selected shape gets subtracted from the first shape.
