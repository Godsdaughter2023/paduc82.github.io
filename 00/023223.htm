suzanne s. barnhill say:
  1. Select an entire chapter (let's say it's Chapter 1).
  2. On the Insert tab, in the Links group, click Bookmark.
  3. In the Bookmark dialog, type the name for your bookmark, such as "Chapter1" (without the quotation marks). It cannot contain spaces but may contain underlines, so you could also use "Chapter_1."
  4. The Index dialog (References | Index | Insert Index) does not make provision for adding switches, so you have to do this manually. Press Alt+F9 to display the INDEX field.
  5. The default INDEX field in English will be { INDEX \c "2" \z "1033" }; modify this to be { INDEX  \b Chapter1 \c "2" \z "1033" }. If your field looks different because you used different options wrt columns, etc., the principle is still the same: you must add the \b switch with the bookmark name.

If you want to create an index for a specific portion of a document, you can enclose that portion (chapter) in a bookmark and then insert an INDEX field that references that bookmark, using the \b switch. See this reference article.
