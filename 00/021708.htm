jay freedman say:
In my copy of Word 2010, when I check or uncheck that option, that setting applies to all documents I open. In other words, despite the heading "Document specific settings", this setting is not document specific but program-wide.

If that isn't your experience, it's possible to force the setting for each document you open. If you already have an AutoOpen macro and an AutoNew macro defined in your Normal.dotm template, then add this line to each of them:

   Options.WarnBeforeSavingPrintingSendingMarkup = False

If you don't have these two macros yet, then create them by pasting this code into a new module within Normal.dotm in the macro editor:

Sub AutoOpen()
    Options.WarnBeforeSavingPrintingSendingMarkup = False
End Sub

Sub AutoNew()
    Options.WarnBeforeSavingPrintingSendingMarkup = False
End Sub

Read Graham Mayor's page at http://www.gmayor.com/installing_macro.htm for background.

ronaldoonenil say: Not sure about a standard Excel formula but a User defined function as below placed in a module will do it.
If your 1235 is in A1 then use 

=Unused_Digits(A1)

Function Unused_Digits(lNum As Long) As String
    Dim strDigits As String
    Dim strNum As String
    Dim i As Integer
    
    strDigits = "0123456789"
    strNum = Str(lNum)
    For i = 1 To Len(strNum)
        If InStr(1, strDigits, Mid(strNum, i, 1), vbTextCompare) > 0 Then
            strDigits = Replace(strDigits, Mid(strNum, i, 1), "")
        End If
    Next i
    Unused_Digits = strDigits
End Function
