mister wizard say:

Hello trotskyicepick,

Windows Search does not include a searchable property for bit rate, but Windows Explorer can show the rate on a view for Mp3 files.  So, you can sort and filter search results by bit rate and group your files that way.

For example, if the files in question are in your Music folder and its sub folders, click on Music in the left pane of Windows Explorer.  Now paste this query into the search window:

ext:mp3

Now right-click on any right pane column header and add the bit rate property to the view. Click on the Bit Rate column header to sort the view by rate.  Now click on the drop-down arrow on the same header and choose "Near CD Quality" from the list.  This will filter the search results so that only files recorded at 128 or above will appear.

Unfortunately, Vista does not support the M4a format, so bit rates for those files cannot be shown in Windows Explorer.
