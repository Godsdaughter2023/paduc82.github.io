jllatham say:

Are you specifically looking for a worksheet formula, or would a UDF work?  If so, then below, with worksheet use as

=FindMaxOfTypeRow(C1)

Change the 3 Const declarations as needed, also you could change the type of the function and variable MaxFound to fit the types of count expected to be used (Integer, Long, Single, Double).

 

Function FindMaxOfTypeRow(typeToMatch As Range) As Long
  'change as needed
  Const ListColumn = "A"
  Const CountColumn = "B"
  Const FirstListRow = 1 ' no labels?
 
  Dim TypesList As Range
  Dim AnyTypeEntry As Range
  Dim MaxFound As Double
  Dim MaxRowFound As Long
 
  Set TypesList = Range(ListColumn & FirstListRow & _
   ":" & Range(ListColumn & Rows.Count).End(xlUp).Address)
  MaxFound = -10000 ' initialize with very small number
  For Each AnyTypeEntry In TypesList
    If AnyTypeEntry = typeToMatch Then
      If Range(CountColumn & AnyTypeEntry.Row) > MaxFound Then
        MaxFound = Range(CountColumn & AnyTypeEntry.Row)
        MaxRowFound = AnyTypeEntry.Row
      End If
    End If
  Next ' end AnyTypeEntryLoop
  FindMaxOfType = MaxRowFound
End Function

I'm thinking about Worksheet function formula to achieve the same, but usually you're better at that than I am (as are so many others here). 

[Edit] And I see that JP Ronse has already done exactly that while I was typing!  Just be sure you either always start at row 1 for your list range or include a "row adjustment" value in the formula (if your list was from A2:A13 vs A1:A12, his formula will work as written, but if you set it up as
{=MATCH(MAX(IF(A2:A13=C1,B2:B13)),B2:B13,0)}

It would return 5, which would be off by 1 row, so it would need to be written as

{=MATCH(MAX(IF(A2:A13=C1,B2:B13)),B2:B13,0)+1}

for that particular case.
