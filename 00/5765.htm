  1. Turn on the Hidden Characters by pressing the ¶ button either on standard or formatting  menu bar (depends upon where its been placed).

  3. scroll to end of document.

  6. That selects the entire document except the last ¶.
  7. Next do command - C to copy the document.

  9. Paste the coped text into new document (command -V).

brian tillman [outlook mvp 2007-2019] say:

I'm not sure how trustworthy Comodo is any more. They've been doing some things lately that are not very civic-minded.

If you access Outlook.com using a web browser, I don't think that you can employ a certificate. The Outlook.com service doesn't have any way to store a certificate for you or any way to sign or encrypt the message. You need a mail client like Outlook 2016 to do that.

When you export the certificate, the first screen of the wizard should be a screen asking if you wish to export the private key. If you don't see that, then something went wrong in the import process. Without the private key, the certificate isn't any good for decrypting messages or for signing messages. My inclination would be to revoke the certificate you have and request another using IE, if you originally used Chrome.

Here's a page from Comodo explaining the export process: Email Certificates FAQ
