To tell if an application 64 bit or not in a 64 bit version of Windows, right click the Taskbar, click Start Task Manager > click the Processes tab

Any entries you see denoted by the *32 are 32 bit applications. Those that do not have it are 64 bit. So if you see winword *32 that means its a 32 bit version of Microsoft Word installed. Please note, you will only see this on the 64 bit version of Windows.

32 bit versions of Windows only run 32 bit software.

aj-700 say:

Possible workaround:

  1. Position the image away from where you want it on the slide.
  2. With the image selected, select "Crop".
  3. While in "crop mode" you can drag the image and it will be partially transparent.
  4. When you have the image lined up where you want it select the "picture" icon at the top of the Format Picture sidebar.
  5. Expand the Crop menu.
  6. Under Picture position, Offset X and Offset Y will show how much you need to add to the Left and Top values (under Crop position) respectively.
  7. When the Crop position has been correctly aligned with the Picture position, Offset X and Offset Y will both have a value of 0.
