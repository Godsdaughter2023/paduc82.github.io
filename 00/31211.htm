crownmountain say:

I have this situation, I have one system with 2 users and the File History drive is connected to that system.  What I did to make this work was as follows:


To do this you have to get into the advanced option of file history via the control panel on the desktop for the user that has the hard drive selected and it is recording files.


On the advanced Window for file history select "recommend this drive"  Then close the windows and log out.


Now log into the user that is not recording file history

Click on select drive and choose network location (do not think Homegroup has to be on for this although I do have it on for other reasons)


Select the PC by name and double click.  Then you should see a folder named "FileHistory_1" or similar


Double click to open and then click select on the bottom of the dialog box (you should also see a folder with the name of the other user within that same folder.  Do not select it just be aware that you are now on the correct drive.)


As you then close back out to the main File History control panel window you should see that it is all green and working well.


Hope that helps.
