As far as removing write protection from the external device is concerned, you can very much do it from registry editor.
 
Registry disclaimer
To do so: Important this section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. Back up the registry
http://windows.microsoft.com/en-US/windows-vista/Back-up-the-registry 
 
Steps to remove write protection from external drive:

a) Go to registry editor.
        a) Click on start orb and type regedit in the search field.
        b) Select regedit from the programs list.

b)  Now that the registry editor is open, navigate to the following key:
 
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\StorageDevicePolicies
 
c) If the key exists, go to step g.
 
d) In case StorageDevicePolicies doesn’t exist, you wil need to create a new one.
 
To do so:
      a) Right click on the blank field and select New - Key.
      b) Name the new key as StorageDevicePolicies.
 
e) Once this is done, highlight StorageDevicePolicies, and then create a New DWORD (32-bit) Value named as WriteProtect.
 
f) Double click on WriteProtect, and set its value
 
g) If the key already exists,  double Click on WriteProtect and change its value from 1 to 0 to enable write access.
 
h) Close the registry editor.
 
Check if it helps.
