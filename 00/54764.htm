Boot from the CD back into the recovery console & Login.

ren winsrv.dll winsrv.old

Note that you receive a message that states that the file was not found, ignore the message and continue to the next step.

3.Go to the CD-ROM or DVD-ROM drive by typing the drive letter of the CD drive followed by a colon, and then press ENTER. For example, if the drive letter for your CD/DVD drive is D:, then type D: and press Enter.

4.Type

cd i386 and then press ENTER.

5.Type

expand winsrv.dl_ %Windir%\System32\winsrv.dll then press ENTER. Make sure that there is a character space between “winsrv.dl_” and “%Windir%.”

6.Restart the computer, and see if the error is resolved.

What's wrong with Windows 10 Version 1803 - Microsoft Community

promusaster say:

What's wrong with Windows 10 Version 1803

This will hopefully be a big list that is human readable. The feedback section does its job for the developers, but users might not look through it to see what's wrong with the latest version. I can only state what I have come across so far, so I hope that the larger base of users can add on to it. All these tests were done on a fresh install to assure that no malware altered anything.

  1. 3rd Party Antivirus:  After installing a few antivirus solutions I noticed that they were all having some issues. Bitdefender was unable to turn on its behavior shield, as well as Malwarebytes's anti-exploit protection. Manually turning these features on did nothing, as they immediately switched off. Depending on the user, this might limit them to Defender because they may assume the 3rd party solution is not working at all. !!All issues in this category have been fixed after the latest OS/Image updates!!
  2. Fluent Design: The new fluent design may bring better visuals but it has a surprisingly high cost. When viewing the application preview, most will stutter when loading on occasion, the exception is file explorer. While viewing videos (only seen in edge so far) and one of these preview tabs, the video will stutter, stop, lock, or rarely cause a browser crash. It can also cause the start menu to load slower while also stuttering itself. Fluent design also demands a lot out of the hardware, ranging from 10%-20% on average, even though the effects used are not as demanding when used in other cases. This can decrease framerate in all tasks, use more power on mobile devices, and generally make the system hotter/nosier than when at non-fluent idles (1% CPU and ~4% GPU vs 14% CPU and ~25% GPU.
  3. 3rd Party Apps: Specifically, virtual machines. The new core isolation for Windows 10 brings security features that are meant to fight Specter and Meltdown, but when enabled, not only can it not be turned from GUI, it also makes it so you can launch virtual machines. This extends to any software that uses virtualization but most only use that virtual machines. The also, semi-newly added Defender anti ransomware, when enabled will not allow access to certain folders, so if an app is not whitelisted ahead of time during install, it may/will break.
  4. Hardware: Although Windows can not directly alter hardware, it's just code after all, it can alter access to hardware capabilities. On certain laptop models, perhaps desktop too, function key commands will not work. An example is my acer aspire, where fn + brightness up/down display the graphic but do not change brightness. Other key combinations may not be affected, such that only certain combinations are no longer working. The monitoring for hardware also seems to off by a large, and I mean up to 90% large, percentage. https://1drv.ms/u/s!AuNOKkIhIQqah5o1jEfTR0g1FlBY0A
  5. Edge: The replacement for IE, in the latest update, like one of the previous versions, certain sites will freeze and crash. A test with Gamejolt found that it crashed 3 out of 4 times. Other sites can have this happen too, it only happened to Youtube once. There were also a few issues with uploading and how the browser rendered text, like this moment, the text I am typing is being typed on top of the previously written line... While watching videos there may also be sound artifacts, points where it randomly can't decode audio, replacing the normal sound with a crackling version.
  6. Typing: The new feature that auto corrects typing, from my last tests, showed that it did not work, and in certain places would not show up at all (the preview of the word you are trying to type).
  7. Cortana: With this update, cortana was moved to the action center, though no one would know since she doesn't say anything other than the normal "link your phone" or "would you like to review...". She no longer points out news, weather, or anything that made her remotely useful. Yes, she can still search for apps on your machine, but in this update, she may not find it. After installing VMWare, she was unable to find it, even though it was at the top of the jump list. Combine these issues with the fact she uses 100+ MB RAM and you essentially have bloatware/Malware that "looks at the sites you are viewing." When using the "Ask Cortana" option in Edge the pane she opens will be blurry.
  8. Misc: These are just a few random things that aren't linked with anything specific, but can cause problems. Random BSOD on internet access. Nightlight gets stuck on/off. Apps may get stuck in an infinite install error loop. Windows update failing to work https://1drv.ms/u/s!AuNOKkIhIQqah5o2BMLMvCiM4vnnjg and the rest is up to the public to find...

malkeleah say: Use crossover cable. Temporarily assign static IP addresses on the same subnet (ex. 192.168.1.2 and 192.168.1.3). Allow sharing per:

http://technet.microsoft.com/en-us/library/bb727037.aspx

If this is a one-time thing, it would probably be easier to pull the hard drive out of the laptop with the data and put it into a USB external drive enclosure and just attach it to the target laptop.
