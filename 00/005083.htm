The Home Use license, like the full retail license, permits you to have Office Pro Plus concurrently activated on two computers that you use,.

  1. Back up PST file from old computer while Outlook is shut down.
  2. Start Outlook on new computer but do not configure mail client yet.  Outlook then creates a small pst file.
  3. Shut down new Outlook
  4. Delete PST file from new computer and replace it with the pst file from the old computer.
  5. Restart new Outlook and it will accept old pst file.  Old pst file becomes new.
  6. Reconfigure mail client(s), and new mail is added to the existing mail in the pst file.
  7. Continue using Outlook with all data in place.
