You don't have a program on your system capable of opening a .dat file and you won't be able to get one either because these can actually be just about any type of file.  In some cases, you can open them if you know the actual file type and in others you can only open them if you're using the same email program AND you are the original sender!  It's a problem without a clear solution other than trial and error with best guesses (or again, contacting the sender to find out exactly what was attached).

If the file is winmail.dat then it's caused by the email programs being used.  Here's information about that: http://facstaff.gpc.edu/~jbenson/resource/winmail.htm.  Sometimes this happens with attachments in email programs, especially when each person is using a different email program (or with some email programs in particular like Yahoo mail).

Here's an articler on opening .dat files: http://www.addictivetips.com/windows-tips/open-dat-file-extension-in-windows-quickly/.  It comes down to figuring out what was the original program that produced it and what was its original file type (so you can change it and use the correct program to open the file).  One way is to ask the sender of the email what the .dat file is - they may give you a completely different filename and then you'll know and can either just change the extension, convert the file, or obtain a program that can open that file type.

Here's another article on how to open .dat files in Vista: http://www.online-tech-tips.com/computer-tips/how-to-open-dat-files/.  It says much the same as the prior article but provides some additional information and hints for finding the correct file type.

Here's a third article on opening .dat files: http://pcsupport.about.com/od/fileextensions/f/dat-file.htm.  Among other things, it suggests trying Notepad first since that program opens a great deal of files and the .dat file might just be text.

From your post, it seems that it isn't an Adobe PDf file - so that's one down.

I hope this helps.

Change your default printer

 

If more than one printer is connected to your computer, you can choose one to use by default. Choose the printer you will use most often. That way, when you print using Windows or your programs, you won't have to select a printer each time.
