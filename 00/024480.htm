-----------------------------

Change the default bullet or numbering style
Show All

To use a customized bullet or numbering style in all your presentations, store them in a template file, blank.pot, that Microsoft PowerPoint will use to start all new presentations.

  1. On the Standard toolbar, click New .
  2. On the View menu, point to Master, and then click Slide Master.
  3. Select each level of body text (you can select all levels at once) for which you want to change the bullet or numbering style, and then make your changes.

How?

  1. On the Slide Master View toolbar, click Close Master View.
  2. On the File menu, click Save As.
  3. In the Save as type box, select Design Template; in the File Name box, type blank.pot, and then click Save.

 Note   Be sure to save the file to the default folder for templates; this is the folder that PowerPoint defaults to when you select Design Template as the file type in the Save As dialog box. If you save the blank.pot file to a different folder, PowerPoint will not find it.

  1. Close the template, and, on the Standard toolbar, click New.

A new, blank presentation opens with your new bullet or numbering style as the default.
