krystlewright_msft say:

Hi Adnan,

To answer your question as to why you got the error message when performing a Cutover migration to Office 365, the answer is Yes.

Because the email migration service can't detect whether on-premises Active Directory groups are security groups, it can't provision any migrated groups as security groups in Office 365. If you want to have security groups in Office 365, you must first provision an empty mail-enabled security group in Office 365 before starting the cutover migration.

Additionally, this migration method only moves mailboxes, mail users, mail contacts, and mail-enabled groups. If any other Active Directory object, such as user mailbox that is not migrated to Office 365 is assigned as a manager or delegate to an object being migrated, you must remove them from the object before migration.

    1. Locate the Word 2003 executable on your hard drive. In Windows 7, mine is at C:\Program Files (x86)\Microsoft Office\OFFICE11\WINWORD.EXE.
    2. Copy the path from the address bar in Windows Explorer.
    3. Press Windows Key + R to open the Run dialog.
    4. Paste the path you copied and edit it to add WINWORD.EXE.
    5. Add quotes around the path and add and the /r switch (preceded by a space) so that you have
      "C:\Program Files (x86)\Microsoft Office\OFFICE11\WINWORD.EXE" /r
    6. Press Enter. You'll get a message about waiting while Windows configures the program, but Word will not open.

    This will make Word 2003 the default file handler for all .doc (and .docx) files. To open .docx files in Word 2016, you'll need to open Word 2016 first. When the "reregistration" finishes, you should be able to double-click a .doc file in Windows Explorer or click a file on the Word 2003 jump list and open it in Word 2003.

    Quotes to Microsoft Word - Microsoft Community

    massaabal-aloosy say:

    Quotes to Microsoft Word

    Hi guys,

    I'm a PhD student and I have to transfer many quotes in the books I've read to the Word document. And because they are so many, I was wondering if there is an easy way instead of writing each and every single word of the quotes to transfer them to the Word document. 

    Thank you so much for any tips you might have

    rohn007 say:

    I guess the "best" way is to find them online and copy/paste them.

    For paper, you have to "scan" them, then use "OCR" (Optical Character Recognition) software to convert them into text you can edit.

    Word 2013/2016 claim to do OCR, but they don't. Their OCR depends on a special format PDF file.

    On the other hand, OneNote (2010, 2013, 2016) does do real OCR.

    So, the most likely way is for you to use a camera, say in your phone, to capture an image of the quote on paper, the import that image into OneNote.  Right click on the picture and select the "Copy text" option. Now you can paste the extracted text into Word. Depending on the quality of the original document and of the picture, you may have to correct some letters or words that are incorrect.

    Or, if you have an "all-in-one" type printer that includes a built in scanner you can use it to do the scan that is then imported into OneNote.

    If you are not satisfied with the quality of OneNote OCR translations, you can also use a 3rd party OCR tools which may do a better job of translating the scans.

    That generally means there are currently no Skype numbers available in that particular state.  Sometimes this is due to lack of participation, but more often due to consumption.  Skype numbers are on a first-come, first-serve basis.  If certain states or area codes within a state are more popular, they may be consumed as fast as they are released by the local telcos Skype obtains them from.  So this has nothing to do with state or country recognition and everything to do with availability and the current supply and demand.  All you can do is frequently check the Skype number availability list to see if one becomes available.  When it does, you can purchase it.
