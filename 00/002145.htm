Turn a multi page word doc into multiple single word docs, with the - Microsoft Community

chrisgoff say:

Turn a multi page word doc into multiple single word docs, with the page title used for the file name

Hi All,

we need to turn multi page word docs into multiple single page word docs with the title of the of the individual page saved, becoming the file name. Any ideas how we can go about this using a macro?


doug robbins - mvp office apps & services (word) say:

Use the following code

Dim Counter As Long, Source As Document, Target As Document
Dim rng As Range
Dim DocName As String
Set Source = ActiveDocument
Selection.HomeKey Unit:=wdStory
Pages = Source.BuiltInDocumentProperties(wdPropertyPages)
Counter = 0
While Counter < Pages
    Counter = Counter + 1
    Source.Bookmarks("\Page").Range.Cut
    Set Target = Documents.Add
    With Target
        .Range.Paste
        Set rng = .Range.Paragraphs(1).Range
        rng.End = rng.End - 1
        DocName = rng.Text
        .SaveAs FileName:=DocName
        .Close
    End With
Wend
