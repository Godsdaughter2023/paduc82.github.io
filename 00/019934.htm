andreas killer say:
At first: I hope you have uninstalled 2003 before you've installed 2010!? If not, I recommend to uninstall both, then install 2010 only. Often if you have 2 Office versions on the same PC you get trouble at some points. (That's my experience, there are also people that have both versions and no trouble..)

Okay, back to your issue: A simple scenario to recreate the behavior:

Open Excel 2010
Make a new file
Write XLS into A1
Save the file as 2003.XLS
Make a new file
Write XLSX into A1
Save the file as 2010.XLSX

Important: Be sure you select the correct file type in the Save As dialog!

Close both files and reopen.

The first thing you should note is that the 2003 file shows Compatibility Mode inside the title bar. From this you can see if you are working with the old file type XLS or with the new file type XLSX (or XLSM). Beside from all others: It is not necessary to convert your old files to the new file type!

But before we get lost in the details, here is the article, please read it later
https://support.microsoft.com/de-de/help/262997...

Now go into the 2003 file
Copy row 1
Switch to the 2010 file
Right-click row 2 and choose Insert

As you see it works.

Copy row 1 from the 2010 file
Switch to the 2003 file
Right-click row 2 and choose Insert

And you get an error. Why is that?

If you go into the 2003 file, select B1, then press END, RIGHT-ARROW you jump to the last column and this formula IV1: =COLUMN() returns 256 (I bet you know that an XLS file has 256 columns, but I have to be sure.)

Now do the same in the 2010 file and you see that the new file type has more columns XFD1: =COLUMN() returns 16384!

The same is for rows, XLS has 65535 but XLSX has 1048576 rows.

And that's the problem, the new file type has more columns which would not fit into the old file type. So if you face issues like this with your old files:

Open the XLS file
Save it as XLSX (or XLSM if the XLS contains macros)
Close and reopen the XLSX file.

That's it. After that you can work with your files as usual.

If you run into other issues, please refer to this article first:
https://support.office.com/en-us/article/use-ex...

If you can solve the issue by yourself, came back here and ask. There are millions of people in here who have had the same problems switching from 2003 to 2010. Try also the search function of this forum, in most cases you'll find the same or similar question already solved in here.
