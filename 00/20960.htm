Sorry, there's no way to eliminate the notices short of preventing the update. Here's how you can set the links to manual updating so that you can at least prevent the messages when the show opens:

  1. In PowerPoint, click on FILE, then Options.
  2. Click on Quick Access Toolbar.
  3. Under Choose command from:, change the dropdown to All Commands.
  4. Scroll down until you find Edit Links to Files. Select it and click on the Add button. OK Out.

A new icon appears at the right end of the Quick Access Toolbar. Now you in your presentation you can select a linked file, then click on the Edit Links to Files button. Uncheck Automatic Updates, then OK out. You can use the same command to manually update the linked file between shows.
