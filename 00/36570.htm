andre da costa say:

1) What benefits do I get with installing the 64-bit since I only have 2GB of memory.
None, because Windows 7 64 bit really takes advantage of systems with 4 GBs or more RAM. Yes, you can run it, but you are seeing any of the true benefits.

2) Are the above benefits worth using the 64-bit, or will I possibly use up more memory because all my 32-bit applications use more memory. I heard that this is very possible when running 32- bit applications under 64-bit Windows.  Really have a concern about this since most windows application still do not run at 64-bit applications.

64 vs 32
I suggest you stick with Windows 7 32 bit. Windows 7 64 bit comes in handy when you need to address at least 4 GBs or more of RAM. Windows 7 32 bit can utilize up to 3.2 GBs of RAM and you are right within that range. Because the memory address space is much larger for 64 bit Windows, that means, you need twice as much memory than 32 bit Windows to accomplish some of the same task.

As for compatibility, you will need 64 bit device drivers for any hardware devices you might have. Also, there is no 16 bit subsystem in Windows 7 64 bit, which means, your applications must be 32 bit only, not 16 bit installer or uninstallers. Also, if you decide to move to Windows 7 64 bit in the future, there is no upgrade path from 32 bit Windows, clean install only.

3) If you were in my position, which would you use.

I would stick with 32 bit Windows or upgrade the amount of installed RAM to 4 GBs then upgrade Windows 7 64 bit. I have seen significant performance benefits in areas such as start up, loading of the desktop, better multi-tasking performance with applications and windows.
