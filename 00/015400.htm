jllatham say:

Named ranges created directly in the Name Manager don't show up in the list! 

Let us assume that your programmer gave the name LookupCol_B as the name for that reference.  If the Lookup sheet is visible, you should be able to type the name, LookupCol_B into the Name Box and Excel should go to the Lookup sheet and select all of the cells currently referenced by that name on the sheet.

I personally would have probably used COUNTA(Lookup!$B:$B) for that part of the formula, but that's possibly a matter of style rather than technical requirement.  COUNT() only returns a count of cells containing numbers, while COUNTA() returns a count of any cell with anything in it (numbers or text).

Example: if column B has a label in B1 and a number in B2 and nothing else, then COUNT($B:$B) will return 1 (for the number in B2), while COUNTA($B:$B) would return 2.  Now, if your column B has a label in row 1 followed by nothing but numbers, then the way it's been done would end up referencing just the numbers in the column, and that may be what is needed in this situation.
